Είναι μια διαδικασία δύο βημάτων, πρώτα φορτώστε το email και αποδώστε το σε HTML μέσω του [Aspose.Email for Python via .NET](https://products.aspose.com/email/python-net/). Δεύτερον, φορτώστε το μετατρεπόμενο HTML χρησιμοποιώντας [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) και αποθηκεύστε το σε μορφή DOTX.
