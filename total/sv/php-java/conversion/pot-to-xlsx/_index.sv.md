Med Aspose.Total för PHP via Java kan du enkelt lägga till POT till XLSX-konvertering till valfri PHP-applikation med bara några steg.

{{% blocks/products/pf/agp/feature-section-col title="Hur konverterar man POT till XLSX?" %}}
1. Öppna och ladda POT-fil med Presentation class
2. Ställ in HTML-sparalternativen med HtmlOptions-objektet eller anropa sparmetoden direkt
3. Spara POT-filen i HTML med hjälp av sparmetoden med angiven sökväg och SaveFormat.HTML som parametrar

5. Anropa sparmetoden för att konvertera den till XLSX-format
