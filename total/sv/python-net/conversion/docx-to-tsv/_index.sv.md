{{% blocks/products/pf/feature-page-summary %}}

För en Python-utvecklare som försöker lägga till en DOCX till TSV-konverteringsfunktion i applikationen. [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API kan hjälpa till att automatisera konverteringsprocessen. Det är ett komplett paket med olika API:er som hanterar olika format.

Det är främst i två steg. Använd först [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) API för att konvertera DOCX-fil till HTML. Efter det, genom att använda Excel Python API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), spara den skapade HTML-koden i önskat Microsoft Excel-format.

{{% blocks/products/pf/agp/feature-section-col title="Hur man konverterar DOCX till TSV i Python" %}}
- **Steg 1** Öppna DOCX-källfilen med [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)-klassen
- Spara DOCX-fil till HTML genom att använda [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/)-metoden genom att ange filnamnet och önskad katalogsökväg
-  **Steg 2** Ladda HTML-fil med en instans av Workbook-klassen med fil och LoadOptions som parametrar
-  Anropa "spara"-metoden medan du anger sökvägen till TSV-filen. Så din DOCX-fil konverteras till TSV på den angivna sökvägen

- För DOCX till TSV-konvertering krävs Python 3.5 eller senare
