{{% blocks/products/pf/feature-page-summary %}}

För en Python-utvecklare som försöker lägga till en PPTM till XLTX-konverteringsfunktion i applikationen. [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API kan hjälpa till att automatisera konverteringsprocessen. Det är ett komplett paket med olika API:er som hanterar olika format inklusive PPTM- och XLTX-filer.

Det är främst i två steg. Använd först [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) API för att konvertera PPTM-fil till HTML. Efter det genom att använda Excel Python API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), spara den skapade HTML-koden i önskat Microsoft Excel-format.

{{% blocks/products/pf/agp/feature-section-col title="Hur man konverterar PPTM till XLTX i Python" %}}
- **Steg 1** Använd klassinstansen [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/) för att öppna PPTM-källfilen 
- Spara PPTM-fil till HTML genom att använda [save](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)-metoden genom att ange filnamnet och önskad katalogsökväg
-  **Steg 2** Ladda HTML-fil med en instans av Workbook-klassen
-  Anropa "spara"-metoden medan du anger sökvägen till XLTX-filen. Så din PPTM-fil konverteras till XLTX på den angivna sökvägen
