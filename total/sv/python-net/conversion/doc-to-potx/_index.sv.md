För en Python-utvecklare, vem försöker lägga till en DOC till POTX-konverteringsfunktion i applikationen? [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API kan hjälpa till att automatisera konverteringsprocessen. Det är ett komplett paket med olika API:er som hanterar olika format.  

Det är främst i två steg. Använd först [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) API för att konvertera DOC-fil till PDF. Efter det genom att använda PowerPoint Python API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/), spara den skapade PDF-filen i Presentation som ett POTX-format.

{{% blocks/products/pf/agp/feature-section-col title="Hur konverterar man DOC till POTX i Python" %}}
- **Steg 1** Öppna käll-DOC-filen med [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)-klassen
- Spara DOC-fil till PDF genom att använda metoden [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/) genom att ange filnamnet och önskad katalogsökväg.
-  **Steg 2** Ladda PDF-fil med en instans av klassen [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)
-  Anropa 'spara'-metoden samtidigt som du anger sökvägen för utdatafilen & SaveFormat.POTX som parametrar.  din DOC-fil konverteras till POTX på den angivna sökvägen.
