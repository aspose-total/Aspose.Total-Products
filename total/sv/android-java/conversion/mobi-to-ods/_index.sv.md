Innan du konverterar MOBI till ODS kan du ta bort oanvänd information från MOBI Mobiument via [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/). Ibland kan du behöva ta bort oanvänd eller dubblerad information för att minska storleken på utdatadokumentet och bearbetningstiden.
