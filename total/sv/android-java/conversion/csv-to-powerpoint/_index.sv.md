[Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) är ett paket med kraftfulla API:er för filautomatisering. Genom att använda två av dess API:er kan du integrera CSV till POWERPOINT-konverteringsfunktionen i dina Android-applikationer. I det första steget kan du exportera CSV till PDF genom att använda [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/). 1. Öppna CSV-filen med klassen [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)
2. Konvertera CSV till PDF och ställ in SaveFormat till AUTO
3. Ladda den konverterade PDF-filen med klassen [Powerpointument](https://reference.aspose.com/pdf/java/com.aspose.pdf/Powerpointument)
4. Spara dokumentet i POWERPOINT-format med [save](https://reference.aspose.com/pdf/java/com.aspose.pdf/Powerpointument#save-java.lang.String-com.aspose.pdf.SaveOptions -) metod

// load the CSV file using Workbook class
Workbook book = new Workbook("input.csv");

Förutom dokumentkonvertering ger [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/) massor av andra funktioner också.
