Genom att använda [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) kan du integrera MOBI till XLSB-konverteringsfunktionen i dina Android-applikationer. För det första kan du konvertera MOBI till HTML genom att använda funktionsrika, dokumentmanipulerings- och konverterings-API [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/). {{% blocks/products/pf/feature-page-section  h2="Ta bort oanvänd information från ett MOBI-dokument i Android via Java" %}}
Innan du konverterar MOBI till XLSB kan du ta bort oanvänd information från MOBI Mobiument via [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/). Ibland kan du behöva ta bort oanvänd eller dubblerad information för att minska storleken på utdatadokumentet och bearbetningstiden. Klassen [CleanupOptions](https://reference.aspose.com/words/java/com.aspose.words/CleanupOptions) låter dig ange alternativ för dokumentrensning. För att ta bort dubbletter av stilar eller bara oanvända stilar eller listor från dokumentet kan du använda metoden [Cleanup](https://reference.aspose.com/words/java/com.aspose.words/Mobiument#cleanup()). Du kan använda [UnusedStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedStyles) och [UnusedBuiltinStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedBuiltinStyles) egenskaper för att upptäcka och ta bort stilar som är markerade som "oanvända".
