Därefter kan du konvertera HTML till RTF genom att använda API för dokumentbearbetning [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/).

{{% blocks/products/pf/agp/feature-section-col title="PPS till RTF-rendering i Android" %}}
1. Öppna PPS-filen med klassen [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Konvertera PPS till HTML genom att använda [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.ISaveOptions-)-metoden och ställ in Html som SaveFormat
3. Ladda den konverterade HTML-filen med klassen [Rtfument](https://reference.aspose.com/words/java/com.aspose.words/Rtfument)
4. Spara dokumentet i RTF-format med metoden [save](https://reference.aspose.com/words/java/com.aspose.words/Rtfument#save(java.lang.String,int)) och ställ in Rtf som SaveFormat
