{{% blocks/products/pf/agp/feature-section-col title="Hur man konverterar RTF till PPT i Android" %}}
1. Öppna RTF-filen med klassen [Rtfument](https://reference.aspose.com/words/java/com.aspose.words/Rtfument)
2. Konvertera RTF-fil till HTML genom att använda [save](https://reference.aspose.com/words/java/com.aspose.words/Rtfument#save(java.lang.String,com.aspose.words.SaveOptions) )) metod
3. Initiera ett nytt [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation) objekt
5. Extrahera innehåll från HTML-fil med BufferedReader och skriv innehållet i din presentationsfil
6. Spara dokumentet till PPT med metoden [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
