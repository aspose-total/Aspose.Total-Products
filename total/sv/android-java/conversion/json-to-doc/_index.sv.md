Du kan konvertera JSON till DOC i dina Android-applikationer i tvåstegsprocess. För det första, genom att använda Powerful Spreadsheet Processing API [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/) kan du tolka JSON till PDF. I det andra steget kan du konvertera PDF till DOC genom att använda Word Processing API [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/). Båda API:erna faller under produktfamiljen [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/).

1. Skapa ett nytt [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook) objekt och läs giltig JSON-data från filen
2. Importera JSON-fil till kalkylblad med klassen [JsonUtility](https://reference.aspose.com/cells/java/com.aspose.cells/JsonUtility) och [Save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.SaveOptions)) den som PDF
3. Ladda PDF-dokument med hjälp av klassen [Document](https://reference.aspose.com/words/java/com.aspose.words/Document)
4. Spara dokumentet i DOC-format med [Save](https://reference.aspose.com/words/java/com.aspose.words/Document#save(java.lang.String,com.aspose.words.SaveOptions)) metod

{{% blocks/products/pf/feature-page-section  h2="Konvertera JSON-format till DOC med vattenstämpel i Android via Java" %}}
Med hjälp av API:t kan du också konvertera JSON till DOC med vattenstämpel. För att lägga till en vattenstämpel till ditt DOC-dokument kan du först analysera JSON-filen till PDF och lägga till en vattenstämpel till den. För att lägga till en vattenstämpel, ladda den nyskapade PDF-filen med klassen [Document](https://reference.aspose.com/words/java/com.aspose.words/Document), skapa en instans av TextWatermarkOptions och ställ in dess egenskaper, anrop Watermark.setText-metoden och skicka vattenstämpeltext och objekt av TextWatermarkOptions. När du har lagt till vattenstämpeln kan du spara dokumentet i DOC.
