{{% blocks/products/pf/agp/feature-section-col title="Hur man konverterar DOT till PPTX i Android" %}}
1. Öppna DOT-filen med klassen [Dotument](https://reference.aspose.com/words/java/com.aspose.words/Dotument)
2. Konvertera DOT-fil till HTML genom att använda [save](https://reference.aspose.com/words/java/com.aspose.words/Dotument#save(java.lang.String,com.aspose.words.SaveOptions) )) metod
3. Initiera ett nytt [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation) objekt
5. Extrahera innehåll från HTML-fil med BufferedReader och skriv innehållet i din presentationsfil
6. Spara dokumentet till PPTX med metoden [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
