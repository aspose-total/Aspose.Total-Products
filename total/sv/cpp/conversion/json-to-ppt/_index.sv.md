{{% blocks/products/pf/feature-page-summary %}}
Du kan konvertera JSON till PPT i vilken C++-applikation som helst i två enkla steg. För det första, genom att använda [Aspose.Cells for C++](https://products.aspose.com/cells/cpp/), kan du analysera JSON till PPTX. Efter det, genom att använda [Aspose.Slides for C++](https://products.aspose.com/slides/cpp/), kan du konvertera PPTX till PPT. Båda API:erna kommer under paketet [Aspose.Total for C++](https://products.aspose.com/total/cpp/).

{{% blocks/products/pf/feature-page-section  h2="Ställ in layout och konvertera JSON-format till PPT via C++" %}}
När du analyserar JSON till PPT kan du också ställa in storleken på rader och kolumner genom att ladda JSON med klassen [IWorkbook](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook). Om du behöver ställa in samma radhöjd för alla rader i kalkylbladet kan du göra det genom att använda [SetStandardHeight](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_cell#a0b79a3163e2b601aa1b6a6a1e3f1467 ) metoden för samlingen [ICells](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_cell). På samma sätt, för att ställa in samma kolumnbredd för alla kolumner i kalkylbladet, använd ICells-samlingens [SetStandardWidth](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_cell#a48f5dbccc3bf4bb9e6e882094b5) metod.
