{{% blocks/products/pf/feature-page-section  h2="Sök i Microsoft Powerpoint-presentationer" %}}

Vi använder [Aspose.Slides for C++](https://products.aspose.com/slides/cpp/), ett underordnat API av [Aspose.Total for C++](https://products.aspose.com/total/cpp/) som är utformat för särskilda presentationsmanipuleringsfunktioner samt uppgifter i samband med att hämta och söka i dess innehåll. Nedanstående kod fungerar på en PowerPoint-presentation. Det börjar med att ladda en presentationsfil med Aspose.Slides. Koden använder sedan SlideUtil::FindAndReplaceText-funktionen för att hitta och ersätta texten genom hela presentationen, med parametern "true" som indikerar skiftlägeskänslig matchning. Slutligen sparar den den uppdaterade presentationen. Så den här koden automatiserar processen att hitta och ersätta specifik text i en PowerPoint-presentation. Dessutom stöder Document C++ Search Library också PDF, Microsoft [Sök i Word-dokument](https://products.aspose.com/total/cpp/search/word/) och andra format.
