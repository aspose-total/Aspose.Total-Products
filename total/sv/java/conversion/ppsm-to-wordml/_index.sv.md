[Aspose.Total för Java](https://products.aspose.com/total/java/) Filformatsautomatiseringsbibliotek ger Java-utvecklare möjlighet att automatisera batchkonverteringsprocessen från PowerPoint PPSM till Word WORDML. Att konvertera dokumentet är en process i två steg och involverar användning av två API:er. Vi kommer att använda [Aspose.Slides för Java](https://products.aspose.com/slides/java/) som är ett PowerPoint-API för presentationsmanipulering och hantering för att konvertera PPSM till HTML. Efter det kommer vi att konvertera HTML till WORDML genom att använda funktionsrika Word Processing API [Aspose.Words for Java](https://products.aspose.com/words/java/).

{{% blocks/products/pf/agp/feature-section-col title="Hur man konverterar PPSM till WORDML via Java" %}}
1. Öppna PPSM-filen med klassen [Presentation](https://apireference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Konvertera PPSM till HTML genom att använda [save](https://apireference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.ISaveOptions-)-metoden och ställ in Html som SaveFormat
3. Ladda den konverterade HTML-filen med klassen [Wordmlument](https://apireference.aspose.com/words/java/com.aspose.words/Wordmlument)
4. Spara dokumentet i WORDML-format med metoden [save](https://apireference.aspose.com/words/java/com.aspose.words/Wordmlument#save(java.lang.String,int))
