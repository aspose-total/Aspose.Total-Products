{{% blocks/products/pf/agp/feature-section-col title="Hur man konverterar OTT till PPT via Java" %}}
1. Öppna OTT-filen med klassen [Document](https://reference.aspose.com/words/java/com.aspose.words/Document)
2. Konvertera OTT-fil till HTML genom att använda [save](https://reference.aspose.com/words/java/com.aspose.words/Document#save(java.lang.String,com.aspose.words.SaveOptions)) metod

5. Extrahera innehåll från HTML-fil med BufferedReader och skriv innehållet i din presentationsfil
6. Spara dokumentet till PPT med metoden [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
