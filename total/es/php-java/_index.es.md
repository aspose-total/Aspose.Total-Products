Aspose.Total para PHP a través de Java es un paquete completo para la manipulación de hojas de cálculo de Excel, así como para presentaciones de PowerPoint y OpenOffice. Los desarrolladores pueden crear, manipular, renderizar y convertir archivos de Excel a través de aplicaciones PHP, así como cargar, leer, escribir y exportar formatos de PowerPoint, incluidos PPT, PPTX, PPS, POT, PPSX, OpenOffice ODP y más formatos. Los desarrolladores pueden integrar fácilmente la API de PHP para el reconocimiento y la generación de códigos de barras.
{{< /blocks/products/pf/total/overview >}}

{{< blocks/products/pf/total/products description="Aspose.Total for PHP via Java incluye las siguientes API:" >}}

{{< blocks/products/pf/total/product-item href="/pdf/php-java/" imgSrc="/total/images/aspose_pdf-for-php-java.svg" productName="Aspose.PDF for PHP via Java">}}

Aspose.PDF for PHP via Java es una potente API que permite a los desarrolladores PHP crear, editar, convertir y proteger documentos PDF sin depender de Adobe Acrobat. Es compatible con aplicaciones de 32 y 64 bits y ofrece funciones avanzadas como gestión de formularios, edición de metadatos, manipulación de texto y páginas, anotaciones, marcadores, marcas de agua, archivos adjuntos y gestión de fuentes personalizadas.

{{< /blocks/products/pf/total/product-item >}}


{{< blocks/products/pf/total/product-item href="/cells/php-java/" imgSrc="/total/images/aspose_cells-for-php-java.svg" productName="Aspose.Cells for PHP via Java">}}
Aspose.Cells para PHP a través de la API de Java está diseñado para administrar y procesar hojas de cálculo de Excel. Maneja tareas como convertir formatos de hojas de cálculo, administrar y representar gráficos, implementar fórmulas, aplicar estilos detallados a hojas de trabajo y mucho más. Con la integración de JavaBridge, los desarrolladores pueden incorporar sin problemas el poder de crear, editar, convertir y renderizar archivos de Excel en sus aplicaciones PHP.
{{< /blocks/products/pf/total/product-item >}}

{{< blocks/products/pf/total/product-item href="/slides/php-java/" imgSrc="/total/images/aspose_slides-for-php-java.svg" productName="Aspose.Slides for PHP via Java">}}
Aspose.Slides para PHP a través de la biblioteca Java permite a los desarrolladores trabajar y procesar presentaciones de manera efectiva. Ofrece lectura, edición y conversión de presentaciones de PowerPoint sin depender de aplicaciones externas. La biblioteca permite cargar, abrir y ver presentaciones, convertirlas a varios formatos, renderizar e imprimir presentaciones, cifrarlas y descifrarlas, manipular entidades de presentación como diapositivas maestras, formas, gráficos, cuadros de audio y video, animaciones y más.
{{< /blocks/products/pf/total/product-item >}}

{{< blocks/products/pf/total/product-item href="/barcode/php-java/" imgSrc="/total/images/aspose_barcode-for-php-java.svg" productName="Aspose.BarCode for PHP via Java">}}
Aspose.BarCode para PHP a través de Java es una colección de API de generación y lectura de códigos de barras diseñadas para aplicaciones PHP. Brinda a los desarrolladores la capacidad de leer varios tipos de códigos de barras, incluidos códigos de barras 1D, 2D y postales, a partir de imágenes capturadas en diferentes ángulos. La API del generador de códigos de barras permite la creación de imágenes de códigos de barras en diferentes formatos, como JPG, PNG, GIF y BMP. Además, ofrece varias funciones de imagen, como manipulación de bordes, ajuste de color, rotación de imágenes, personalización de resolución y más. Al utilizar la API, los desarrolladores pueden incorporar fácilmente la funcionalidad del código de barras en sus aplicaciones PHP.
