{{% blocks/products/pf/agp/feature-section-col title="Cómo convertir POT a DIF a través de Java" %}}
1. Abra el archivo POT usando la clase [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Convierta POT a HTML usando [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.ISaveOptions-) método
3. Cargue el documento HTML utilizando la clase [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)
4. Guarde el documento en formato DIF usando [save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.GuardarOpciones)) método
