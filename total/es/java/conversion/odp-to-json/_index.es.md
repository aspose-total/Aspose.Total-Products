{{% blocks/products/pf/feature-page-summary %}}
Convertir ODP a formato JSON a través de [Aspose.Total for Java](https://products.aspose.com/total/java/) es un proceso simple de dos pasos. En el primer paso, puede exportar ODP a HTML usando [Aspose.Slides for Java](https://products.aspose.com/slides/java/). En segundo lugar, al usar [Aspose.Cells for Java](https://products.aspose.com/cells/java/), puede convertir HTML a JSON.

{{% blocks/products/pf/agp/feature-section-col title="Convierta ODP a formato JSON a través de Java" %}}
1. Abra el archivo ODP usando la clase [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Convierta ODP a HTML usando [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.ISaveOptions-) método
3. Cargue el documento HTML utilizando la clase [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)
4. Guarde el documento en formato JSON usando [save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.GuardarOpciones)) método
