{{% blocks/products/pf/agp/feature-section-col title="Cómo convertir DOCX a PPTM a través de Java" %}}
1. Abra el archivo DOCX usando la clase [Document](https://reference.aspose.com/words/java/com.aspose.words/Document)

6. Guarde el documento en PPTM usando el método [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
