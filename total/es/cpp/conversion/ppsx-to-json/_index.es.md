Al utilizar [Aspose.Total for C++](https://products.aspose.com/total/cpp/), puede convertir archivos PPSX a formato JSON dentro de aplicaciones C++ en dos sencillos pasos. En primer lugar, al usar [Aspose.Slides for C++](https://products.aspose.com/slides/cpp/), puede exportar PPSX a HTML. Después de eso, al usar [Aspose.Cells for C++](https://products.aspose.com/cells/cpp/) API de programación de hojas de cálculo, puede convertir HTML a JSON.
