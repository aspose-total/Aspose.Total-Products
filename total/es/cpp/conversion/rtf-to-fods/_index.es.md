{{% blocks/products/pf/feature-page-summary %}}
Puede incluir fácilmente la función de conversión de RTF a FODS dentro de sus aplicaciones C++. Mediante el uso de la API de manipulación y conversión de rtfumentos rica en funciones, potente y fácil de usar [Aspose.Words for C++](https://products.aspose.com/words/cpp/), puede exportar RTF a HTML. Después de eso, usando [Aspose.Cells for C++](https://products.aspose.com/cells/cpp/), puede convertir HTML a FODS. Ambas API se incluyen en el paquete [Aspose.Total for C++](https://products.aspose.com/total/cpp/).

{{% blocks/products/pf/agp/feature-section-col title="API de C++ para convertir RTF a FODS" %}}
1. Abra el archivo RTF usando la referencia de clase [Rtfumento](https://reference.aspose.com/words/cpp/class/aspose.words.rtfument)
2. Convierta RTF a HTML usando la función miembro [Guardar](https://reference.aspose.com/words/cpp/class/aspose.words.rtfument#save_string_saveformat)
3. Cargue el rtfumento HTML utilizando la referencia de clase [IWorkbook](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook)
4. Guarde el rtfumento en formato FODS usando la función miembro [Guardar](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook#a5dc7de23f7ceba76a05dc1d49f51502e)

{{% blocks/products/pf/feature-page-section  h2="Guardar archivo FODS para transmitir a través de C++" %}}
Después de convertir RTF a FODS, [Aspose.Cells for C++](https://products.aspose.com/cells/cpp/) le permite guardar su rtfumento para transmitir. Para guardar archivos en una secuencia, cree un objeto MemoryStream o FileStream y guarde el archivo en ese objeto de secuencia llamando al [IWorkbook](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook) [Guardar](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook#a77072cfb929787df9ad1f38b02f58349) del objeto. Especifique el formato de archivo deseado usando la enumeración [SaveFormat](https://reference.aspose.com/cells/cpp/namespace/aspose.cells#a11cae527e4e68f1adcac8f47ea64481a) al llamar a [Save](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook#a77072cfb929787df9ad1f38b02f58349).
