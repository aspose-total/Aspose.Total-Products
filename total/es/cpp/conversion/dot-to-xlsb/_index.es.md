Puede incluir fácilmente la función de conversión de DOT a XLSB dentro de sus aplicaciones C++. Mediante el uso de la API de manipulación y conversión de dotumentos rica en funciones, potente y fácil de usar [Aspose.Words for C++](https://products.aspose.com/words/cpp/), puede exportar DOT a HTML. Después de eso, usando [Aspose.Cells for C++](https://products.aspose.com/cells/cpp/), puede convertir HTML a XLSB. Ambas API se incluyen en el paquete [Aspose.Total for C++](https://products.aspose.com/total/cpp/).
