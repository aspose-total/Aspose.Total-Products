¿Necesita desarrollar una aplicación o utilidad de análisis XLS? Con [Aspose.Cells for .NET](https://products.aspose.com/cells/es/net/), una API secundaria de [Aspose.Total for .NET](https://products.aspose.com/total/es/net/), cualquier desarrollador de .NET puede integrar el código API anterior dentro de su aplicación de análisis de documentos.La potente biblioteca .NET permite programar cualquier solución de análisis de documentos para extraer imágenes y texto.Además, puede admitir muchos formatos populares, incluido el formato XLS.<br /><br />
