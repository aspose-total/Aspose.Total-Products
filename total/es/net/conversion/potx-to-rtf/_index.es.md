Para automatizar el proceso de conversión por lotes de cualquier presentación POTX de PowerPoint a archivos de documentos de Word, usaremos [Aspose.Slides for .NET](https://products.aspose.com/slides/net) y [Aspose.Words para .NET](https://products.aspose.com/words/net) API. La primera es una API de manipulación de presentaciones de PowerPoint que le permite crear o modificar diapositivas de Microsoft PowerPoint. Mientras que este último es una API de procesamiento de textos para procesar o manipular documentos de Microsoft Word. Ambas API forman parte del paquete [Aspose.Total for .NET](https://products.aspose.com/total/net). {{% blocks/products/pf/agp/code-block title="Este ejemplo de código muestra cómo convertir un POTX a RTF usando C#" offSpacer="" %}}

```cs
// Load the Microsoft PowerPoint POTX file
Aspose.Slides.Presentation potx = new Aspose.Slides.Presentation("source.potx");

var stream = new MemoryStream();

potx.Save(stream, Aspose.Slides.Export.SaveFormat.Html);
stream.Flush();
stream.Seek(0, SeekOrigin.Begin);
// stream.Position = 0;

// Load the content of the presentation to a Word document
var rtf = new Aspose.Words.Document(stream);
      
// Save the Word RTF document
rtf.Save("output.rtf", Aspose.Words.SaveFormat.Rtf);

```
