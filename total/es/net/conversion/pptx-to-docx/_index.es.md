Mientras que este último es una API de procesamiento de textos para procesar o manipular documentos de Microsoft Word. Ambas API forman parte del paquete [Aspose.Total for .NET](https://products.aspose.com/total/net). {{% blocks/products/pf/agp/code-block title="Este ejemplo de código muestra cómo convertir un PPTX a DOCX usando C#" offSpacer="" %}}

```cs
// Load the Microsoft PowerPoint PPTX file
Aspose.Slides.Presentation pptx = new Aspose.Slides.Presentation("source.pptx");

var stream = new MemoryStream();

pptx.Save(stream, Aspose.Slides.Export.SaveFormat.Html);
stream.Flush();
stream.Seek(0, SeekOrigin.Begin);
// stream.Position = 0;

// Load the content of the presentation to a Word document
var docx = new Aspose.Words.Document(stream);
      
// Save the Word DOCX document
docx.Save("output.docx", Aspose.Words.SaveFormat.Docx);

```
