{{% blocks/products/pf/feature-page-summary %}}
Al utilizar [Aspose.Total para .NET](https://products.aspose.com/total/net/), puede incluir la función de conversión de DOTX a XLSB dentro de cualquier aplicación .NET, C#, ASP.NET y VB.NET en dos sencillos pasos. En primer lugar, al usar [Aspose.Words for .NET](https://products.aspose.com/words/net/), puede exportar DOTX a HTML. Después de eso, al usar [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) API de programación de hojas de cálculo, puede convertir HTML a XLSB.

{{% blocks/products/pf/agp/feature-section-col title="API .NET para convertir DOTX a XLSB" %}}
1. Abra el archivo DOTX usando la clase [Documento](https://reference.aspose.com/words/net/aspose.words/Document)
2. Convierta DOTX a HTML usando el método [Guardar](https://reference.aspose.com/words/net/aspose.words.Document/save/methods/4)
3. Cargue el Documento HTML utilizando la clase [Libro de trabajo](https://reference.aspose.com/cells/net/aspose.cells/workbook)
4. Guarde el Documento en formato XLSB usando el método [Guardar](https://reference.aspose.com/cells/net/aspose.cells.workbook/save/methods/4) y configure `XLSB` como SaveFormat

{{% blocks/products/pf/feature-page-section  h2="Agregue propiedades personalizadas en un archivo XLSB a través de C#" %}}
Al convertir DOTX a XLSB, [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) le permite agregar propiedades personalizadas en sus Documentos XLSB. Para agregar una propiedad personalizada, puede usar el método [Add](https://reference.aspose.com/cells/net/aspose.cells.properties/customDocumentpropertycollection/methods/add/index) para la [CustomDocumentPropertyCollection](https://reference.aspose.com/cells/net/aspose.cells.properties/customDocumentpropertycollection) clase. El método Add agrega la propiedad al archivo de Excel y devuelve una referencia para la propiedad del nuevo Documento como [Aspose.Cells.Properties.DocumentProperty](https://reference.aspose.com/cells/net/aspose.cells.properties /propiedadDocumento) objeto.
