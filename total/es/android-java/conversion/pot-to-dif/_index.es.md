{{% blocks/products/pf/feature-page-summary %}}
Puede convertir fácilmente un archivo POT a DIF en sus aplicaciones de Android en un proceso de dos pasos a través de [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/). En el primer paso, puede exportar el archivo POT a HTML utilizando [Aspose.Slides for Android via Java](https://products.aspose.com/slides/android-java/). En segundo lugar, al usar [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/), puede convertir HTML a DIF.

{{% blocks/products/pf/agp/feature-section-col title="Cómo convertir POT a DIF en Android" %}}
1. Abra el archivo POT usando la clase [Presentación](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Convierta POT a HTML usando [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.ISaveOptions-) método
3. Cargue el documento HTML utilizando la clase [Libro de trabajo](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)
4. Guarde el documento en formato DIF usando [guardar](https://reference.aspose.com/cells/java/com.aspose.cells/)

{{% blocks/products/pf/agp/feature-section-col title="Requisitos de conversión" %}}

Alternativamente, puede obtener un archivo ZIP de [descargas](https://downloads.aspose.com/total/androidjava).
