Puede convertir fácilmente un archivo POTX a XLAM en sus aplicaciones de Android en un proceso de dos pasos a través de [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/). 1. Abra el archivo POTX usando la clase [Presentación](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Convierta POTX a HTML usando [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.ISaveOptions-) método
3. Cargue el documento HTML utilizando la clase [Libro de trabajo](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)
4. Guarde el documento en formato XLAM usando [guardar](https://reference.aspose.com/cells/java/com.aspose.cells/)
