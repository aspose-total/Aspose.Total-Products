{{% blocks/products/pf/feature-page-section  h2="Eliminar información no utilizada de un mobiumento MOBI en Android a través de Java" %}}
Antes de convertir MOBI a DIF, puede eliminar la información no utilizada del mobiumento MOBI a través de [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/). En ocasiones, es posible que deba eliminar información no utilizada o duplicada para reducir el tamaño del mobiumento de salida y el tiempo de procesamiento. La clase [CleanupOptions](https://reference.aspose.com/words/java/com.aspose.words/CleanupOptions) le permite especificar opciones para la limpieza de mobiumentos. Para eliminar estilos duplicados o simplemente estilos o listas no utilizados del mobiumento, puede usar el método [Cleanup](https://reference.aspose.com/words/java/com.aspose.words/Mobiument#cleanup()). Puede usar [UnusedStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedStyles) y [UnusedBuiltinStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedBuiltinStyles) para detectar y eliminar estilos marcados como "sin usar".
