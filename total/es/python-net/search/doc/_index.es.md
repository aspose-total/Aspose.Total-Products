{{% blocks/products/pf/agp/feature-section-col title="Cómo buscar en un archivo DOC a través de Python" %}}

1. API de referencia dentro del proyecto directamente desde PyPI ([Aspose.Words](https://pypi.org/project/aspose-words/)) 
1. Cree un objeto de clase de documento o cargue un archivo DOC usándolo.
1. Cree un objeto de clase DocumentBuilder y agregue texto al archivo usando el método DocumentBuilder.write().
1. Utilice la función range.replace para buscar y reemplazar texto junto con FindReplaceOptions.
1. O utilice range.replace_regex para expresiones regulares junto con replacement.FindReplaceOptions.
1. Llame al método 'guardar' para guardar el documento.
