{{% blocks/products/pf/feature-page-summary %}}

Para un desarrollador de Python, que está tratando de agregar una función de conversión de POTX a XLSM dentro de la aplicación. La API [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) puede ayudar a automatizar el proceso de conversión. Es un paquete completo de varias API que manejan diferentes formatos, incluidos archivos POTX y XLSM.

Es principalmente en dos pasos. En primer lugar, use la API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) para convertir el archivo POTX a HTML. Después de eso, usando Excel Python API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), guarde el HTML creado en el formato de Microsoft Excel deseado.

{{% blocks/products/pf/agp/feature-section-col title="Cómo convertir POTX a XLSM en Python" %}}
- **Paso 1** Use la instancia de clase [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/) para abrir el archivo POTX de origen 
- Guarde el archivo POTX en HTML utilizando el método [save](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/) proporcionando el nombre del archivo y la ruta del directorio deseado
-  **Paso 2** Cargue un archivo HTML con una instancia de la clase Workbook
-  Llame al método `save` mientras especifica la ruta del archivo XLSM de salida. Entonces su archivo POTX se convierte a XLSM en la ruta especificada
