{{% blocks/products/pf/feature-page-summary %}}

Para un desarrollador de Python, que está tratando de agregar una función de conversión de XLS a PPTM dentro de la aplicación, La API [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) puede ayudar a automatizar el proceso de conversión. Es un paquete completo de varias API que manejan diferentes formatos, incluidos archivos XLS y PPTM..

Es principalmente en dos pasos.. Primero use [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) API para convertir archivos XLS a PDF. Después de eso, usando la API de PowerPoint Python [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/), guarde el PDF creado en el formato de Microsoft PowerPoint deseado.

{{% blocks/products/pf/agp/feature-section-col title="Cómo convertir XLS a PPTM en Python" %}}
- **Paso 1** Use la instancia de clase Workbook para abrir el archivo XLS de origen 
- Guarde el archivo XLS en PDF utilizando el método save proporcionando el nombre del archivo y la ruta del directorio deseado
-  **Paso 2** Cargue el archivo PDF usando la clase [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)
-  Llame al método [save](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/) mientras especifica la ruta del archivo PPTM de salida. Entonces su archivo XLS se convierte a PPTM en la ruta especificada
