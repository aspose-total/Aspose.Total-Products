ODS aranabilir android tabanlı bir mobil yardımcı program geliştirmeniz mi gerekiyor? Aspose.Cells for Android via Java ile Aspose.Total for Android for Java'nın alt API'si olan herhangi bir android geliştiricisi, belgeler arasında metin aramayı programlamak için yukarıdaki API kodunu entegre edebilir.
