{{% blocks/products/pf/agp/feature-section-col title="Come convertire PPTM in ODS?" %}}
1. Apri e carica il file PPTM utilizzando la classe Presentazione
2. Imposta le opzioni di salvataggio HTML utilizzando l'oggetto HtmlOptions o chiama direttamente il metodo save
3. Salva il file PPTM in HTML utilizzando il metodo di salvataggio con il percorso specificato e SaveFormat.HTML come parametri
4. Utilizzare la classe Workbook per caricare l'HTML esportato
5. Chiama il metodo save per convertirlo in formato ODS

{{% blocks/products/pf/agp/feature-section-col title="Requisiti di conversione" %}}
1. PHP 7 o versioni successive sono installate
2. La macchina deve disporre di Oracle JDK 7 o versioni successive
3. Scarica il binario PHP/Java Bridge [JavaBridge.jar](http://php-java-bridge.sourceforge.net/pjb/download.php) e la libreria PHP java/Java.inc [Java.inc](http://php-java-bridge.sourceforge.net/pjb/download.php) e salvalo nella cartella delle impostazioni
4. Regola l'impostazione relativa alla piattaforma (Windows, Linux, MacOS ecc.) per [Aspose.Cells](https://docs.aspose.com/cells/php-java/setup-and-installation-guidelines/) e [Aspose.Slides](https://docs.aspose.com/slides/php-java/installation/)
