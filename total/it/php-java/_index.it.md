{{< blocks/products/pf/total/overview >}}
Aspose.Total per PHP via Java è un pacchetto completo per la manipolazione di fogli di calcolo Excel e per presentazioni PowerPoint e OpenOffice. Gli sviluppatori possono creare, manipolare, eseguire il rendering e convertire file Excel tramite applicazioni PHP, nonché caricare, leggere, scrivere ed esportare formati PowerPoint tra cui PPT, PPTX, PPS, POT, PPSX, OpenOffice ODP e altri formati. Gli sviluppatori possono facilmente integrare l'API PHP per il riconoscimento e la generazione di codici a barre.
{{< /blocks/products/pf/total/overview >}}

{{< blocks/products/pf/total/products description="Aspose.Total for PHP via Java include le seguenti API:" >}}

{{< blocks/products/pf/total/product-item href="/pdf/php-java/" imgSrc="/total/images/aspose_pdf-for-php-java.svg" productName="Aspose.PDF for PHP via Java">}}

Aspose.PDF for PHP via Java è una potente API che consente agli sviluppatori PHP di creare, modificare, convertire e proteggere documenti PDF senza dover ricorrere ad Adobe Acrobat. Supporta applicazioni a 32 e 64 bit e offre funzionalità avanzate come la gestione dei moduli, la modifica dei metadati, la manipolazione di testo e pagine, annotazioni, segnalibri, filigrane, allegati e gestione dei font personalizzati.

{{< /blocks/products/pf/total/product-item >}}


{{< blocks/products/pf/total/product-item href="/cells/php-java/" imgSrc="/total/images/aspose_cells-for-php-java.svg" productName="Aspose.Cells for PHP via Java">}}
Aspose.Cells per PHP tramite API Java è progettato per la gestione e l'elaborazione di fogli di calcolo Excel. Gestisce attività come la conversione di formati di fogli di calcolo, la gestione e il rendering di grafici, l'implementazione di formule, l'applicazione di stili dettagliati ai fogli di lavoro e molto altro. Con l'integrazione JavaBridge, gli sviluppatori possono incorporare senza problemi la potenza di creazione, modifica, conversione e rendering di file Excel nelle loro applicazioni PHP.
{{< /blocks/products/pf/total/product-item >}}

{{< blocks/products/pf/total/product-item href="/slides/php-java/" imgSrc="/total/images/aspose_slides-for-php-java.svg" productName="Aspose.Slides for PHP via Java">}}
Aspose.Slides per PHP tramite la libreria Java consente agli sviluppatori di lavorare ed elaborare le presentazioni in modo efficace. Offre la lettura, la modifica e la conversione di presentazioni PowerPoint senza fare affidamento su applicazioni esterne. La libreria consente di caricare, aprire e visualizzare presentazioni, convertirle in vari formati, eseguire il rendering e stampare presentazioni, crittografarle e decrittografarle, manipolare entità di presentazione come diapositive master, forme, grafici, frame audio e video, animazioni e altro ancora.
{{< /blocks/products/pf/total/product-item >}}

{{< blocks/products/pf/total/product-item href="/barcode/php-java/" imgSrc="/total/images/aspose_barcode-for-php-java.svg" productName="Aspose.BarCode for PHP via Java">}}
Aspose.BarCode per PHP via Java è una raccolta di API per la generazione e la lettura di codici a barre progettate per applicazioni PHP. Offre agli sviluppatori la possibilità di leggere vari tipi di codici a barre, inclusi codici a barre 1D, 2D e postali, da immagini acquisite da diverse angolazioni. L'API del generatore di codici a barre consente la creazione di immagini di codici a barre in diversi formati come JPG, PNG, GIF e BMP. Inoltre, offre diverse funzionalità di imaging, come la manipolazione dei bordi, la regolazione del colore, la rotazione dell'immagine, la personalizzazione della risoluzione e altro ancora. Utilizzando l'API, gli sviluppatori possono incorporare facilmente la funzionalità del codice a barre nelle loro applicazioni PHP.
