{{% blocks/products/pf/agp/feature-section-col title="Come convertire RTF in POWERPOINT in Android" %}}
1. Aprire il file RTF utilizzando la classe [Rtfument](https://reference.aspose.com/words/java/com.aspose.words/Rtfument)

6. Salvare il rtfumento in POWERPOINT utilizzando il metodo [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
