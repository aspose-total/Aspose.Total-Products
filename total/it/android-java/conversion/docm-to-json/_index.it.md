{{% blocks/products/pf/feature-page-summary %}}
Puoi convertire DOCM in formato JSON nelle tue applicazioni Android tramite [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/). Utilizzando l'API di conversione e manipolazione dei docmumenti [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/), puoi esportare DOCM in HTML. Successivamente, utilizzando [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/), puoi convertire HTML in JSON.

{{% blocks/products/pf/agp/feature-section-col title="Converti DOCM in formato JSON in Android" %}}
1. Aprire il file DOCM utilizzando la classe [Document](https://reference.aspose.com/words/java/com.aspose.words/Document)
2. Converti DOCM in HTML utilizzando [Save](https://reference.aspose.com/words/java/com.aspose.words/Document#save(java.lang.String,com.aspose.words.SaveOptions) ) metodo
3. Carica il docmumento HTML utilizzando la classe [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)
4. Salvare il docmumento in formato JSON utilizzando [Save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.SaveOptions)).
