{{% blocks/products/pf/agp/feature-section-col title="Come convertire PPTX in XLSX in Android" %}}
1. Aprire il file PPTX utilizzando la classe [Presentazione](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Converti PPTX in HTML utilizzando [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.ISaveOptions-) metodo
3. Carica il documento HTML utilizzando la classe [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)
4. Salva il documento in formato XLSX utilizzando [save](https://reference.aspose.com/cells/java/com.aspose.cells/)
