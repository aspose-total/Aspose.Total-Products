{{% blocks/products/pf/agp/feature-section-col title="Come cercare file PDF tramite Android" %}}

1. Installa "Aspose.PDF per Java".
1. Aggiungi un riferimento alla libreria (importa la libreria) al tuo progetto Java.
1. Apri il file PDF di origine utilizzando la classe Document.
1. Crea un oggetto TextFragmentAbsorber per trovare tutte le istanze dell'input.
1. oppure definire i criteri di ricerca in Regex.
1. Accetta l'assorbitore per la pagina utilizzando getPages().accept(textFragmentAbsorber).
1. Ottieni la raccolta di frammenti di testo estratti tramite textFragmentAbsorber.getTextFragments().
1. Passa in rassegna i frammenti per ottenere l'output.
