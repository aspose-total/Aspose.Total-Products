È un pacchetto completo di varie API che gestiscono diversi formati, inclusi file POTM e XLTM.

È principalmente in due fasi. Innanzitutto usa l'API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) per convertire il file POTM in HTML. Successivamente, utilizzando l'API Python di Excel [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), salva l'HTML creato nel formato Microsoft Excel desiderato.
