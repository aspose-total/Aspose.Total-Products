{{% blocks/products/pf/agp/feature-section-col title="Come cercare nel file WORD tramite Python" %}}

1. API di riferimento all'interno del progetto direttamente da PyPI ([Aspose.Words](https://pypi.org/project/aspose-words/)) 
1. Crea un oggetto classe Document o carica il file WORD utilizzandolo.
1. Crea un oggetto classe DocumentBuilder e aggiungi testo nel file utilizzando il metodo DocumentBuilder.write().
1. Utilizza la funzione range.replace per trovare e sostituire il testo insieme a FindReplaceOptions.
1. Oppure usa range.replace_regex per l'espressione regolare insieme a replace.FindReplaceOptions.
1. Chiama il metodo "salva" per salvare il documento.
