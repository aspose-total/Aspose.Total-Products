{{% blocks/products/pf/feature-page-summary %}}
Utilizzando [Aspose.Total for .NET](https://products.aspose.com/total/net/) puoi includere la funzione di conversione da DOTX a XLTX all'interno di qualsiasi applicazione .NET, C#, ASP.NET e VB.NET in due semplici passaggi. Innanzitutto, utilizzando [Aspose.Words for .NET](https://products.aspose.com/words/net/), puoi esportare DOTX in HTML. Successivamente, utilizzando l'API di programmazione per fogli di calcolo [Aspose.Cells for .NET](https://products.aspose.com/cells/net/), è possibile convertire HTML in XLTX.

{{% blocks/products/pf/agp/feature-section-col title="API .NET per convertire DOTX in XLTX" %}}
1. Aprire il file DOTX utilizzando la classe [Document](https://apiference.aspose.com/words/net/aspose.words/Document)
2. Converti DOTX in HTML utilizzando il metodo [Save](https://apiference.aspose.com/words/net/aspose.words.Document/save/methods/4)
3. Caricare il Documento HTML utilizzando la classe [Workbook](https://apiference.aspose.com/cells/net/aspose.cells/workbook)
4. Salvare il Documento in formato XLTX utilizzando il metodo [Salva](https://apiference.aspose.com/cells/net/aspose.cells.workbook/save/methods/4) e impostare `XLTX` come SaveFormat

Durante la conversione di DOTX in XLTX, [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) consente di aggiungere proprietà personalizzate nei Documenti XLTX. Il metodo Add aggiunge la proprietà al file Excel e restituisce un riferimento per la nuova proprietà del Documento come [Aspose.Cells.Properties.DocumentProperty](https://apiference.aspose.com/cells/net/aspose.cells.properties /proprietà Documento).
