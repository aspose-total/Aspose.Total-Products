{{% blocks/products/pf/agp/feature-section-col title="API .NET per convertire WORDML in XLSM" %}}
1. Aprire il file WORDML utilizzando la classe [Document](https://apiference.aspose.com/words/net/aspose.words/document)
2. Converti WORDML in HTML utilizzando il metodo [Save](https://apiference.aspose.com/words/net/aspose.words.document/save/methods/4)

4. Salvare il documento in formato XLSM utilizzando il metodo [Salva](https://apiference.aspose.com/cells/net/aspose.cells.workbook/save/methods/4) e impostare `XLSM` come SaveFormat
