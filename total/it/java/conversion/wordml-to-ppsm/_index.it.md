Spesso gli sviluppatori devono convertire i file WORDML in PPSM a livello di codice. Utilizzando le librerie Java di File Automation [Aspose.Total for Java](https://products.aspose.com/total/java/) puoi automatizzare il processo di rendering in pochi semplici passaggi. Puoi caricare il tuo file WORDML utilizzando [Aspose.Words for Java](https://products.aspose.com/words/java/) e convertirlo in HTML. Successivamente, utilizzando la potente API Java per la manipolazione di PowerPoint [Aspose.Slides for Java](https://products.aspose.com/slides/java/) puoi creare una nuova presentazione, scrivere contenuto HTML al suo interno e salvarlo come PPSM .

{{% blocks/products/pf/agp/feature-section-col title="Come convertire WORDML in PPSM tramite Java" %}}
1. Aprire il file WORDML utilizzando la classe [Document](https://apiference.aspose.com/words/java/com.aspose.words/Document)
2. Converti file WORDML in HTML utilizzando [save](https://apiference.aspose.com/words/java/com.aspose.words/Document#save(java.lang.String,com.aspose.words.SaveOptions)) metodo
3. Inizializzare un nuovo oggetto [Presentation](https://apiference.aspose.com/slides/java/com.aspose.slides/Presentation)
5. Estrarre il contenuto dal file HTML utilizzando BufferedReader e scrivere il contenuto nel file di presentazione
6. Salvare il documento in PPSM utilizzando il metodo [save](https://apiference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)

In alternativa, puoi ottenere un file ZIP da [downloads](https://releases.aspose.com/total/java).
