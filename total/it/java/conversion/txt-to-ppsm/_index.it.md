{{% blocks/products/pf/agp/feature-section-col title="Come convertire TXT in PPSM tramite Java" %}}
1. Aprire il file TXT utilizzando la classe [Txtument](https://apiference.aspose.com/words/java/com.aspose.words/Txtument)

5. Estrarre il contenuto dal file HTML utilizzando BufferedReader e scrivere il contenuto nel file di presentazione
6. Salvare il txtumento in PPSM utilizzando il metodo [save](https://apiference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
