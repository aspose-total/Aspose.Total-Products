Utilizzando [Aspose.Total for C++](https://products.aspose.com/total/cpp/) puoi convertire il file ODP in formato JSON all'interno delle applicazioni C++ in due semplici passaggi. Innanzitutto, utilizzando [Aspose.Slides for C++](https://products.aspose.com/slides/cpp/), puoi esportare ODP in HTML.
