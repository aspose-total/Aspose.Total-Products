{{% blocks/products/pf/feature-page-section  h2="Converti DOTX protetto in formato JSON tramite C++" %}}
Utilizzando l'API, puoi anche aprire il dotxumento protetto da password. Se il dotxumento DOTX di input è protetto da password, non è possibile convertirlo in formato JSON senza utilizzare la password. A tale scopo, utilizzare un sovraccarico del costruttore speciale, che accetta un oggetto LoadOptions. Questo oggetto contiene la proprietà Password, che specifica la stringa della password.
