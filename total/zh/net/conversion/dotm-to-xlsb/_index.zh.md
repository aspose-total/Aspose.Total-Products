通过使用 [Aspose.Total for .NET](https://products.aspose.com/total/net/)，您可以在任何 .NET、C#、ASP.NET 和 VB.NET 应用程序中包含 DOTM 到 XLSB 的转换功能两个简单的步骤。首先，通过使用 [Aspose.Words for .NET](https://products.aspose.com/words/net/)，您可以将 DOTM 导出为 HTML。之后，通过使用 [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) 电子表格编程 API，您可以将 HTML 转换为 XLSB。

{{% blocks/products/pf/agp/feature-section-col title=".NET API 将 DOTM 转换为 XLSB" %}}
1.使用[Dotmument](https://reference.aspose.com/words/net/aspose.words/dotmument)类打开DOTM文件
2. 使用 [Save](https://reference.aspose.com/words/net/aspose.words.dotmument/save/methods/4) 方法将 DOTM 转换为 HTML
3. 使用 [Workbook](https://reference.aspose.com/cells/net/aspose.cells/workbook) 类加载 HTML 文档
4. 使用 [Save](https://reference.aspose.com/cells/net/aspose.cells.workbook/save/methods/4) 方法将文档保存为 XLSB 格式，并将 `XLSB` 设置为 SaveFormat
