您可以在 C++ 应用程序中轻松包含 WORDML 到 XLTX 的转换功能。通过使用功能丰富、功能强大且易于使用的文档操作和转换 API [Aspose.Words for C++](https://products.aspose.com/words/cpp/)，您可以将 WORDML 导出为 HTML。之后，通过使用 [Aspose.Cells for C++](https://products.aspose.com/cells/cpp/)，您可以将 HTML 转换为 XLTX。这两个 API 都属于 [Aspose.Total for C++](https://products.aspose.com/total/cpp/) 包。

{{% blocks/products/pf/agp/feature-section-col title="将 WORDML 转换为 XLTX 的 C++ API" %}}
1. 用[Wordmlument](https://reference.aspose.com/words/cpp/class/aspose.words.wordmlument)类参考打开WORDML文件

4. 使用 [Save](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook#a5dc7de23f7ceba76a05dc1d49f51502e) 成员函数将文档保存为 XLTX 格式
