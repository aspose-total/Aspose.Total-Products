5. 使用 BufferedReader 从 HTML 文件中提取内容并将内容写入您的演示文件
6. 使用 [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-) 方法将文档保存到 PPTM
