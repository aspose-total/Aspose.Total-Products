- 使用 MailMessage.load 类打开源 EMLX 文件
- 在指定输出 HTML 文件路径和相关的 HTML 保存选项作为参数时调用 `save` 方法。 所以你的 EMLX 文件在指定的路径被转换成 HTML

- 使用相关文件路径调用 save 方法。所以最后EMLX被转换了
