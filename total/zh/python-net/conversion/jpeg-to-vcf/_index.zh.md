{{% blocks/products/pf/agp/feature-section-col title="如何在 Python 中将 JPEG 转换为 VCF" %}}

- 使用 [Document](https://reference.aspose.com/words/python-net/aspose.words/document/) 类打开源 JPEG 文件
- 调用 `save` 方法，同时指定输出 HTML 文件路径和相关的 HTML Save 选项作为参数。 所以你的 JPEG 文件在指定的路径被转换成 HTML

- 使用相关文件路径调用 save 方法。所以最后JPEG被转换了
