- 使用 MailMessage.load 类打开源 EMAIL 文件
- 在指定输出 HTML 文件路径和相关的 HTML 保存选项作为参数时调用 `save` 方法。 所以你的 EMAIL 文件在指定的路径被转换成 HTML
- 现在使用 [Document](https://reference.aspose.com/words/python-net/aspose.words/document/) 加载保存的 HTML 文件
- 使用相关文件路径调用 save 方法。所以最后EMAIL被转换了
