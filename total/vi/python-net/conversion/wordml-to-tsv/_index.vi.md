Đối với nhà phát triển Python, người đang cố gắng thêm tính năng chuyển đổi WORDML sang TSV trong ứng dụng. [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API có thể giúp tự động hóa quá trình chuyển đổi. Đó là một gói đầy đủ các API khác nhau xử lý các định dạng khác nhau.

Nó chủ yếu trong hai bước. Trước tiên, hãy sử dụng API [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) để chuyển đổi tệp WORDML sang HTML. Sau đó, bằng cách sử dụng Excel Python API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), hãy lưu HTML đã tạo thành định dạng Microsoft Excel mong muốn.

- **Bước 1** Mở tệp WORDML nguồn bằng lớp [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)

-  **Bước 2** Tải tệp HTML bằng một phiên bản của lớp Workbook với tệp và LoadOptions làm tham số
-  Gọi phương thức `save` trong khi chỉ định đường dẫn tệp TSV đầu ra. Vì vậy, tệp WORDML của bạn được chuyển đổi thành TSV theo đường dẫn đã chỉ định

{{% blocks/products/pf/agp/feature-section-col title="Yêu cầu chuyển đổi" %}}

- Để chuyển đổi WORDML sang TSV, cần có Python 3.5 trở lên
