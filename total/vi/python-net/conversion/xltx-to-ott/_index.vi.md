Đối với nhà phát triển Python, người đang cố gắng thêm tính năng chuyển đổi XLTX sang OTT trong ứng dụng. Nó chủ yếu trong hai bước. Đầu tiên sử dụng API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) để chuyển đổi tệp XLTX sang HTML.
