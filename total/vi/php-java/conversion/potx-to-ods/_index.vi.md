Aspose.Total for PHP via Java là một bộ toàn diện cung cấp nhiều tính năng để làm việc với bảng tính Excel, bản trình bày Powerpoint cũng như tạo và nhận dạng mã vạch. Với Aspose.Total cho PHP thông qua Java, bạn có thể dễ dàng thêm chuyển đổi POTX sang ODS vào bất kỳ ứng dụng PHP nào chỉ trong vài bước.

Đầu tiên, sử dụng Aspose.Slides cho PHP thông qua Java để xuất tệp POTX sang HTML. Sau đó, sử dụng Aspose.Cells cho PHP thông qua Java để chuyển đổi tệp HTML thành ODS. Cả Aspose.Cells cho PHP qua Java và Aspose.Slides cho PHP qua Java đều là các API con của gói API Aspose.Total cho PHP thông qua Java.
