Untuk pengembang Python, yang mencoba menambahkan fitur konversi POWERPOINT ke CSV di dalam aplikasi. API [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) dapat membantu mengotomatiskan proses konversi. Ini adalah paket lengkap dari berbagai API yang menangani berbagai format termasuk file POWERPOINT dan CSV.

Ini terutama dalam dua langkah. Pertama gunakan API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) untuk mengonversi file POWERPOINT ke HTML. Setelah itu dengan menggunakan Excel Python API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), simpan HTML yang telah dibuat ke dalam format Microsoft Excel yang diinginkan.

{{% blocks/products/pf/agp/feature-section-col title="Cara Mengonversi POWERPOINT ke CSV dengan Python" %}}
- **Langkah 1** Gunakan instance kelas [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/) untuk membuka file POWERPOINT sumber

-  **Langkah 2** Muat file HTML dengan instance kelas Workbook
-  Panggil metode `save` sambil menentukan jalur file CSV keluaran. Jadi file POWERPOINT Anda dikonversi ke CSV di jalur yang ditentukan
