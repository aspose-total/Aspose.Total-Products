- **Langkah 1** Buka file DOCM sumber menggunakan kelas [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)

-  **Langkah 2** Muat file HTML dengan instance kelas Workbook dengan file dan LoadOptions sebagai parameter
-  Panggil metode `save` sambil menentukan jalur file EXCEL keluaran. Jadi file DOCM Anda dikonversi ke EXCEL di jalur yang ditentukan
