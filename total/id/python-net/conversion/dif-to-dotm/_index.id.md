Untuk pengembang Python, yang mencoba menambahkan fitur konversi DIF ke DOTM di dalam aplikasi. API [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) dapat membantu mengotomatiskan proses konversi. Ini adalah paket lengkap dari berbagai API yang menangani berbagai format termasuk file DIF dan DOTM.

Ini terutama dalam dua langkah. Pertama gunakan API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) untuk mengonversi file DIF ke HTML. Setelah itu dengan menggunakan Word Python API [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/), simpan HTML yang telah dibuat ke dalam format Microsoft Word yang diinginkan.

{{% blocks/products/pf/agp/feature-section-col title="Cara Mengonversi DIF ke DOTM dengan Python" %}}
- **Langkah 1** Buka file DIF sumber menggunakan kelas Workbook
- Simpan file DIF ke HTML dengan menggunakan metode save(file, SaveFormat.HTML) dengan memberikan nama file dan jalur direktori yang diinginkan
-  **Langkah 2** Muat file HTML dengan instance kelas [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)
-  Panggil metode `save` sambil menentukan jalur file DOTM keluaran. Jadi file DIF Anda dikonversi ke DOTM di jalur yang ditentukan

- Untuk konversi DIF ke DOTM, diperlukan Python 3.5 atau lebih baru
