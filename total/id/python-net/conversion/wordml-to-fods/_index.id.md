- **Langkah 1** Buka file WORDML sumber menggunakan kelas [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)

-  Panggil metode `save` sambil menentukan jalur file FODS keluaran. Jadi file WORDML Anda dikonversi ke FODS di jalur yang ditentukan
