Apakah Anda seorang pengembang Android yang ingin menambahkan fitur konversi dokumen ke aplikasi Anda? [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) Pustaka Otomatisasi Format File dapat membantu Anda mengotomatiskan proses konversi dokumen di aplikasi Android Anda. Untuk mengonversi file WORD ke POT, pertama-tama Anda dapat menggunakan API manipulasi dokumen [Aspose.Words for Android Java](https://products.aspose.com/words/android-java/) untuk mengonversi format file WORD ke HTML. Setelah itu dengan menggunakan PowerPoint API [Aspose.Slides for Android Java](https://products.aspose.com/slides/android-java/), Anda dapat membuat Presentasi baru, menulis konten HTML di dalamnya, dan menyimpannya sebagai POT.

{{% blocks/products/pf/agp/feature-section-col title="Cara Mengonversi WORD ke POT di Android" %}}
1. Buka file WORD menggunakan kelas [Wordument](https://reference.aspose.com/words/java/com.aspose.words/Wordument)
2. Konversi file WORD ke HTML dengan menggunakan [save](https://reference.aspose.com/words/java/com.aspose.words/Wordument#save(java.lang.String,com.aspose.words.SaveOptions )) metode
3. Inisialisasi objek [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation) baru
5. Ekstrak konten dari file HTML menggunakan BufferedReader dan tulis konten di file presentasi Anda
6. Simpan dokumen ke POT menggunakan metode [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
