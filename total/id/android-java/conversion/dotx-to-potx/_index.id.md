{{% blocks/products/pf/agp/feature-section-col title="Cara Mengonversi DOTX ke POTX di Android" %}}
1. Buka file DOTX menggunakan kelas [Dotxument](https://reference.aspose.com/words/java/com.aspose.words/Dotxument)
2. Konversi file DOTX ke HTML dengan menggunakan [save](https://reference.aspose.com/words/java/com.aspose.words/Dotxument#save(java.lang.String,com.aspose.words.SaveOptions )) metode
3. Inisialisasi objek [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation) baru
5. Ekstrak konten dari file HTML menggunakan BufferedReader dan tulis konten di file presentasi Anda
6. Simpan dokumen ke POTX menggunakan metode [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
