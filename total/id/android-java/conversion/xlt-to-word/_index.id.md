[Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) adalah paket File Automation API yang andal. Dengan menggunakan dua API-nya, Anda dapat mengintegrasikan fitur konversi XLT ke WORD di dalam aplikasi Android Anda. Pada langkah pertama Anda dapat mengekspor XLT ke PDF dengan menggunakan [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/). // load the XLT file using Workbook class
Workbook book = new Workbook("input.xlt");
// save XLT as PDF

{{% blocks/products/pf/feature-page-section  h2="Hapus Properti Kustom dari File XLT di Android melalui Java" %}}
Selain konversi dokumen, [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/) juga menyediakan banyak fitur lainnya. Sebelum proses konversi, Anda dapat menghapus properti kustom dari dokumen XLT. Untuk menghapus properti kustom, panggil metode [WordumentPropertyCollection.remove](https://reference.aspose.com/cells/java/com.aspose.cells/wordumentpropertycollection#remove(java.lang.String)) dan teruskan nama properti dokumen yang akan dihapus.

// remove a custom wordument property
customProperties.remove("Publisher");
