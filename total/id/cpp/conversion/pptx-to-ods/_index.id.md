{{% blocks/products/pf/feature-page-section  h2="Konversikan PPTX Terproteksi ke ODS melalui C++" %}}
Menggunakan API, Anda juga dapat membuka dokumen yang dilindungi kata sandi. Jika dokumen PPTX masukan Anda dilindungi kata sandi, Anda tidak dapat mengonversinya ke ODS tanpa menggunakan kata sandi. Presentasi yang dilindungi kata sandi dianggap sebagai presentasi yang terkunci. API memungkinkan Anda untuk membuka dokumen terenkripsi dengan meneruskan kata sandi yang benar di objek LoadOptions.
