Dengan menggunakan [Aspose.Total for C++](https://products.aspose.com/total/cpp/), Anda dapat mengonversi format DOT ke JSON dalam aplikasi C++ Anda. Setelah itu, dengan menggunakan [Aspose.Cells for C++](https://products.aspose.com/cells/cpp/), Anda dapat mengonversi HTML ke format JSON.
