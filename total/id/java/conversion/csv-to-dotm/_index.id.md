{{% blocks/products/pf/agp/feature-section-col title="Cara Mengonversi CSV ke DOTM melalui Java API" %}}
1. Buka file CSV menggunakan kelas [Buku Kerja](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)
2. Konversi CSV ke PDF dan atur SaveFormat ke AUTO
3. Muat file PDF yang dikonversi menggunakan kelas [Document](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document)
4. Simpan dokumen ke format DOTM menggunakan [save](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document#save-java.lang.String-com.aspose.pdf.SaveOptions-) metode dan atur Docx sebagai SaveFormat
