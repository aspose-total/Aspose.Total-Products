{{% blocks/products/pf/agp/feature-section-col title="Cara Mengonversi DOT ke POT melalui Java" %}}
1. Buka file DOT menggunakan kelas [Document](https://reference.aspose.com/words/java/com.aspose.words/Document)
2. Konversi file DOT ke HTML dengan menggunakan [save](https://reference.aspose.com/words/java/com.aspose.words/Document#save(java.lang.String,com.aspose.words.SaveOptions)) metode

5. Ekstrak konten dari file HTML menggunakan BufferedReader dan tulis konten di file presentasi Anda
6. Simpan dokumen ke POT menggunakan metode [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
