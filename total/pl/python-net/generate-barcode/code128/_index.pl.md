{{% blocks/products/pf/feature-page-summary %}}

Dla programisty, który próbuje wygenerować niestandardowe obrazy kodów kreskowych CODE128 za pomocą aplikacji Python, [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API może pomóc zautomatyzować proces tworzenia kodów kreskowych w dowolnym kolorze lub rozmiarze zgodnie z potrzebami biznesowymi. Jest to pełny pakiet różnych interfejsów API obsługujących różne formaty, a także kody kreskowe 1D, 2D i pocztowe. [Aspose.BarCode for Python via .NET](https://products.aspose.com/barcode/python-net/) API, które jest częścią pakietu [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/), ułatwia ten proces generowania. Ponadto programiści mogą łatwo dostosować i dostosować wygląd wszystkich elementów kodu kreskowego, w tym kolory pasków, obramowań, tła i podpisów tekstowych, a także ich położenie.

{{% blocks/products/pf/agp/feature-section-col title="Jak wygenerować kod kreskowy CODE128 w Pythonie" %}}

- Użyj obiektu klasy BarcodeGenerator z odpowiednim [typy kodowania](https://docs.aspose.com/barcode/python-net/api-reference/aspose.barcode.generation/#enumerations)
- Ustaw odpowiednie parametry, takie jak lokalizacja i wymiary
- Zapisz przy użyciu metody save(), przekazując odpowiednią ścieżkę obrazu i typ, taki jak BarCodeImageFormat.PNG

{{% blocks/products/pf/agp/feature-section-col title="Wymagania dotyczące tworzenia" %}}

- Do tworzenia kodów kreskowych CODE128 wymagany jest Python 3.5 lub nowszy
- Odwołaj się do API w ramach projektu bezpośrednio z PyPI ([Aspose.BarCode](https://pypi.org/project/aspose-barcode-for-python-via-net/)) 
- Lub Użyj następujących poleceń pip ```pip install aspose-barcode-for-python-via-net``` 
- Ponadto system operacyjny oparty na Microsoft Windows lub Linux (zobacz więcej dla [BarCode](https://docs.aspose.com/barcode/python-net/system-requirements/))
