Dla programisty Pythona, który próbuje dodać funkcję konwersji CSV do POTX w aplikacji, [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API może pomóc zautomatyzować proces konwersji. Jest to pełny pakiet różnych interfejsów API obsługujących różne formaty, w tym pliki CSV i POTX.

To głównie w dwóch krokach. Najpierw użyj API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), aby przekonwertować plik CSV na PDF. Następnie za pomocą PowerPoint Python API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) zapisz utworzony plik PDF w żądanym formacie Microsoft PowerPoint.

- **Krok 1** Użyj instancji klasy Workbook, aby otworzyć źródłowy plik CSV

-  Wywołaj metodę [save](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/), określając ścieżkę wyjściowego pliku POTX. Więc twój plik CSV jest konwertowany na POTX w określonej ścieżce

{{% blocks/products/pf/agp/feature-section-col title="Wymagania konwersji" %}}

- Do konwersji CSV na POTX wymagany jest Python 3.5 lub nowszy
