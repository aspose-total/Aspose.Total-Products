Dla programisty Pythona, który próbuje dodać funkcję konwersji POWERPOINT do EXCEL w aplikacji. [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API może pomóc zautomatyzować proces konwersji. Jest to pełny pakiet różnych interfejsów API obsługujących różne formaty, w tym pliki POWERPOINT i EXCEL.

To głównie w dwóch krokach. Najpierw użyj interfejsu API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/), aby przekonwertować plik POWERPOINT na HTML. Następnie za pomocą Excel Python API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) zapisz utworzony kod HTML w żądanym formacie Microsoft Excel.

{{% blocks/products/pf/agp/feature-section-col title="Jak przekonwertować POWERPOINT na EXCEL w Pythonie" %}}
- **Krok 1** Użyj instancji klasy [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/), aby otworzyć źródłowy plik POWERPOINT

-  Wywołaj metodę `save` podczas określania wyjściowej ścieżki pliku EXCEL. Więc twój plik POWERPOINT jest konwertowany na EXCEL w określonej ścieżce

{{% blocks/products/pf/agp/feature-section-col title="Wymagania konwersji" %}}

- Do konwersji POWERPOINT na EXCEL wymagany jest Python 3.5 lub nowszy
