Dla programisty Pythona, który próbuje dodać funkcję konwersji XLSB do PPT w aplikacji, [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API może pomóc zautomatyzować proces konwersji. Jest to pełny pakiet różnych interfejsów API obsługujących różne formaty, w tym pliki XLSB i PPT.

To głównie w dwóch krokach. Najpierw użyj API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), aby przekonwertować plik XLSB na PDF. Następnie za pomocą PowerPoint Python API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) zapisz utworzony plik PDF w żądanym formacie Microsoft PowerPoint.

- **Krok 1** Użyj instancji klasy Workbook, aby otworzyć źródłowy plik XLSB

-  Wywołaj metodę [save](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/), określając ścieżkę wyjściowego pliku PPT. Więc twój plik XLSB jest konwertowany na PPT w określonej ścieżce

{{% blocks/products/pf/agp/feature-section-col title="Wymagania konwersji" %}}

- Do konwersji XLSB na PPT wymagany jest Python 3.5 lub nowszy
