Dla programisty Pythona, który próbuje dodać funkcję konwersji TSV do DOT w aplikacji. Jest to pełny pakiet różnych interfejsów API obsługujących różne formaty, w tym pliki TSV i DOT.

To głównie w dwóch krokach. Najpierw użyj interfejsu API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), aby przekonwertować plik TSV na HTML. Następnie za pomocą Word Python API [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) zapisz utworzony kod HTML w żądanym formacie Microsoft Word.

- **Krok 1** Otwórz źródłowy plik TSV przy użyciu klasy Workbook
- Zapisz plik TSV do formatu HTML, używając metody save(file, SaveFormat.HTML), podając nazwę pliku i żądaną ścieżkę do katalogu

-  Wywołaj metodę `save` podczas określania wyjściowej ścieżki pliku DOT. Więc twój plik TSV jest konwertowany na DOT w określonej ścieżce
