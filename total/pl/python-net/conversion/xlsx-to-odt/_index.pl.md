To głównie w dwóch krokach. Najpierw użyj interfejsu API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), aby przekonwertować plik XLSX na HTML. Następnie za pomocą Word Python API [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) zapisz utworzony kod HTML w żądanym formacie Microsoft Word.

- **Krok 1** Otwórz źródłowy plik XLSX przy użyciu klasy Workbook

-  **Krok 2** Załaduj plik HTML z instancją klasy [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)
-  Wywołaj metodę `save` podczas określania wyjściowej ścieżki pliku ODT. Więc twój plik XLSX jest konwertowany na ODT w określonej ścieżce
