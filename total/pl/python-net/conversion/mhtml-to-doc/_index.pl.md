Dla programisty Pythona, kto próbuje dodać funkcję konwersji MHTML i HtmlFixed do Microsft Word DOC w aplikacji? API [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) może pomóc zautomatyzować proces konwersji. Jest to pełny pakiet różnych interfejsów API obsługujących różne formaty, w tym formaty Microsoft Word, archiwum internetowe MHTML, HTML i format HtmlFixed do zapisywania dokumentu w formacie HTML przy użyciu absolutnie pozycjonowanych elementów. [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) API, które jest częścią pakietu [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/), ułatwia eksportowanie i importowanie MHTML, PDF i różnych innych formatów przy użyciu Pythona.

{{% blocks/products/pf/agp/feature-section-col title="Jak przekonwertować MHTML na DOC w Pythonie?" %}}

- Otwórz źródłowy plik MHTML za pomocą klasy [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)
