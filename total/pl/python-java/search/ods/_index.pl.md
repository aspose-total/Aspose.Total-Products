{{% blocks/products/pf/agp/feature-section-col title="Jak przeszukać plik ODS za pomocą Pythona" %}}

1. Referencyjne interfejsy API w projekcie bezpośrednio z PyPI ([Aspose.Cells](https://pypi.org/project/aspose-cells/)) 
1. Załaduj plik ODS
1. Utwórz opcje zamiany.
1. Ustaw opcje uwzględniania wielkości liter i dopasowywania tekstu.
1. Lub Ustaw setRegexKey na true, aby wskazać, że wyszukiwanym kluczem jest wyrażenie regularne.
1. Zastąp tekst za pomocą funkcji zamiany.

<h2>Opracuj wyszukiwarkę plików ODS za pomocą Pythona</h2>

Chcesz opracować aplikację lub narzędzie z możliwością wyszukiwania ODS? Dzięki Aspose.Cells for Python via Java, podrzędnemu API Aspose.Total for Python via Java, każdy programista Pythona może zintegrować powyższy kod API w celu wyszukiwania programów w dokumentach.
