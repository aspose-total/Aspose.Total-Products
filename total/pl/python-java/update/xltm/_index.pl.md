{{% blocks/products/pf/agp/feature-section-col title="Jak zaktualizować plik XLTM w Pythonie?" %}}

- Utwórz nowy obiekt klasy [Workbook](https://reference.aspose.com/cells/python-java/asposecells.api/Workbook) mając jako parametr źródłowy plik XLTM
- Dostęp do odpowiedniego Arkusza przy użyciu metody [getWorksheets().get(index)](https://reference.aspose.com/cells/python/asposecells.api/workbook#Worksheets)
- Wstaw nowe dane do wybranej komórki za pomocą metody [Worksheet.getCells().get(indexValue).putValue()](https://reference.aspose.com/cells/python/asposecells.api/worksheet#Cells)
- Zapisz plik jako plik .xltm za pomocą metody save() przekazując plik ze ścieżką jako parametrem
