{{% blocks/products/pf/feature-page-section  h2="Konwertuj chroniony POTM na XLTM za pomocą C++" %}}
Korzystając z interfejsu API, możesz również otworzyć dokument chroniony hasłem. Jeśli wejściowy dokument POTM jest chroniony hasłem, nie można go przekonwertować na XLTM bez użycia hasła. Gdy dokument jest chroniony hasłem, oznacza to, że wymusza pewne ograniczenia prezentacji. Aby usunąć ograniczenia, należy wprowadzić hasło. Prezentacja chroniona hasłem jest uważana za prezentację zablokowaną. API umożliwia otwarcie zaszyfrowanego dokumentu poprzez podanie prawidłowego hasła w obiekcie LoadOptions.
