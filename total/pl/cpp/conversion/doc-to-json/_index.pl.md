Używając [Aspose.Total for C++](https://products.aspose.com/total/cpp/) możesz przekonwertować DOC na format JSON w swoich aplikacjach C++. Po pierwsze, używając [Aspose.Words for C++](https://products.aspose.com/words/cpp/), możesz wyeksportować DOC do HTML. Następnie, używając [Aspose.Cells for C++](https://products.aspose.com/cells/cpp/), możesz przekonwertować HTML na format JSON.

{{% blocks/products/pf/agp/feature-section-col title="Konwertuj DOC na format JSON za pomocą C++" %}}
1. Otwórz plik DOC, korzystając z odwołania do klasy [Document](https://reference.aspose.com/words/cpp/class/aspose.words.document)
2. Konwertuj DOC na HTML za pomocą funkcji członkowskiej [Save](https://reference.aspose.com/words/cpp/class/aspose.words.document#save_string_saveformat)
3. Załaduj dokument HTML, używając odwołania do klasy [IWorkbook](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook)
4. Zapisz dokument w formacie JSON za pomocą funkcji członkowskiej [Save](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook#a9460f52a2dec8f4bf623a4905167d997)
