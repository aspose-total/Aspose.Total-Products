{{% blocks/products/pf/feature-page-summary %}}
Możesz zintegrować funkcję konwersji MD na XLTX w swoich aplikacjach na Androida w dwuetapowym procesie. Po pierwsze, używając [Aspose.PDF dla Androida przez Javę](https://products.aspose.com/pdf/android-java/) możesz zamienić MD na XLSX. Po drugie, możesz przekonwertować XLSX na XLTX, korzystając z interfejsu API Powerful Spreadsheet Processing [Aspose.Cells na Androida przez Javę](https://products.aspose.com/cells/android-java/). Oba interfejsy API należą do rodziny produktów [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/).

1. Otwórz plik MD za pomocą klasy [Document](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document)

4. Zapisz dokument w formacie XLTX, używając [save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.SaveOptions)) metoda
