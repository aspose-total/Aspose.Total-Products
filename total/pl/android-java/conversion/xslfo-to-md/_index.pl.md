Możesz zintegrować funkcję konwersji XSLFO na MD w swoich aplikacjach na Androida w dwuetapowym procesie. Po pierwsze, używając [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/) możesz zamienić XSLFO na XLSX. Po drugie, możesz przekonwertować XLSX na MD, korzystając z interfejsu API Powerful Spreadsheet Processing [Aspose.Cells na Androida przez Javę](https://products.aspose.com/cells/android-java/). Oba interfejsy API należą do rodziny produktów [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/).
