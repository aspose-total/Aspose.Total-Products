<h2>Opracuj aplikację do rozdzielania plików PDF przy użyciu platformy .NET</h2>

Chcesz opracować skrypt .NET lub aplikację narzędziową, aby łatwo dzielić pliki PDF? Dzięki [Aspose.PDF for .NET](https://products.aspose.com/pdf/pl/net/), podrzędnemu API [Aspose.Total for .NET](https://products.aspose.com/total/pl/net/), każdy programista .NET może zintegrować powyższy kod API z aplikacją rozdzielającą programy.Potężna biblioteka .NET do dzielenia dokumentów obsługuje wiele popularnych formatów, w tym format PDF, taki jak PDF, dokumenty Microsoft Word, arkusze kalkulacyjne Excel i prezentacje PowerPoint.<br /><br />
