<h2>Opracuj aplikację konwersji plików PDF przy użyciu .NET</h2>

Musisz opracować aplikację bazującą na technologii .NET, która umożliwi łatwe zapisywanie i eksportowanie plików PDF do dokumentu XLTX? Dzięki [Aspose.Total for .NET](https://products.aspose.com/total/pl/net/) każdy programista .NET może zintegrować powyższy kod API w celu zaprogramowania aplikacji konwertującej w różnych formatach, w tym Microsoft Word, Excel, PowerPoint, PDF, plikach e-mail, obrazach i innych formatach.
