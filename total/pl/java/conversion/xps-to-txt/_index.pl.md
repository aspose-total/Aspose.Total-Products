Używając [Aspose.Total for Java](https://products.aspose.com/total/java/) możesz zintegrować funkcję konwersji XPS na TXT w swoich aplikacjach Java w dwuetapowym procesie. W drugim kroku możesz przekonwertować XLSX na TXT, korzystając z interfejsu API do programowania arkuszy kalkulacyjnych [Aspose.Cells for Java](https://products.aspose.com/cells/java/).
