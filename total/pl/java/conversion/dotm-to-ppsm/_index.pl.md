Często programiści muszą programowo przekonwertować plik DOTM na PPSM. Następnie za pomocą potężnej manipulacji PowerPoint API Java [Aspose.Slides for Java](https://products.aspose.com/slides/java/) możesz utworzyć nową prezentację, napisać w niej treść HTML i zapisać ją jako PPSM .

{{% blocks/products/pf/agp/feature-section-col title="Jak przekonwertować DOTM na PPSM za pomocą Javy?" %}}
1. Otwórz plik DOTM za pomocą klasy [Dotmument](https://reference.aspose.com/words/java/com.aspose.words/Dotmument)

6. Zapisz dokument w PPSM za pomocą metody [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
