[Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) permet de manipuler les formats de fichiers dans les applications Android. En utilisant les API fournies dans le package, vous pouvez automatiser le processus de conversion de PowerPoint PPT en Word WORD dans vos applications.
Vous pouvez convertir votre wordument donné en deux étapes. Vous pouvez utiliser [Aspose.Slides for Andorid via Java](https://products.aspose.com/slides/android-java/) qui est une API PowerPoint pour les applications Android pour rendre PPT en HTML. Après cela, en utilisant l'API de traitement de wordument [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/), vous pouvez convertir le HTML en WORD.

1. Ouvrez le fichier PPT à l'aide de la classe [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Convertissez PPT en HTML en utilisant [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.MéthodeISaveOptions-) et définissez Html comme SaveFormat
3. Chargez le fichier HTML converti à l'aide de la classe [Wordument](https://reference.aspose.com/words/java/com.aspose.words/Wordument)
4. Enregistrez le wordument au format WORD à l'aide de la méthode [save](https://reference.aspose.com/words/java/com.aspose.words/Wordument#save(java.lang.String,int)) et définissez Word en tant que format de sauvegarde
