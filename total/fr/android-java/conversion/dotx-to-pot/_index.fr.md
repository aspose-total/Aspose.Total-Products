Êtes-vous un développeur Android qui souhaite ajouter une fonctionnalité de conversion de dotxuments à votre application ? [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) Les bibliothèques d'automatisation du format de fichier peuvent vous aider à automatiser le processus de conversion de dotxuments dans vos applications Android. Afin de convertir un fichier DOTX en POT, vous pouvez d'abord utiliser l'API de manipulation de dotxument [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/) pour convertir le format de fichier DOTX en HTML. Après cela, en utilisant l'API PowerPoint [Aspose.Slides for Android via Java](https://products.aspose.com/slides/android-java/), vous pouvez créer une nouvelle présentation, y écrire du contenu HTML et l'enregistrer sous POT.

{{% blocks/products/pf/agp/feature-section-col title="Comment convertir DOTX en POT sur Android" %}}
1. Ouvrez le fichier DOTX à l'aide de la classe [Dotxument](https://reference.aspose.com/words/java/com.aspose.words/Dotxument)
2. Convertissez le fichier DOTX en HTML en utilisant [save](https://reference.aspose.com/words/java/com.aspose.words/Dotxument#save(java.lang.String,com.aspose.words.SaveOptions )) méthode
3. Initialisez un nouvel objet [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
5. Extrayez le contenu du fichier HTML à l'aide de BufferedReader et écrivez le contenu dans votre fichier de présentation
6. Enregistrez le dotxument dans POT à l'aide de la méthode [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
