{{% blocks/products/pf/feature-page-section  h2="Supprimer les informations inutilisées d'un wordument WORD dans Android via Java" %}}
Avant de convertir WORD en ODS, vous pouvez supprimer les informations inutilisées du wordument WORD via [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/). Parfois, vous devrez peut-être supprimer des informations inutilisées ou en double pour réduire la taille du wordument de sortie et le temps de traitement. La classe [CleanupOptions](https://reference.aspose.com/words/java/com.aspose.words/CleanupOptions) vous permet de spécifier des options pour le nettoyage des worduments. Pour supprimer les styles en double ou uniquement les styles ou listes inutilisés du wordument, vous pouvez utiliser la méthode [Cleanup](https://reference.aspose.com/words/java/com.aspose.words/Wordument#cleanup()). Vous pouvez utiliser les [UnusedStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedStyles) et [UnusedBuiltinStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedBuiltinStyles) pour détecter et supprimer les styles marqués comme « inutilisés ».
