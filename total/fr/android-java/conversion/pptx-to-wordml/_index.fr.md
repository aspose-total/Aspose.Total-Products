Vous pouvez convertir votre wordmlument donné en deux étapes. Vous pouvez utiliser [Aspose.Slides for Andorid via Java](https://products.aspose.com/slides/android-java/) qui est une API PowerPoint pour les applications Android pour rendre PPTX en HTML. 4. Enregistrez le wordmlument au format WORDML à l'aide de la méthode [save](https://reference.aspose.com/words/java/com.aspose.words/Wordmlument#save(java.lang.String,int)) et définissez Wordml en tant que format de sauvegarde
