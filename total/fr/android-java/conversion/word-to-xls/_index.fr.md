En utilisant [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/), vous pouvez intégrer la fonction de conversion WORD en XLS dans vos applications Android. Premièrement, vous pouvez convertir WORD en HTML en utilisant une API de manipulation et de conversion de worduments riche en fonctionnalités [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/). {{% blocks/products/pf/feature-page-section  h2="Supprimer les informations inutilisées d'un wordument WORD dans Android via Java" %}}
Avant de convertir WORD en XLS, vous pouvez supprimer les informations inutilisées du wordument WORD via [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/). Parfois, vous devrez peut-être supprimer des informations inutilisées ou en double pour réduire la taille du wordument de sortie et le temps de traitement. La classe [CleanupOptions](https://reference.aspose.com/words/java/com.aspose.words/CleanupOptions) vous permet de spécifier des options pour le nettoyage des worduments. Pour supprimer les styles en double ou uniquement les styles ou listes inutilisés du wordument, vous pouvez utiliser la méthode [Cleanup](https://reference.aspose.com/words/java/com.aspose.words/Wordument#cleanup()). Vous pouvez utiliser les [UnusedStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedStyles) et [UnusedBuiltinStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedBuiltinStyles) pour détecter et supprimer les styles marqués comme « inutilisés ».
