[Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) permet de manipuler les formats de fichiers dans les applications Android. En utilisant les API fournies dans le package, vous pouvez automatiser le processus de conversion de PowerPoint PPSX en Word TEXT dans vos applications.
Vous pouvez convertir votre textument donné en deux étapes. Vous pouvez utiliser [Aspose.Slides for Andorid via Java](https://products.aspose.com/slides/android-java/) qui est une API PowerPoint pour les applications Android pour rendre PPSX en HTML. Après cela, en utilisant l'API de traitement de textument [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/), vous pouvez convertir le HTML en TEXT.

{{% blocks/products/pf/agp/feature-section-col title="Rendu PPSX vers TEXT sous Android" %}}
1. Ouvrez le fichier PPSX à l'aide de la classe [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Convertissez PPSX en HTML en utilisant [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.MéthodeISaveOptions-) et définissez Html comme SaveFormat
3. Chargez le fichier HTML converti à l'aide de la classe [Textument](https://reference.aspose.com/words/java/com.aspose.words/Textument)
4. Enregistrez le textument au format TEXT à l'aide de la méthode [save](https://reference.aspose.com/words/java/com.aspose.words/Textument#save(java.lang.String,int)) et définissez Text en tant que format de sauvegarde
