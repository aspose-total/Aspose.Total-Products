Vous pouvez utiliser [Aspose.Slides for Andorid via Java](https://products.aspose.com/slides/android-java/) qui est une API PowerPoint pour les applications Android pour rendre PPSM en HTML. 1. Ouvrez le fichier PPSM à l'aide de la classe [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Convertissez PPSM en HTML en utilisant [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.MéthodeISaveOptions-) et définissez Html comme SaveFormat

4. Enregistrez le dotxument au format DOTX à l'aide de la méthode [save](https://reference.aspose.com/words/java/com.aspose.words/Dotxument#save(java.lang.String,int)) et définissez Dotx en tant que format de sauvegarde
