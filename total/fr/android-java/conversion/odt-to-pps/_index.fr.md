Êtes-vous un développeur Android qui souhaite ajouter une fonctionnalité de conversion de odtuments à votre application ? [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) Les bibliothèques d'automatisation du format de fichier peuvent vous aider à automatiser le processus de conversion de odtuments dans vos applications Android. Afin de convertir un fichier ODT en PPS, vous pouvez d'abord utiliser l'API de manipulation de odtument [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/) pour convertir le format de fichier ODT en HTML. Après cela, en utilisant l'API PowerPoint [Aspose.Slides for Android via Java](https://products.aspose.com/slides/android-java/), vous pouvez créer une nouvelle présentation, y écrire du contenu HTML et l'enregistrer sous PPS.

{{% blocks/products/pf/agp/feature-section-col title="Comment convertir ODT en PPS sur Android" %}}
1. Ouvrez le fichier ODT à l'aide de la classe [Odtument](https://reference.aspose.com/words/java/com.aspose.words/Odtument)
2. Convertissez le fichier ODT en HTML en utilisant [save](https://reference.aspose.com/words/java/com.aspose.words/Odtument#save(java.lang.String,com.aspose.words.SaveOptions )) méthode
3. Initialisez un nouvel objet [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
5. Extrayez le contenu du fichier HTML à l'aide de BufferedReader et écrivez le contenu dans votre fichier de présentation
6. Enregistrez le odtument dans PPS à l'aide de la méthode [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
