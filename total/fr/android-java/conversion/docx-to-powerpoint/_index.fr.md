{{% blocks/products/pf/feature-page-summary %}}
Êtes-vous un développeur Android qui souhaite ajouter une fonctionnalité de conversion de docxuments à votre application ? [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) Les bibliothèques d'automatisation du format de fichier peuvent vous aider à automatiser le processus de conversion de docxuments dans vos applications Android. Afin de convertir un fichier DOCX en POWERPOINT, vous pouvez d'abord utiliser l'API de manipulation de docxument [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/) pour convertir le format de fichier DOCX en HTML. Après cela, en utilisant l'API PowerPoint [Aspose.Slides for Android via Java](https://products.aspose.com/slides/android-java/), vous pouvez créer une nouvelle présentation, y écrire du contenu HTML et l'enregistrer sous POWERPOINT.

{{% blocks/products/pf/agp/feature-section-col title="Comment convertir DOCX en POWERPOINT sur Android" %}}
1. Ouvrez le fichier DOCX à l'aide de la classe [Docxument](https://reference.aspose.com/words/java/com.aspose.words/Docxument)
2. Convertissez le fichier DOCX en HTML en utilisant [save](https://reference.aspose.com/words/java/com.aspose.words/Docxument#save(java.lang.String,com.aspose.words.SaveOptions )) méthode
3. Initialisez un nouvel objet [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
5. Extrayez le contenu du fichier HTML à l'aide de BufferedReader et écrivez le contenu dans votre fichier de présentation
6. Enregistrez le docxument dans POWERPOINT à l'aide de la méthode [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)

Vous pouvez également convertir votre DOCX protégé par mot de passe en POWERPOINT dans vos applications Android. Si votre docxument DOCX d'entrée est protégé par un mot de passe, vous ne pouvez pas le convertir au format POWERPOINT sans utiliser le mot de passe. Pour ouvrir un docxument crypté, vous pouvez définir le mot de passe correct dans l'objet LoadOptions et le transmettre au constructeur Docxument.
