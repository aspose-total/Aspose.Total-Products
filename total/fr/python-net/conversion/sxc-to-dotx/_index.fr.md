Après cela, en utilisant l'API Word Python [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/), enregistrez le code HTML créé au format Microsoft Word souhaité.

- **Étape 1** Ouvrez le fichier SXC source à l'aide de la classe Workbook
- Enregistrez le fichier SXC au format HTML en utilisant la méthode save(file, SaveFormat.HTML) en fournissant le nom du fichier et le chemin du répertoire souhaité
-  **Étape 2** Charger le fichier HTML avec une instance de la classe [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)
-  Appelez la méthode `save` tout en spécifiant le chemin du fichier DOTX de sortie. Ainsi, votre fichier SXC est converti en DOTX au chemin spécifié
