Pour un développeur Python, qui essaie d'ajouter une fonction de conversion PPS en CSV dans l'application. L'API [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) peut aider à automatiser le processus de conversion. Il s'agit d'un ensemble complet de diverses API traitant différents formats, y compris les fichiers PPS et CSV.

C'est principalement en deux étapes. Utilisez d'abord l'API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) pour convertir le fichier PPS en HTML. Après cela, en utilisant l'API Excel Python [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), enregistrez le code HTML créé au format Microsoft Excel souhaité.

- **Étape 1** Utiliser l'instance de classe [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/) pour ouvrir le fichier PPS source

-  Appelez la méthode `save` tout en spécifiant le chemin du fichier CSV de sortie. Ainsi, votre fichier PPS est converti en CSV au chemin spécifié
