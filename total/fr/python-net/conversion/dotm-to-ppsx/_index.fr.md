Pour un développeur Python, qui essaie d'ajouter une fonctionnalité de conversion DOTM en PPSX dans l'application ? L'API [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) peut aider à automatiser le processus de conversion. Il s'agit d'un ensemble complet de diverses API traitant de différents formats.  

C'est principalement en deux étapes. Utilisez d'abord l'API [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) pour convertir le fichier DOTM en PDF. Après cela, en utilisant l'API PowerPoint Python [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/), enregistrez le PDF créé dans la présentation au format PPSX.

{{% blocks/products/pf/agp/feature-section-col title="Comment convertir DOTM en PPSX en Python " %}}
- **Étape 1** Ouvrez le fichier DOTM source à l'aide de la classe [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)
- Enregistrez le fichier DOTM au format PDF en utilisant la méthode [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/) en fournissant le nom du fichier et le chemin du répertoire souhaité.
-  **Étape 2** Charger le fichier PDF avec une instance de la classe [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)
-  Appelez la méthode `save` tout en spécifiant le chemin du fichier de sortie et SaveFormat.PPSX comme paramètres. Ainsi, votre fichier DOTM est converti en PPSX au chemin spécifié.

{{% blocks/products/pf/agp/feature-section-col title="Exigences de conversion" %}}

- Pour la conversion DOTM en PPSX, Python 3.5 ou version ultérieure est requis

- Utilisez les commandes pip suivantes ```pip install aspose.slides``` et ```pip install aspose.words```.
