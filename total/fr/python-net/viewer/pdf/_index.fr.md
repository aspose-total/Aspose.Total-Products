{{% blocks/products/pf/agp/feature-section-col title="Lire le fichier PDF via Python pour l'application Viewer" %}}

1. Incluez les packages pertinents dans le projet directement depuis PyPI ([Aspose.PDF](https://pypi.org/project/aspose-pdf/))
1. Créez un objet de classe de document et chargez le fichier source PDF en l'utilisant.
1. Extrayez le texte du PDF en instanciant un objet TextAbsorber.
1. Invoquez la méthode accept() pour lire l’intégralité du texte de votre propre application Viewer développée.
1. Ou convertissez le fichier source en HTML en utilisant la méthode de sauvegarde de la même API.
1. Et utilisez subprocess.run pour appeler le navigateur afin d'afficher le fichier PDF.
