{{% blocks/products/pf/agp/feature-section-col title="Comment convertir POWERPOINT en EXCEL ?" %}}
1. Ouvrir et charger le fichier POWERPOINT à l'aide de la classe Presentation
2. Définissez les options d'enregistrement HTML à l'aide de l'objet HtmlOptions ou appelez directement la méthode d'enregistrement
3. Enregistrez le fichier POWERPOINT au format HTML en utilisant la méthode d'enregistrement avec le chemin spécifié et SaveFormat.HTML en tant que paramètres
4. Utiliser la classe Workbook pour charger le HTML exporté
5. Appelez la méthode save pour le convertir au format EXCEL
