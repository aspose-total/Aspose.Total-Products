Aspose.Total pour PHP via Java est une suite complète qui fournit un large éventail de fonctionnalités pour travailler avec des feuilles de calcul Excel, des présentations Powerpoint ainsi que la génération et la reconnaissance de codes-barres. Avec Aspose.Total pour PHP via Java, vous pouvez facilement ajouter la conversion XLSX en POT à n'importe quelle application PHP en quelques étapes seulement.

Tout d'abord, utilisez Aspose.Cells pour PHP via Java pour exporter le fichier XLSX au format PDF. Ensuite, utilisez Aspose.Slides pour PHP via Java pour convertir le fichier PDF en POT. Aspose.Cells pour PHP via Java et Aspose.Slides pour PHP via Java sont des API enfants d'Aspose.Total pour PHP via le package d'API Java.

{{% blocks/products/pf/agp/feature-section-col title="Comment convertir XLSX en POT ?" %}}
1. Ouvrir et charger le fichier XLSX à l'aide de la classe Workbook
2. Définir les options d'enregistrement PDF à l'aide de l'objet PdfSaveOptions
3. Enregistrez le fichier XLSX au format PDF en utilisant la méthode d'enregistrement
4. Utilisez la classe Presentation pour charger le PDF exporté dans Slides à l'aide de getSlides()->addFromPdf
5. Appelez la méthode save pour le convertir au format POT

{{% blocks/products/pf/agp/feature-section-col title="Exigences de conversion" %}}
1. PHP 7 ou des versions supérieures sont installées
2. La machine doit avoir Oracle JDK 7 ou des versions supérieures
3. Téléchargez le binaire PHP/Java Bridge [JavaBridge.jar](http://php-java-bridge.sourceforge.net/pjb/download.php) et la bibliothèque PHP java/Java.inc [Java.inc](http://php-java-bridge.sourceforge.net/pjb/download.php) et enregistrez-le dans le dossier des paramètres
4. Ajustez le paramètre correspondant à la plate-forme (Windows, Linux, MacOS, etc.) pour [Aspose.Cells](https://docs.aspose.com/cells/php-java/setup-and-installation-guidelines/) et [Aspose.Slides](https://docs.aspose.com/slides/php-java/installation/)
