Aspose.Total pour PHP via Java est un package complet pour la manipulation de feuilles de calcul Excel ainsi que pour les présentations PowerPoint et OpenOffice. Les développeurs peuvent créer, manipuler, rendre et convertir des fichiers Excel via des applications PHP ainsi que charger, lire, écrire et exporter des formats PowerPoint, notamment PPT, PPTX, PPS, POT, PPSX, OpenOffice ODP et d'autres formats. Les développeurs peuvent facilement intégrer l'API PHP pour la reconnaissance et la génération de codes-barres.
{{< /blocks/products/pf/total/overview >}}

{{< blocks/products/pf/total/products description="Aspose.Total for PHP via Java inclut les API suivantes :" >}}

{{< blocks/products/pf/total/product-item href="/cells/php-java/" imgSrc="/total/images/aspose_cells-for-php-java.svg" productName="Aspose.Cells for PHP via Java">}}
Aspose.Cells for PHP via Java API est conçu pour gérer et traiter des feuilles de calcul Excel. Il gère des tâches telles que la conversion des formats de feuille de calcul, la gestion et le rendu des graphiques, la mise en œuvre de formules, l'application de styles détaillés aux feuilles de calcul, et bien plus encore. Grâce à l'intégration de JavaBridge, les développeurs peuvent incorporer de manière transparente la puissance de création, d'édition, de conversion et de rendu de fichiers Excel dans leurs applications PHP.
{{< /blocks/products/pf/total/product-item >}}

{{< blocks/products/pf/total/product-item href="/slides/php-java/" imgSrc="/total/images/aspose_slides-for-php-java.svg" productName="Aspose.Slides for PHP via Java">}}
Aspose.Slides pour PHP via la bibliothèque Java permet aux développeurs de travailler et de traiter efficacement les présentations. Il propose la lecture, l'édition et la conversion de présentations PowerPoint sans recourir à des applications externes. La bibliothèque permet de charger, d'ouvrir et de visualiser des présentations, de les convertir en différents formats, de rendre et d'imprimer des présentations, de les crypter et de les décrypter, de manipuler des entités de présentation telles que des diapositives principales, des formes, des graphiques, des images audio et vidéo, des animations, etc.
{{< /blocks/products/pf/total/product-item >}}

{{< blocks/products/pf/total/product-item href="/barcode/php-java/" imgSrc="/total/images/aspose_barcode-for-php-java.svg" productName="Aspose.BarCode for PHP via Java">}}
Aspose.BarCode pour PHP via Java est une collection d'API de génération et de lecture de codes à barres conçues pour les applications PHP. Il offre aux développeurs la possibilité de lire différents types de codes-barres, y compris les codes-barres 1D, 2D et postaux, à partir d'images capturées sous différents angles. L'API du générateur de codes-barres permet la création d'images de codes-barres dans différents formats tels que JPG, PNG, GIF et BMP. De plus, il offre plusieurs fonctionnalités d'imagerie, telles que la manipulation des bordures, le réglage des couleurs, la rotation de l'image, la personnalisation de la résolution, etc. En utilisant l'API, les développeurs peuvent facilement intégrer la fonctionnalité de code-barres dans leurs applications PHP.
