{{% blocks/products/pf/agp/feature-section-col title="Comment rechercher un fichier PDF via Java" %}}

1. Installez « Aspose.PDF pour Java ».
1. Ajoutez une référence de bibliothèque (importez la bibliothèque) à votre projet Java.
1. Ouvrez le fichier source PDF à l'aide de la classe Document.
1. Créez un objet TextFragmentAbsorber pour rechercher toutes les instances de l'entrée.
1. ou définissez les critères de recherche dans Regex.
1. Acceptez l'absorbeur de la page en utilisant getPages().accept(textFragmentAbsorber).
1. Obtenez la collection de fragments de texte extraits via textFragmentAbsorber.getTextFragments().
1. Parcourez les fragments pour obtenir le résultat.
