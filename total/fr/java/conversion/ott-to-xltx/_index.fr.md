Avant de convertir OTT en XLTX, vous pouvez supprimer les informations inutilisées du document OTT via [Aspose.Words for Java](https://products.aspose.com/words/java/). Vous pouvez utiliser [UnusedStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedStyles) et [UnusedBuiltinStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedBuiltinStyles) pour détecter et supprimer les styles marqués comme « inutilisés ».
