Vous pouvez convertir CGM en RTF en utilisant deux étapes simples. Vous devez d'abord rendre le fichier CGM au format DOC à l'aide de [Aspose.PDF for Java](https://products.aspose.com/pdf/java/). Après cela, en utilisant la puissante API de traitement de documents [Aspose.Words for Java](https://products.aspose.com/words/java/), vous pouvez convertir DOC en RTF. Les deux API font partie du package [Aspose.Total for Java](https://products.aspose.com/total/java/).
