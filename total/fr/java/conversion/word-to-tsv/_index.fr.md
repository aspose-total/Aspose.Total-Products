Avant de convertir WORD en TSV, vous pouvez supprimer les informations inutilisées du document WORD via [Aspose.Words for Java](https://products.aspose.com/words/java/). Vous pouvez utiliser [UnusedStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedStyles) et [UnusedBuiltinStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedBuiltinStyles) pour détecter et supprimer les styles marqués comme « inutilisés ».
