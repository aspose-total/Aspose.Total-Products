{{% blocks/products/pf/agp/feature-section-col title="Lire le fichier XLTM via Python pour l'application Viewer" %}}

1. Référencez les packages requis dans le projet directement depuis PyPI ([Aspose.Cells](https://pypi.org/project/aspose-cells/))
1. Créez un objet de classe Workbook et chargez le fichier source XLTM en l'utilisant.
1. Obtenez toutes les collections de feuilles de calcul.
1. Parcourez chaque feuille ainsi que chaque ligne et cellule pour obtenir sa valeur.
1. Pour le visualiser, convertissez le fichier source XLTM en HTML en utilisant la méthode de sauvegarde de la même API.
1. Et utilisez subprocess.run pour appeler le navigateur afin d'afficher le fichier XLTM.
1. Ou lisez le fichier en mémoire et intégrez-le dans l'application de visualisation Python en conséquence.
