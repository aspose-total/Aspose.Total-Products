{{% blocks/products/pf/agp/feature-section-col title="Gérer les propriétés XLSM via Python" %}}

1. Référencez les API au sein du projet directement depuis PyPI ([Aspose.Cells](https://pypi.org/project/aspose-cells/)).
1. Chargez le fichier XLSM à l’aide de la classe Workbook.
1. Obtenez toutes les propriétés personnalisées via getWorksheets().getCustomDocumentProperties().
1. Utilisez la méthode get(index) pour obtenir la valeur de la propriété.
1. Utilisez la méthode add() pour ajouter plus de propriétés.
1. Utilisez la méthode Remove() pour supprimer une propriété.
1. Appelez la méthode save() pour enregistrer le document XLSM mis à jour.
