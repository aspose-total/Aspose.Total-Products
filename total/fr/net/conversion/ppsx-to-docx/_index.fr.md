Afin d'automatiser le processus de toute présentation PowerPoint ppsx vers la conversion par lots de fichiers docx Word, nous utiliserons [Aspose.Slides pour .NET](https://products.aspose.com/slides/net) et [Aspose.Words pour les API .NET](https://products.aspose.com/words/net). Les deux API font partie du package [Aspose.Total pour .NET](https://products.aspose.com/total/net). 1. Ajouter une référence à Aspose.Slides pour .NET et Aspose.Words pour .NET

1. Enregistrez le document en utilisant [Aspose.Words.Document.Save("output.docx", SaveFormat.Docx)](https://apireference.aspose.com/words/net/aspose.words.document/save/methods/3)
