<h2>Développer la fusion de fichiers PDF à l'aide de .NET</h2>

Besoin de développer un script .NET ou une application utilitaire pour fusionner facilement plusieurs fichiers PDF ?Avec [Aspose.PDF for .NET](https://products.aspose.com/pdf/fr/net/), API enfant de [Aspose.Total for .NET](https://products.aspose.com/total/fr/net/), tout développeur .NET peut intégrer le code API ci-dessus pour programmer la fusion d'applications entre documents.La puissante bibliothèque .NET permet de programmer une solution de fusion de documents pouvant prendre en charge de nombreux formats populaires, notamment le format PDF.<br /><br />
