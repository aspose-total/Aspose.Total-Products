{{% blocks/products/pf/agp/feature-section-col title="How to Convert POTM to XLS?" %}}
1. Open and load POTM file using Presentation class
2. Set the HTML save options using HtmlOptions object or directly call the save method
3. Save the POTM file into HTML using save method with specified path and SaveFormat.HTML as parameters
4. Use Workbook class to load the exported HTML
5. Call the save method to convert it into XLS format
