- title: View or Modify PPT Properties via Python
  steps:
  - Reference APIs within the project directly from PyPI ([Aspose.Slides](https://pypi.org/project/Aspose.Slides/)).

  - Get all Properteis via document_properties.
  - Loop through all properteis and print.
  - To update, use set_custom_property_value method.
  - Save the presentation with the updated metedata information.
