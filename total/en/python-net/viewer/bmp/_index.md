howto:
- title: View BMP Document via Online App
  steps:
  - Upload BMP file to view.
  - Do it by clicking inside the drop area via drag and drop of viewer app.
  - Depending on the size of BMP file and internet speed wait for few seconds.
  - It will be automatically rendered and opened in the browser for view.
  - Navigate between pages in case of multi pages BMP document.
  - Set page view zoom-in or zoom-out.
- title: Load or Read BMP File via Python
  steps:
  - Reference APIs within the project directly from PyPI ([Aspose.Imaging](https://pypi.org/project/aspose-imaging-python-net/))
  - Use Image.load function to load source BMP file from disk.
  - In case of reading and loading from stream
  - Use open method with appropriate file path
  - Then use Image.load with the stream object as parameter
