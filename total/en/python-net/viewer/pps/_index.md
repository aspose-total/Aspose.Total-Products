- title: View PPS Document via Online App
  steps:
  - Import PPS file to view by uploading it.
  - Do it by clicking inside the drop area via drag and drop of viewer app.
  - Depending on the size of PPS file and internet speed, wait for few seconds.
  - It will be automatically rendered and opened in the browser for view.
  - Navigate between pages in case of multi pages PPS document.
  - Set page view zoom-in or zoom-out.
- title: Load or Open PPS File via Python for Viewer App
  steps:

  - And use subprocess.run to invoke the browser to view PPS file.
