howto:
- title: View DOCX Document via Online App
  steps:
  - Import DOCX file to view by uploading it.
  - Do it by clicking inside the drop area via drag and drop of viewer app.
  - Depending on the size of DOCX file and internet speed wait for few seconds.
  - It will be automatically rendered and opened in the browser for view.
  - Navigate between pages in case of multi pages DOCX document.
  - Set page view zoom-in or zoom-out.
- title: Load or Read DOCX File via Python for Viewer App
  steps:
  - Reference APIs within the project directly from PyPI ([Aspose.Words](https://pypi.org/project/aspose-words/))
  - Create document class object and load source DOCX file using it.
  - Iterate through each node using get_child_nodes to get content for your own developed
    Viewer app.
  - Or Convert the source file to HTML using save method of same API.
  - And use subprocess.run to invoke the browser to view DOCX file.
