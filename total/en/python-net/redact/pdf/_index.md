- title: Redact PDF File via Python
  steps:
  - Reference APIs within the project directly from PyPI ([Aspose.PDF](https://pypi.org/project/aspose-pdf/))
  - Load PDF file using Document class
  - Use TextFragmentAbsorber object having search terms as parameter
  - Set Search Options
  - Iterate through each fragment collection to redact
  - Save PDF document
