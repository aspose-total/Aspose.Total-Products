- title: Redact PPT Presentation Online
  steps:
  - Import PPT file by uploading it.
  - Do it by clicking inside the drop area via drag and drop of redaction app.
  - Depending on the size of PPT file and internet speed wait for few seconds.
  - Type the text to search and redact

  - Press the "REDACT" button
  - Download the redaction file instantly.
- title: Redact PPT File via Python
  steps:
  - Reference APIs within the project directly from PyPI ([Aspose.Slides](https://pypi.org/project/Aspose.Slides/))
  - Load PPT file using Presentation class
  - Iterate through the slides
  - Get text frames of the slide in each iteration
  - Iterate through text frames and then through frame paragraphs
  - Access the text portions and check the redaction text
  - Repace text if exists
  - Save the PPT file
