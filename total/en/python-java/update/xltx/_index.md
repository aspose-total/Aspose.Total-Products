{{% blocks/products/pf/agp/feature-section-col title="How to Update XLTX File in Python" %}}

- Create new [Workbook](https://reference.aspose.com/cells/python-java/asposecells.api/Workbook) class object having the source XLTX file as parameter
- Access of relevant Worksheet using [getWorksheets().get(index)](https://reference.aspose.com/cells/python/asposecells.api/workbook#Worksheets) method
- Insert new data in the accessed cell using [Worksheet.getCells().get(indexValue).putValue()](https://reference.aspose.com/cells/python/asposecells.api/worksheet#Cells) method
- Save the file as .xltx file using save() method by passing the file with path as the parameter
