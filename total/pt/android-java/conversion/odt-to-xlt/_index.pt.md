{{% blocks/products/pf/feature-page-section  h2="Remover informações não utilizadas de um odtumento ODT no Android via Java" %}}
Antes de converter ODT para XLT, você pode remover informações não utilizadas do Odtumento ODT por meio do [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/). Às vezes, pode ser necessário remover informações não utilizadas ou duplicadas para reduzir o tamanho do odtumento de saída e o tempo de processamento. A classe [CleanupOptions](https://reference.aspose.com/words/java/com.aspose.words/CleanupOptions) permite especificar opções para limpeza de odtumentos. Para remover estilos duplicados ou apenas estilos ou listas não utilizados do odtumento, você pode usar o método [Cleanup](https://reference.aspose.com/words/java/com.aspose.words/Odtument#cleanup()). Você pode usar os [UnusedStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedStyles) e [UnusedBuiltinStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedBuiltinStyles) para detectar e remover estilos marcados como “não usados”.
