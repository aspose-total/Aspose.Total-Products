{{% blocks/products/pf/feature-page-summary %}}
Você pode integrar o recurso de conversão PS para DIF em seus aplicativos Android em um processo de duas etapas. Em primeiro lugar, usando [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/) você pode converter PS para XLSX. Em segundo lugar, você pode converter XLSX para DIF usando a API de processamento de planilha poderosa [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/). Ambas as APIs estão na família de produtos [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/).

{{% blocks/products/pf/agp/feature-section-col title="API do Android para renderizar PS para DIF" %}}
1. Abra o arquivo PS usando a classe [Document](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document)
2. Converta PS para XLSX usando [save](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document#save-java.lang.String-com.aspose.pdf.SaveOptions-) método
3. Carregue o documento XLSX usando a classe [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)
4. Salve o documento no formato DIF usando [save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.SaveOptions)) método
