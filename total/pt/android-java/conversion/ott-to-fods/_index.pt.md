{{% blocks/products/pf/feature-page-section  h2="Remover informações não utilizadas de um ottumento OTT no Android via Java" %}}
Antes de converter OTT para FODS, você pode remover informações não utilizadas do Ottumento OTT por meio do [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/). Às vezes, pode ser necessário remover informações não utilizadas ou duplicadas para reduzir o tamanho do ottumento de saída e o tempo de processamento. A classe [CleanupOptions](https://reference.aspose.com/words/java/com.aspose.words/CleanupOptions) permite especificar opções para limpeza de ottumentos. Para remover estilos duplicados ou apenas estilos ou listas não utilizados do ottumento, você pode usar o método [Cleanup](https://reference.aspose.com/words/java/com.aspose.words/Ottument#cleanup()). Você pode usar os [UnusedStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedStyles) e [UnusedBuiltinStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedBuiltinStyles) para detectar e remover estilos marcados como “não usados”.
