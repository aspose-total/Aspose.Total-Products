{{% blocks/products/pf/feature-page-summary %}}
Você pode integrar o recurso de conversão XPS para FODS em seus aplicativos Android em um processo de duas etapas. Em primeiro lugar, usando [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/) você pode converter XPS para XLSX. Em segundo lugar, você pode converter XLSX para FODS usando a API de processamento de planilha poderosa [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/). Ambas as APIs estão na família de produtos [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/).
