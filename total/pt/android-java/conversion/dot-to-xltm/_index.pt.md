{{% blocks/products/pf/feature-page-section  h2="Remover informações não utilizadas de um dotumento DOT no Android via Java" %}}
Antes de converter DOT para XLTM, você pode remover informações não utilizadas do Dotumento DOT por meio do [Aspose.Words for Android via Java](https://products.aspose.com/words/android-java/). Às vezes, pode ser necessário remover informações não utilizadas ou duplicadas para reduzir o tamanho do dotumento de saída e o tempo de processamento. A classe [CleanupOptions](https://reference.aspose.com/words/java/com.aspose.words/CleanupOptions) permite especificar opções para limpeza de dotumentos. Para remover estilos duplicados ou apenas estilos ou listas não utilizados do dotumento, você pode usar o método [Cleanup](https://reference.aspose.com/words/java/com.aspose.words/Dotument#cleanup()). Você pode usar os [UnusedStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedStyles) e [UnusedBuiltinStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedBuiltinStyles) para detectar e remover estilos marcados como “não usados”.
