Você pode converter o formato DOTX para JSON em seus aplicativos Android via [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/). 4. Salve o dotxumento no formato JSON usando [Save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.SaveOptions)) método
