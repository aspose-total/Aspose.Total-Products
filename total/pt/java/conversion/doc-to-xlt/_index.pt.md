Antes de converter DOC para XLT, você pode remover informações não utilizadas do Documento DOC via [Aspose.Words for Java](https://products.aspose.com/words/java/). Você pode usar os [UnusedStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedStyles) e [UnusedBuiltinStyles](https://reference.aspose.com/words/java/com.aspose.words/cleanupoptions#UnusedBuiltinStyles) para detectar e remover estilos marcados como “não usados”.
