1. Abra o arquivo DOC usando a classe [Document](https://reference.aspose.com/words/java/com.aspose.words/Document)

6. Salve o documento no PPSX usando o método [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
