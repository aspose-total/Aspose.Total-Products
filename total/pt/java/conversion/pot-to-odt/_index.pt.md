[Aspose.Total for Java](https://products.aspose.com/total/java/) Bibliotecas de Automação de Formato de Arquivo capacitam os desenvolvedores Java a automatizar o processo de conversão em lote do PowerPoint POT para Word ODT. A conversão do documento é um processo de duas etapas e envolve o uso de duas APIs. Usaremos [Aspose.Slides for Java](https://products.aspose.com/slides/java/) que é uma API do PowerPoint para manipulação e gerenciamento de apresentações para converter POT em HTML. Depois disso, usando a API de processamento de texto rica em recursos [Aspose.Words for Java](https://products.aspose.com/words/java/), converteremos o HTML em ODT.

{{% blocks/products/pf/agp/feature-section-col title="Como converter POT para ODT via Java" %}}
1. Abra o arquivo POT usando a classe [Apresentação](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Converta POT em HTML usando [salvar](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.ISaveOptions-) e defina Html como SaveFormat
3. Carregue o arquivo HTML convertido usando a classe [Document](https://reference.aspose.com/words/java/com.aspose.words/Document)
4. Salve o documento no formato ODT usando o método [save](https://reference.aspose.com/words/java/com.aspose.words/Document#save(java.lang.String,int))
