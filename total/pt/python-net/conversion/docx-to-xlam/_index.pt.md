Depois disso, usando o Excel Python API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/), salve o HTML criado no formato desejado do Microsoft Excel.

- **Etapa 1** Abra o arquivo DOCX de origem usando a classe [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)
- Salve o arquivo DOCX em HTML usando o método [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/), fornecendo o nome do arquivo e o caminho do diretório desejado
-  **Etapa 2** Carregar arquivo HTML com uma instância da classe Workbook com arquivo e LoadOptions como parâmetros
-  Chame o método `save` ao especificar o caminho do arquivo XLAM de saída. Portanto, seu arquivo DOCX é convertido em XLAM no caminho especificado
