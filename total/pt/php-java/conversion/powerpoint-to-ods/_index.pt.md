{{% blocks/products/pf/agp/feature-section-col title="Como converter POWERPOINT para ODS?" %}}
1. Abra e carregue o arquivo POWERPOINT usando a classe Apresentação
2. Defina as opções de salvar HTML usando o objeto HtmlOptions ou chame diretamente o método save
