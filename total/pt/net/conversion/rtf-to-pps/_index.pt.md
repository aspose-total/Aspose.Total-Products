Para automatizar o processo de qualquer arquivo de documento do Word para conversão em lote de apresentação do PowerPoint pps, usaremos [Aspose.Words for .NET](https://products.aspose.com/words/net) e [Aspose.Slides para .NET](https://products.aspose.com/slides/net) APIs. O primeiro é uma API de processamento de texto para processar ou manipular documentos do Microsoft Word. Considerando que o último é uma API de manipulação de apresentação que permite criar ou modificar slides do Microsoft PowerPoint. Ambas as APIs fazem parte do pacote [Aspose.Total for .NET](https://products.aspose.com/total/net). Você pode [baixar] diretamente (https://releases.aspose.com/) do Nuget ou pode usar os seguintes comandos do Console do Gerenciador de Pacotes.
