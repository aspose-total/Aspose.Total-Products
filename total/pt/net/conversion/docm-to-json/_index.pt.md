Ao converter DOCM para JSON, você também pode definir o intervalo para o formato JSON de saída. Para definir o intervalo, você pode abrir o HTML convertido usando a classe Workbook, obter CellsCollection da planilha que contém os dados, criar um intervalo de CellsCollection especificando índices de linha e coluna e chamar o método ExportRangeToJson com referências aos objetos Range & ExportRangeToJsonOptions. Por fim, você pode salvar os dados JSON em um arquivo por meio do método File.WriteAllText.
