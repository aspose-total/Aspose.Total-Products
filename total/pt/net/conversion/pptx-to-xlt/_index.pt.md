{{% blocks/products/pf/feature-page-section  h2="Converter PPTX Protegido para XLT via C#" %}}
Ao converter o arquivo PPTX para XLT, se o documento PPTX de entrada estiver protegido por senha, você não poderá convertê-lo para XLT sem descriptografar o documento. Quando seu documento é protegido por senha, isso significa que ele impõe certas restrições na apresentação. Para remover as restrições, a senha deve ser inserida. Uma apresentação protegida por senha é considerada uma apresentação bloqueada. A API permite que você abra o documento criptografado passando a senha correta em um objeto LoadOptions.
