アプリケーション内に POT から XLSM への変換機能を追加しようとしている Python 開発者向け。 [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API は、変換プロセスの自動化に役立ちます。 POT や XLSM ファイルなど、さまざまな形式を扱うさまざまな API の完全なパッケージです。

主に2段階です。 まず [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) API を使用して POT ファイルを HTML に変換します。 その後、Excel Python API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) を使用して、作成した HTML を目的の Microsoft Excel 形式で保存します。

- **ステップ1** [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/) クラス インスタンスを使用してソース POT ファイルを開く
