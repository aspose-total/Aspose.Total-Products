アプリケーション内に XLSM から PPSX への変換機能を追加しようとしている Python 開発者向け, [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API は、変換プロセスの自動化に役立ちます。 XLSM や PPSX ファイルなど、さまざまな形式を扱うさまざまな API の完全なパッケージです。.
