1. または、replacement.FindReplaceOptions とともに正規表現として range.replace_regex を使用します。
1. 「save」メソッドを呼び出してドキュメントを保存します。

<h2>Python による DOC ファイル検索アプリケーションの開発</h2>

DOC 検索可能なアプリまたはユーティリティを開発する必要がありますか? Aspose.Words for Python via .NET を使用すると、Aspose.Total for Python via .NET の子 API を使用して、Python 開発者は上記の API コードを統合して、ドキュメント全体のコンテンツ検索をプログラムできます。 強力な Python ライブラリを使用すると、DOC 形式を含む多くの一般的な形式をサポートできるあらゆる検索アプリ ソリューションをプログラミングできます。<br /><br />
