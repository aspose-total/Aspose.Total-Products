<h2>C++ で ODT パスワード保護アプリケーションを開発する</h2>

ODT パスワード保護アプリまたはユーティリティを開発する必要がありますか?[Aspose.Total for C++](https://products.aspose.com/total/ja/cpp/) の子 API である [Aspose.Words for C++](https://products.aspose.com/words/cpp/) を使用すると、どの C++ 開発者でも上記の API コードをドキュメント保護アプリケーション内に統合できます。強力な C++ ライブラリを使用すると、あらゆるドキュメントのパスワード保護ソリューションをプログラミングして、ファイルを保護およびロックできます。さらに、ODT 形式を含む多くの一般的な形式をサポートできます。<br /><br />
