<h2>.NET 経由で ODP パスワード保護アプリケーションを開発する</h2>

ODP パスワード保護アプリまたはユーティリティを開発する必要がありますか?[Aspose.Total for .NET](https://products.aspose.com/total/ja/net/) の子 API である [Aspose.Slides for .NET](https://products.aspose.com/slides/ja/net/) を使用すると、.NET 開発者は上記の API コードをドキュメント保護アプリケーション内に統合できます。強力な .NET ライブラリを使用すると、プレゼンテーションを保護してロックするためのあらゆるドキュメントのパスワード保護ソリューションをプログラミングできます。さらに、ODP 形式を含む多くの一般的な形式をサポートできます。<br /><br />
