<h2>.NET経由でDOCファイルビューアアプリケーションを開発する</h2>

DOC ビューア アプリまたはユーティリティを開発する必要がありますか?[Aspose.Total for .NET](https://products.aspose.com/total/ja/net/) の子 API である [Aspose.Words for .NET](https://products.aspose.com/words/ja/net/) を使用すると、.NET 開発者は上記の API コードをドキュメント ビューア アプリケーション内に統合できます。強力な .NET ライブラリにより、DOC 形式を含む多くの一般的な形式をサポートできるあらゆるドキュメント表示ソリューションをプログラミングできます。<br /><br />
