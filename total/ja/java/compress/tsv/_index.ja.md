<h2> 📊 Java APIを使用してTSVファイルを圧縮する：ストレージとパフォーマンスを最適化するためのタブ区切りデータ</h2>

## ✅ 主なユースケース

- **タブ区切りデータセットのストレージスペースを最小化：** TSVファイルを圧縮して、巨大なデータセットをスリムで管理しやすく、コスト効率の良いストレージに保つ。
