응용 프로그램 내에서 POTX를 WORD로 변환 기능을 추가하려는 Python 개발자의 경우? [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API는 변환 프로세스를 자동화하는 데 도움이 될 수 있습니다. 다양한 형식을 처리하는 다양한 API의 전체 패키지입니다.  

주로 2단계입니다. 먼저 [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) API를 사용하여 POTX 파일을 PDF로 변환합니다. 그런 다음 Microsoft Word Python API [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/)를 사용하여 생성된 PDF를 Microsoft Word에 WORD 형식으로 저장합니다.

-  **1 단계** [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/) 클래스의 인스턴스로 PDF 파일 로드
-  출력 파일 경로 및 SaveFormat.PDF를 매개변수로 지정하면서 `save` 메소드를 호출하십시오. 따라서 POTX 파일은 지정된 경로에서 PDF로 변환됩니다.
- **2 단계** [Document](https://reference.aspose.com/words/python-net/aspose.words/document/) 클래스를 사용하여 PDF 파일 열기
- 파일 이름과 원하는 디렉토리 경로를 제공하여 [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/) 메서드를 사용하여 PDF 파일을 WORD 파일로 저장합니다.
- 다음은 PowerPoint 프레젠테이션을 Word [Microsoft Powerpoint to Word](https://products.aspose.com/total/python-net/conversion/)로 변환하는 또 다른 코드 조각입니다.
