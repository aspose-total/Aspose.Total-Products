Ezenkívül az API lehetővé teszi a JSON ODP-re történő elemzését meghatározott elrendezési beállításokkal. Mindezek a lehetőségek lehetővé teszik, hogy adatait az igényeinek megfelelően jelenítse meg. A következő kódrészlet bemutatja, hogyan állíthatja be az elrendezési beállításokat.
