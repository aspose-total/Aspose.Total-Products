Két egyszerű lépéssel integrálhatja a XPS-POTX konverziós funkciót Android-alkalmazásaiba. Az első lépésben exportálhatja a XPS-et PPTX-be az [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/) használatával. Ezt követően az [Aspose.Slides for Android via Java](https://products.aspose.com/slides/android-java/) használatával konvertálhatja a PPTX-t POTX-vé. Mindkét API az [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) csomagba tartozik.
