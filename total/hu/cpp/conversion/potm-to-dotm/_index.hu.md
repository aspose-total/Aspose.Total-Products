{{% blocks/products/pf/feature-page-summary %}}
Az [Aspose.Total for C++](https://products.aspose.com/total/cpp/) a C++ fájlformátum-automatizálási könyvtárak teljes csomagja. A csomagban elérhető API-k gazdag funkcióinak használatával könnyedén konvertálhatjuk a PowerPoint POTM-ot Word DOTM-ba. Az átalakítás végrehajtásához először használhatja az [Aspose.Slides for C++](https://products.aspose.com/slides/cpp/) API-t a POTM HTML-lé konvertálásához. Ezt követően a funkciókban gazdag szövegszerkesztő API [Aspose.Words for C++](https://products.aspose.com/words/cpp/) használatával a HTML-t DOTM formátumba konvertálhatja.

{{% blocks/products/pf/agp/feature-section-col title="C++ API a POTM DOTM-vé konvertálásához" %}}
1. Töltse be a POTM-fájlt a [Presentation](https://reference.aspose.com/slides/cpp/class/aspose.slides.presentation) osztályhivatkozás használatával
2. Rendelje meg a POTM-ot HTML-ben a [Mentés](https://reference.aspose.com/slides/cpp/class/aspose.slides.presentation#afcd59ec697bf05c10f78c3869de2ec9e) tagfunkcióval, és állítsa be a HTML-t SaveFormat-ként
3. Töltse be a konvertált HTML-fájlt a [Dotmument](https://reference.aspose.com/words/cpp/class/aspose.words.dotmument) osztályhivatkozás használatával
4. Mentse a dokumentumot DOTM formátumba a [Mentés](https://reference.aspose.com/words/cpp/class/aspose.words.dotmument#save_string) tagfunkcióval

Alternatív megoldásként letöltheti az offline MSI telepítőt vagy a DLL-eket ZIP-fájlban a [downloads](https://downloads.aspose.com/total/cpp) webhelyről.

SharedPtr<Presentation> prs = MakeObject<Presentation>(u"input.potm");
