{{% blocks/products/pf/feature-page-summary %}}
Az [Aspose.Total for C++](https://products.aspose.com/total/cpp/) a C++ fájlformátum-automatizálási könyvtárak teljes csomagja. A csomagban elérhető API-k gazdag funkcióinak használatával könnyedén konvertálhatjuk a PowerPoint PPSX-ot Word DOCX-ba. Az átalakítás végrehajtásához először használhatja az [Aspose.Slides for C++](https://products.aspose.com/slides/cpp/) API-t a PPSX HTML-lé konvertálásához. Ezt követően a funkciókban gazdag szövegszerkesztő API [Aspose.Words for C++](https://products.aspose.com/words/cpp/) használatával a HTML-t DOCX formátumba konvertálhatja.

{{% blocks/products/pf/agp/feature-section-col title="C++ API a PPSX DOCX-vé konvertálásához" %}}
1. Töltse be a PPSX-fájlt a [Presentation](https://reference.aspose.com/slides/cpp/class/aspose.slides.presentation) osztályhivatkozás használatával
2. Rendelje meg a PPSX-ot HTML-ben a [Mentés](https://reference.aspose.com/slides/cpp/class/aspose.slides.presentation#afcd59ec697bf05c10f78c3869de2ec9e) tagfunkcióval, és állítsa be a HTML-t SaveFormat-ként
3. Töltse be a konvertált HTML-fájlt a [Docxument](https://reference.aspose.com/words/cpp/class/aspose.words.docxument) osztályhivatkozás használatával
4. Mentse a dokumentumot DOCX formátumba a [Mentés](https://reference.aspose.com/words/cpp/class/aspose.words.docxument#save_string) tagfunkcióval
