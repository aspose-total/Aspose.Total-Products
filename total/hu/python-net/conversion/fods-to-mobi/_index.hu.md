Főleg két lépésben. Először is használja az [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) API-t a FODS fájl HTML formátumba konvertálásához. Ezután a Word Python API [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) használatával mentse a létrehozott HTML-t a kívánt Microsoft Word formátumba.
