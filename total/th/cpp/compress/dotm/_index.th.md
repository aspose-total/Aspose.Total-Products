<h2>พัฒนาแอปพลิเคชันการบีบอัดไฟล์ DOTM โดยใช้ C ++</h2>

ต้องการพัฒนาสคริปต์ C++ หรือแอปยูทิลิตี้เพื่อบีบอัดไฟล์ DOTM หลายไฟล์ได้อย่างง่ายดายหรือไม่ด้วย API ลูก [Aspose.Words for C++](https://products.aspose.com/words/cpp/) ของ [Aspose.Total for C++](https://products.aspose.com/total/th/cpp/) นักพัฒนา C++ ทุกคนสามารถรวมโค้ด API ข้างต้นเพื่อตั้งโปรแกรมแอปบีบอัดข้ามเอกสารได้ไลบรารี C++ อันทรงพลังสำหรับการบีบอัดเอกสาร รองรับรูปแบบยอดนิยมมากมาย รวมถึงรูปแบบ DOTM<br />

มีสามตัวเลือกในการติดตั้ง Aspose.Words สำหรับ C++ หรือ Aspose.Total สำหรับ C++ ในสภาพแวดล้อมนักพัฒนาของคุณโปรดเลือกรายการที่ตรงกับความต้องการของคุณและปฏิบัติตามคำแนะนำทีละขั้นตอน:<br /><br />

- ติดตั้ง [แพ็คเกจ NuGet](https://www.nuget.org/packages/Aspose.Words.Cpp/) ดู [เอกสารประกอบ](https://docs.aspose.com/words/cpp/installation/#install-or-update-aspose-words-for-cpp-using-nuget)
- ติดตั้งไลบรารีโดยใช้ [คอนโซลตัวจัดการแพ็คเกจ](https://docs.aspose.com/words/cpp/installation/#install-or-update-asposewords-using-package-manager-console) ภายใน Visual Studio IDE
- ติดตั้งไลบรารี่ด้วยมือโดยใช้ [ตัวติดตั้ง Windows](https://docs.aspose.com/words/cpp/installation/#install-asposewords-for-c-by%20hand)

{{% blocks/products/pf/agp/feature-section-col title="ความต้องการของระบบ" %}}
คุณสามารถใช้ไลบรารี C++ นี้เพื่อพัฒนาซอฟต์แวร์บนระบบปฏิบัติการ Microsoft Windows, Linux และ macOS:<br /><br />

- GCC >= 6.3.0 และ Clang >= 3.9.1 จำเป็นสำหรับ Linux
- Xcode >= 12.5.1, Clang และ libc++ จำเป็นสำหรับ macOS

<br /><br />
หากคุณพัฒนาซอฟต์แวร์สำหรับ Linux หรือ macOS โปรดตรวจสอบข้อมูลเกี่ยวกับการพึ่งพาไลบรารีเพิ่มเติม (แพ็คเกจฟอนต์คอนฟิกและ mesa-glu) ใน [เอกสารประกอบผลิตภัณฑ์](https://docs.aspose.com/words/cpp/system-requirements/)
