<h2>พัฒนาแอปพลิเคชันตัวแยกไฟล์ ODT โดยใช้ C ++</h2>

ต้องการพัฒนาสคริปต์ C++ หรือแอพยูทิลิตี้เพื่อแยกไฟล์ ODT ได้อย่างง่ายดายหรือไม่ด้วย [Aspose.Words for C++](https://products.aspose.com/words/cpp/) ซึ่งเป็น API ลูกของ [Aspose.Total for C++](https://products.aspose.com/total/th/cpp/) นักพัฒนา C++ ทุกคนสามารถรวมโค้ด API ข้างต้นเข้ากับแอปตัวแยกโปรแกรมได้ไลบรารี C++ อันทรงพลังสำหรับการแยกเอกสารรองรับรูปแบบยอดนิยมมากมาย รวมถึงรูปแบบ ODT เช่นไฟล์ PDF, Word, Excel และ PowerPoint<br /><br />

- ติดตั้ง [แพ็คเกจ NuGet](https://www.nuget.org/packages/Aspose.Words.Cpp/) ดู [เอกสารประกอบ](https://docs.aspose.com/words/cpp/installation/#install-or-update-aspose-words-for-cpp-using-nuget)

คุณสามารถใช้ไลบรารี C++ นี้เพื่อพัฒนาซอฟต์แวร์บนระบบปฏิบัติการ Microsoft Windows, Linux และ macOS:<br /><br />

- GCC >= 6.3.0 และ Clang >= 3.9.1 จำเป็นสำหรับ Linux
- Xcode >= 12.5.1, Clang และ libc++ จำเป็นสำหรับ macOS

<br /><br />
