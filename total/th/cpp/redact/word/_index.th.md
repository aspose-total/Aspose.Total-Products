<h2>พัฒนาแอปพลิเคชัน Redaction Word ผ่าน C ++</h2>

ต้องการพัฒนาแอปหรือยูทิลิตี้การแก้ไข Word หรือไม่ด้วย [Aspose.Words for C++](https://products.aspose.com/words/cpp/) ซึ่งเป็น API ลูกของ [Aspose.Total for C++](https://products.aspose.com/total/th/cpp/) นักพัฒนา C++ ทุกคนสามารถรวมโค้ด API ข้างต้นภายในแอปพลิเคชันการแก้ไขเอกสารได้ไลบรารี C++ อันทรงพลังช่วยให้สามารถตั้งโปรแกรมโซลูชันการตรวจทานเอกสารได้นอกจากนี้ยังสามารถรองรับรูปแบบยอดนิยมมากมายรวมถึงรูปแบบ Word<br /><br />

คุณสามารถใช้ไลบรารี C++ นี้เพื่อพัฒนาซอฟต์แวร์บนระบบปฏิบัติการ Microsoft Windows, Linux และ macOS:<br /><br />

- GCC >= 6.3.0 และ Clang >= 3.9.1 จำเป็นสำหรับ Linux
- Xcode >= 12.5.1, Clang และ libc++ จำเป็นสำหรับ macOS

<br /><br />
หากคุณพัฒนาซอฟต์แวร์สำหรับ Linux หรือ macOS โปรดตรวจสอบข้อมูลเกี่ยวกับการพึ่งพาไลบรารีเพิ่มเติม (แพ็คเกจฟอนต์คอนฟิกและ mesa-glu) ใน [เอกสารประกอบผลิตภัณฑ์](https://docs.aspose.com/words/cpp/system-requirements/)
