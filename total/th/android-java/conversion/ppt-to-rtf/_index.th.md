[Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) อนุญาตให้จัดการรูปแบบไฟล์ภายในแอปพลิเคชัน Android ด้วยการใช้ API ที่ให้มาในแพ็คเกจ คุณสามารถทำให้กระบวนการแปลง PowerPoint PPT เป็น Word RTF ในแอปของคุณเป็นแบบอัตโนมัติ

4. บันทึกเอกสารในรูปแบบ RTF โดยใช้เมธอด [save](https://reference.aspose.com/words/java/com.aspose.words/Rtfument#save(java.lang.String,int)) และตั้งค่า Rtf เป็น SaveFormat
