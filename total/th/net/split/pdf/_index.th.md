<h2>พัฒนาแอปพลิเคชัน PDF File Splitter โดยใช้ .NET</h2>

ต้องการพัฒนาสคริปต์ .NET หรือแอพยูทิลิตี้เพื่อแยกไฟล์ PDF ได้อย่างง่ายดายหรือไม่ ด้วย [Aspose.PDF for .NET](https://products.aspose.com/pdf/th/net/) ซึ่งเป็น API ลูกของ [Aspose.Total for .NET](https://products.aspose.com/total/th/net/) นักพัฒนา .NET ทุกคนสามารถรวมโค้ด API ข้างต้นเข้ากับแอปตัวแยกโปรแกรมได้ไลบรารี .NET อันทรงพลังสำหรับการแยกเอกสารรองรับรูปแบบยอดนิยมมากมาย รวมถึงรูปแบบ PDF เช่น PDF, เอกสาร Microsoft Word, สเปรดชีต Excel และงานนำเสนอ PowerPoint<br /><br />

- ติดตั้งไลบรารีโดยใช้ [Package Manager Console](https://docs.aspose.com/pdf/net/installation/#install-package-using-the-package-manager-console) ภายใน Visual Studio IDE

- Microsoft .NET Framework เริ่มต้นจากเวอร์ชัน 2.0 แรกสุด และลงท้ายด้วย '.NET Framework 4.8' ล่าสุด
- .NET Core เริ่มต้นจากเวอร์ชัน 2.0 แรกสุด และลงท้ายด้วย '.NET 6' ล่าสุด
- โมโน >= 2.6.7
<br /><br />
เนื่องจากโค้ด .NET ไม่ต้องอาศัยฮาร์ดแวร์หรือระบบปฏิบัติการพื้นฐาน แต่ต้องใช้บน Virtual Machine เท่านั้น คุณจึงมีอิสระในการพัฒนาซอฟต์แวร์ทุกประเภทสำหรับ Windows, macOS, Android, iOS และ Linuxเพียงตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง .NET Framework, .NET Core, Windows Azure, Mono หรือ Xamarin เวอร์ชันที่เกี่ยวข้อง<br /><br />
เราขอแนะนำให้ใช้ Microsoft Visual Studio, Xamarin และ MonoDevelop IDE เพื่อสร้างแอปพลิเคชัน C#, F#, VB.NET
