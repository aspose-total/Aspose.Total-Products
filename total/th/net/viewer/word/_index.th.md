<h2>พัฒนาแอพพลิเคชั่นตัวแสดงไฟล์ Word ผ่าน .NET</h2>

ต้องการพัฒนาแอปหรือยูทิลิตี้ตัวแสดง Word หรือไม่ด้วย [Aspose.Words for .NET](https://products.aspose.com/words/th/net/) ซึ่งเป็น API ลูกของ [Aspose.Total for .NET](https://products.aspose.com/total/th/net/) นักพัฒนา .NET ทุกคนสามารถรวมโค้ด API ข้างต้นภายในแอปพลิเคชันดูเอกสารได้ไลบรารี .NET อันทรงพลังช่วยให้สามารถเขียนโปรแกรมโซลูชันการแสดงเอกสารใดๆ ที่สามารถรองรับรูปแบบยอดนิยมมากมาย รวมถึงรูปแบบ Word<br /><br />

- ติดตั้ง [แพ็คเกจ NuGet](https://www.nuget.org/packages/Aspose.Words/) ดู [เอกสารประกอบ](https://docs.aspose.com/words/net/installation/#install-or-update-aspose-words-for-net-using-nuget)

- ติดตั้งไลบรารี่ด้วยมือโดยใช้ [ตัวติดตั้ง Windows](https://docs.aspose.com/words/net/installation/#install-asposewords-for-net-using-installer)
