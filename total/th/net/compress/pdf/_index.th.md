<h2>พัฒนาแอปพลิเคชันการบีบอัดไฟล์ PDF โดยใช้ .NET</h2>

ต้องการพัฒนาสคริปต์ .NET หรือแอพยูทิลิตี้เพื่อบีบอัดไฟล์ PDF หลายไฟล์อย่างง่ายดายหรือไม่ด้วย API ลูก [Aspose.PDF for .NET](https://products.aspose.com/pdf/net/) ของ [Aspose.Total for .NET](https://products.aspose.com/total/net/) นักพัฒนา .NET ทุกคนสามารถรวมโค้ด API ข้างต้นเพื่อตั้งโปรแกรมแอปบีบอัดข้ามเอกสารได้ไลบรารี .NET อันทรงพลังสำหรับการบีบอัดเอกสาร รองรับรูปแบบยอดนิยมมากมาย รวมถึงรูปแบบ PDF<br />

- ติดตั้ง [NuGet Package](https://www.nuget.org/packages/Aspose.PDF/) ดู [Documentation](https://docs.aspose.com/pdf/net/installation/#installing-asposepdf-for-net-through-nuget)
- ติดตั้งไลบรารีโดยใช้ [Package Manager Console](https://docs.aspose.com/pdf/net/installation/#install-package-using-the-package-manager-console) ภายใน Visual Studio IDE
