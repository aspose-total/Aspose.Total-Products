<h2>พัฒนาแอปพลิเคชันการบีบอัดไฟล์ XLT โดยใช้ .NET</h2>

ต้องการพัฒนาสคริปต์ .NET หรือแอพยูทิลิตี้เพื่อบีบอัดไฟล์ XLT หลายไฟล์อย่างง่ายดายหรือไม่ด้วย [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) ซึ่งเป็น API ลูกของ [Aspose.Total for .NET](https://products.aspose.com/total/net/) นักพัฒนา .NET ทุกคนสามารถรวมโค้ด API ข้างต้นเพื่อตั้งโปรแกรมแอปบีบอัดข้ามเอกสารได้ไลบรารี .NET อันทรงพลังสำหรับการบีบอัดเอกสาร รองรับรูปแบบยอดนิยมมากมาย รวมถึงรูปแบบ XLT<br /><br />

- ติดตั้งไลบรารี่ด้วยมือโดยใช้ [Windows Installer](https://docs.aspose.com/cells/net/installing-aspose-cells-on-windows/)

ผลิตภัณฑ์ของเราสามารถข้ามแพลตฟอร์มได้อย่างสมบูรณ์และรองรับการใช้งาน .NET หลักทั้งหมดตามข้อกำหนด '.NET Standard 2.0':<br /><br />

- Microsoft .NET Framework เริ่มต้นจากเวอร์ชัน 2.0 แรกสุด และลงท้ายด้วย '.NET Framework 4.8' ล่าสุด

- โมโน >= 2.6.7

เราขอแนะนำให้ใช้ Microsoft Visual Studio, Xamarin และ MonoDevelop IDE เพื่อสร้างแอปพลิเคชัน C#, F#, VB.NET
