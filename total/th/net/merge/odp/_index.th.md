<h2>พัฒนา ODP File Merger โดยใช้ .NET</h2>

ต้องการพัฒนาแอปยูทิลิตี้การควบรวมกิจการ .NET เพื่อรวมไฟล์ ODP หลายไฟล์ได้อย่างง่ายดายหรือไม่ด้วย [Aspose.Slides for .NET](https://products.aspose.com/slides/th/net/) ซึ่งเป็น Child API ของ [Aspose.Total for .NET](https://products.aspose.com/total/th/net/) นักพัฒนา .NET ทุกคนสามารถรวมโค้ด API ข้างต้นเพื่อเชื่อมเอกสารเข้าด้วยกันได้อย่างง่ายดายไลบรารี .NET อันทรงพลังสำหรับการรวมเอกสาร รองรับรูปแบบยอดนิยมมากมาย รวมถึงรูปแบบ ODP<br /><br />
