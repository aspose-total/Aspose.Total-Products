DOCM — ไฟล์ `.docm` รองรับมาโครที่ฝังอยู่และเหมาะสำหรับเอกสารที่ต้องรวมพฤติกรรมที่เป็นโปรแกรมได้; Python APIs สามารถสร้างเอกสารที่เปิดใช้งานมาโครได้โดยการประกอบเนื้อหา OOXML และแนบโครงการ VBA ที่ได้รับลงนามหรือไม่ได้รับลงนามผ่านการอัตโนมัติ การสร้างโปรแกรมช่วยให้มัโครถูกห่อหุ้มอย่างสม่ำเสมอและสามารถตรวจสอบหรือลงนามเป็นส่วนหนึ่งของการสร้าง
