Aspose.Cells עבור PHP באמצעות Java API מיועד לניהול ועיבוד גיליונות אלקטרוניים של Excel.  הוא מטפל במשימות כגון המרת פורמטים של גיליונות אלקטרוניים, ניהול ועיבוד תרשימים, הטמעת נוסחאות, החלת סגנונות מפורטים על גליונות עבודה ועוד הרבה יותר.  עם שילוב JavaBridge, מפתחים יכולים לשלב בצורה חלקה את הכוח של יצירה, עריכה, המרה ועיבוד קובצי Excel לתוך יישומי PHP שלהם.
{{< /blocks/products/pf/total/product-item >}}

{{< blocks/products/pf/total/product-item href="/slides/php-java/" imgSrc="/total/images/aspose_slides-for-php-java.svg" productName="Aspose.Slides for PHP via Java">}}
Aspose.Slides עבור PHP באמצעות ספריית Java מאפשרת למפתחים לעבוד ולעבד מצגות בצורה יעילה.  הוא מציע קריאה, עריכה והמרה של מצגות PowerPoint מבלי להסתמך על יישומים חיצוניים.  הספרייה מאפשרת טעינה, פתיחה וצפייה במצגות, המרתן לפורמטים שונים, רינדור והדפסת מצגות, הצפנתן ופענוחן, מניפולציה של ישויות מצגות כמו שקופיות מאסטר, צורות, תרשימים, פריימים של אודיו ווידאו, אנימציות ועוד.
{{< /blocks/products/pf/total/product-item >}}

{{< blocks/products/pf/total/product-item href="/barcode/php-java/" imgSrc="/total/images/aspose_barcode-for-php-java.svg" productName="Aspose.BarCode for PHP via Java">}}
Aspose.BarCode עבור PHP דרך Java הוא אוסף של ממשקי API ליצירת ברקוד וקריאה המיועדים ליישומי PHP.  הוא מספק למפתחים את היכולת לקרוא סוגים שונים של ברקודים, כולל ברקודים דו-ממדיים, דו-ממדיים וברקודים דואר, מתוך תמונות שצולמו בזוויות שונות.  ה-API של מחולל הברקודים מאפשר יצירת תמונות ברקוד בפורמטים שונים כגון JPG, PNG, GIF ו-BMP.  בנוסף, הוא מציע מספר תכונות הדמיה, כגון מניפולציה בגבול, התאמת צבע, סיבוב תמונה, התאמה אישית של רזולוציה ועוד.  על ידי שימוש ב-API, מפתחים יכולים לשלב בקלות פונקציונליות ברקוד ביישומי PHP שלהם.
