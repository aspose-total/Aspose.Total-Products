{{% blocks/products/pf/agp/feature-section-col title="Jak převést PPT na XLSX?" %}}
1. Otevřete a načtěte soubor PPT pomocí třídy Prezentace

4. K načtení exportovaného HTML použijte třídu Workbook
5. Zavolejte metodu uložení a převeďte ji do formátu XLSX
