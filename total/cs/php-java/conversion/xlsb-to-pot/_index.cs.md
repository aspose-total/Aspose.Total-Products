Nejprve použijte Aspose.Cells pro PHP přes Javu k exportu souboru XLSB do PDF. Poté použijte Aspose.Slides pro PHP přes Java k převodu souboru PDF na POT. Aspose.Cells pro PHP přes Javu a Aspose.Slides pro PHP přes Javu jsou podřízená rozhraní API Aspose.Total pro PHP přes balíček Java API.
