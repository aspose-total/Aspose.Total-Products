Aspose.Total pro PHP přes Javu je kompletní balíček pro manipulaci s tabulkovým procesorem Excel a také pro prezentace PowerPoint a OpenOffice. Vývojáři mohou vytvářet, manipulovat, vykreslovat a převádět soubory Excel prostřednictvím aplikací PHP a také načítat, číst, zapisovat a exportovat formáty PowerPoint včetně PPT, PPTX, PPS, POT, PPSX, OpenOffice ODP a dalších formátů. Vývojáři mohou snadno integrovat PHP API pro rozpoznávání a generování čárových kódů.

{{< blocks/products/pf/total/product-item href="/cells/php-java/" imgSrc="/total/images/aspose_cells-for-php-java.svg" productName="Aspose.Cells for PHP via Java">}}
Aspose.Cells for PHP přes Java API je určen pro správu a zpracování excelových tabulek. Zvládá úkoly, jako je převod formátů tabulek, správa a vykreslování grafů, implementace vzorců, použití podrobných stylů na listy a mnoho dalšího. Díky integraci JavaBridge mohou vývojáři bez problémů začlenit sílu vytváření, úprav, převodu a vykreslování souborů Excel do svých aplikací PHP.
{{< /blocks/products/pf/total/product-item >}}

{{< blocks/products/pf/total/product-item href="/slides/php-java/" imgSrc="/total/images/aspose_slides-for-php-java.svg" productName="Aspose.Slides for PHP via Java">}}
Aspose.Slides pro PHP prostřednictvím knihovny Java umožňuje vývojářům efektivně pracovat s prezentacemi a zpracovávat je. Nabízí čtení, úpravy a převod prezentací PowerPoint bez spoléhání se na externí aplikace. Knihovna umožňuje načítání, otevírání a prohlížení prezentací, jejich převod do různých formátů, vykreslování a tisk prezentací, jejich šifrování a dešifrování, manipulaci s entitami prezentace, jako jsou hlavní snímky, tvary, grafy, audio a video snímky, animace a další.
{{< /blocks/products/pf/total/product-item >}}

{{< blocks/products/pf/total/product-item href="/barcode/php-java/" imgSrc="/total/images/aspose_barcode-for-php-java.svg" productName="Aspose.BarCode for PHP via Java">}}
Aspose.BarCode for PHP via Java je sbírka API pro generování a čtení čárových kódů navržená pro aplikace PHP. Poskytuje vývojářům možnost číst různé typy čárových kódů, včetně 1D, 2D a poštovních čárových kódů, z obrázků pořízených z různých úhlů. API generátoru čárových kódů umožňuje vytvářet obrázky čárových kódů v různých formátech, jako jsou JPG, PNG, GIF a BMP. Navíc nabízí několik zobrazovacích funkcí, jako je manipulace s okraji, úprava barev, rotace obrazu, přizpůsobení rozlišení a další. Využitím API mohou vývojáři snadno začlenit funkce čárových kódů do svých aplikací PHP.
