1. Voláním metody 'save()' uložíte dokument předáním výstupního názvu souboru.

<h2>Vyviňte aplikaci pro Android pro vyhledávání souborů ODP</h2>

Potřebujete vyvinout ODP prohledávatelný mobilní nástroj pro Android? S Aspose.Slides pro Android přes Javu, podřízené API Aspose.Total pro Android přes Javu, může každý vývojář Androidu integrovat výše uvedený kód API do programování textového vyhledávání v dokumentech. Výkonná knihovna pro Android umožňuje programátorům vyvinout řešení pro vyhledávání dokumentů, které podporuje mnoho oblíbených formátů včetně formátu ODP.<br /><br />
