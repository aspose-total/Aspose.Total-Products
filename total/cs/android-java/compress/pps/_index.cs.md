<h2>Vyvinout aplikaci pro kompresi souborů PPS pro Android</h2>

Potřebujete vyvinout skript nebo nástroj pro Android pro snadnou kompresi více souborů PPS? S [Aspose.Slides for Android via Java](https://products.aspose.com/slides/cs/android-java/), podřízenými API [Aspose.Total for Android via Java](https://products.aspose.com/total/cs/android-java/), může každý vývojář Androidu integrovat výše uvedený kód API k naprogramování kompresní aplikace napříč dokumenty. Výkonná knihovna Android pro kompresi dokumentů podporuje mnoho oblíbených formátů včetně formátu PPS.<br /><br />
