[Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) Knihovny Automatizace formátu souborů vám mohou pomoci automatizovat proces převodu dokumentů ve vašich aplikacích pro Android. Poté můžete pomocí PowerPoint API [Aspose.Slides pro Android Java](https://products.aspose.com/slides/android-java/) vytvořit novou prezentaci, napsat do ní obsah HTML a uložit ji jako PPSM.

{{% blocks/products/pf/agp/feature-section-col title="Jak převést RTF na PPSM v Androidu" %}}
1. Otevřete soubor RTF pomocí třídy [Rtfument](https://reference.aspose.com/words/java/com.aspose.words/Rtfument)
2. Převeďte soubor RTF na HTML pomocí [uložit](https://reference.aspose.com/words/java/com.aspose.words/Rtfument#save(java.lang.String,com.aspose.words.SaveOptions) )) metoda
3. Inicializujte nový objekt [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
5. Extrahujte obsah ze souboru HTML pomocí BufferedReader a zapište obsah do souboru prezentace
6. Uložte dokument do PPSM pomocí metody [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
