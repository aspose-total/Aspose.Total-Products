Funkci převodu PCL na CSV můžete integrovat do svých aplikací pro Android ve dvou krocích. Za prvé, pomocí [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/) můžete převést PCL na XLSX. Za druhé, můžete převést XLSX na CSV pomocí Powerful Spreadsheet Processing API [Aspose.Cells pro Android přes Java](https://products.aspose.com/cells/android-java/). Obě rozhraní API spadají do rodiny produktů [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/).

1. Otevřete soubor PCL pomocí třídy [Document](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document)
2. Převeďte PCL na XLSX pomocí [uložit](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document#save-java.lang.String-com.aspose.pdf.SaveOptions-) metoda

4. Uložte dokument do formátu CSV pomocí [uložit](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.SaveOptions)).
