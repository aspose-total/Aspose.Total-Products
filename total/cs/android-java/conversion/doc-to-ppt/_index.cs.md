1. Otevřete soubor DOC pomocí třídy [Document](https://reference.aspose.com/words/java/com.aspose.words/Document)

6. Uložte dokument do PPT pomocí metody [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.io.OutputStream-int-)
