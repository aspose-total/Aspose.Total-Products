Pro vývojáře Pythonu, který se snaží přidat do aplikace funkci převodu JPEG na MSG? Jedná se o dvoukrokový proces, nejprve načtěte soubor JPEG a vykreslete jej do HTML pomocí [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/). - Otevřete zdrojový soubor JPEG pomocí třídy [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)

- Zavolejte metodu uložení s příslušnou cestou k souboru. Takže nakonec je JPEG převeden
