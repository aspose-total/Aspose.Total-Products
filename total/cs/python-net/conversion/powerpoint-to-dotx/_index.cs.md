Pro vývojáře Pythonu, který se snaží přidat funkci převodu POWERPOINT na DOTX v rámci aplikace? [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API může pomoci automatizovat proces převodu. Je to plný balík různých API zabývajících se různými formáty.  

Je to hlavně ve dvou krocích. Nejprve použijte [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) API pro převod POWERPOINT souboru do PDF. Poté pomocí Microsoft Word Python API [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) uložte vytvořený PDF do Microsoft Wordu ve formátu DOTX.

{{% blocks/products/pf/agp/feature-section-col title="Jak převést POWERPOINT na DOTX v Pythonu" %}}
-  **Krok 1** Načtěte soubor PDF s instancí třídy [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)
-  Volejte metodu `save` a zadávejte cestu k výstupnímu souboru a SaveFormat.PDF jako parametry. Váš soubor POWERPOINT je tedy převeden na PDF v zadané cestě.
- **Krok 2** Otevřete soubor PDF pomocí třídy [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)
- Uložte soubor PDF do souboru DOTX pomocí metody [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/) zadáním názvu souboru a požadované cesty k adresáři.
- Zde je další úryvek kódu pro PowerPoint Presentation to Word [Microsoft Powerpoint to Word](https://products.aspose.com/total/python-net/conversion/) převod.
