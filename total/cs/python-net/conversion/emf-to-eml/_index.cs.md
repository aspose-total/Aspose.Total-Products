Pro vývojáře Pythonu, který se snaží přidat do aplikace funkci převodu EMF na EML? Jedná se o dvoukrokový proces, nejprve načtěte soubor EMF a vykreslete jej do HTML pomocí [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/). Za druhé načtěte převedený HTML pomocí [Aspose.Email for Python via .NET](https://products.aspose.com/email/python-net/) a uložte jej do formátu EML.

{{% blocks/products/pf/agp/feature-section-col title="Jak převést EMF na EML v Pythonu" %}}

- Otevřete zdrojový soubor EMF pomocí třídy [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)

- Nyní načtěte uložený soubor HTML pomocí MailMessage.load
- Zavolejte metodu uložení s příslušnou cestou k souboru. Takže nakonec je EMF převeden
