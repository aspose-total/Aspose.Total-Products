<h2>Vyvíjejte aplikaci DOC File Splitter pomocí .NET</h2>

Potřebujete vyvinout skript .NET nebo obslužnou aplikaci pro snadné rozdělení souborů DOC?S [Aspose.Words for .NET](https://products.aspose.com/words/cs/net/), podřízeným API [Aspose.Total for .NET](https://products.aspose.com/total/cs/net/), může každý vývojář .NET integrovat výše uvedený kód API do aplikace splitteru. Výkonná knihovna .NET pro rozdělování dokumentů podporuje mnoho oblíbených formátů včetně formátu DOC, jako jsou soubory PDF, Word, Excel a PowerPoint.<br /><br />
