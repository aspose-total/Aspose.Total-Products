Pomocí [Aspose.Total for .NET](https://products.aspose.com/total/net/) můžete zahrnout funkci převodu DOC na XLSX v jakékoli aplikaci .NET, C#, ASP.NET a VB.NET v dva jednoduché kroky. Poté můžete pomocí rozhraní [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) Spreadsheet Programming API převést HTML na XLSX.
