{{% blocks/products/pf/feature-page-summary %}}
Pomocí [Aspose.Total for .NET](https://products.aspose.com/total/net/) můžete převést soubor PPT na XLTX v rámci libovolné aplikace .NET, C#, ASP.NET a VB.NET ve dvou jednoduché kroky. Za prvé, pomocí [Aspose.Slides for .NET](https://products.aspose.com/slides/net/) můžete exportovat PPT do HTML. Ppté můžete pomocí rozhraní [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) Spreadsheet Programming API převést HTML na XLTX.

{{% blocks/products/pf/agp/feature-section-col title="Jak převést PPT na XLTX přes C#" %}}
1. Otevřete soubor PPT pomocí třídy [Presentation](https://apireference.aspose.com/slides/net/aspose.slides/presentation)
2. Exportujte PPT jako HTML pomocí metody [Save](https://apireference.aspose.com/slides/net/aspose.slides.presentation/save/methods/5)
3. Načtěte dokument HTML pomocí třídy [Workbook](https://apireference.aspose.com/cells/net/aspose.cells/workbook)
4. Uložte dokument do XLTX pomocí metody [Save](https://apireference.aspose.com/cells/net/aspose.cells.workbook/save/methods/4)

{{% blocks/products/pf/feature-page-section  h2="Převést chráněný PPT na XLTX přes C#" %}}
Při převodu souboru PPT na XLTX, pokud je váš vstupní dokument PPT chráněn heslem, nemůžete jej převést na XLTX bez dešifrování dokumentu. Když je váš dokument chráněn heslem, znamená to, že vyžaduje určitá omezení prezentace. Pro odstranění omezení je nutné zadat heslo. Prezentace chráněná heslem je považována za zamčenou prezentaci. Rozhraní API vám umožňuje otevřít zašifrovaný dokument předáním správného hesla v objektu LoadOptions.
