{{% blocks/products/pf/feature-page-section  h2="Omezit úpravy dokumentů DOC prostřednictvím .NET" %}}
Při ukládání dokumentu z EMLX do DOC může být nutné chránit výstupní dokument. Někdy může být nutné omezit možnost upravovat dokument a povolit s ním pouze určité akce. [Aspose.Words for .NET](https://products.aspose.com/words/net/) API vám umožňuje ovládat způsob, jakým omezujete obsah pomocí [ProtectionType](https://reference.aspose.com/words/net/aspose.words/protectiontype) parametr výčtu.
