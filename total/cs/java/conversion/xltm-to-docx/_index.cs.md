Nejprve použijete [Aspose.Cells for Java](https://products.aspose.com/cells/java) API k převodu daného XLTM dokumentu do PDF a poté pomocí [Aspose.PDF for Java](https://products.aspose.com/pdf/java) API, můžete snadno převést dokument PDF na DOCX. {{% blocks/products/pf/agp/feature-section-col title="Jak převést XLTM na DOCX přes Java API" %}}
1. Otevřete soubor XLTM pomocí třídy [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)
2. Převeďte XLTM na PDF a nastavte SaveFormat na AUTO
3. Načtěte převedený soubor PDF pomocí třídy [Document](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document)
4. Uložte dokument ve formátu DOCX pomocí [save](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document#save-java.lang.String-com.aspose.pdf.SaveOptions-) a nastavte Docx jako SaveFormat
