Nejprve použijete [Aspose.Cells for Java](https://products.aspose.com/cells/java) API k převodu daného CSV dokumentu do PDF a poté pomocí [Aspose.PDF for Java](https://products.aspose.com/pdf/java) API, můžete snadno převést dokument PDF na DOCM. {{% blocks/products/pf/agp/feature-section-col title="Jak převést CSV na DOCM přes Java API" %}}
1. Otevřete soubor CSV pomocí třídy [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)
2. Převeďte CSV na PDF a nastavte SaveFormat na AUTO
3. Načtěte převedený soubor PDF pomocí třídy [Document](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document)
4. Uložte dokument ve formátu DOCM pomocí [save](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document#save-java.lang.String-com.aspose.pdf.SaveOptions-) a nastavte Docx jako SaveFormat
