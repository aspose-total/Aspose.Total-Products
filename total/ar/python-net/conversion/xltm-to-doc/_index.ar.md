إنها بشكل أساسي في خطوتين. استخدم أولاً [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) API لتحويل ملف XLTM إلى HTML. بعد ذلك باستخدام Word Python API [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) ، احفظ HTML الذي تم إنشاؤه في تنسيق Microsoft Word المطلوب.

- **الخطوة 1** افتح ملف XLTM المصدر باستخدام فئة Workbook
