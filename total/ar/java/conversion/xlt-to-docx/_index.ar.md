{{% blocks/products/pf/agp/feature-section-col title="كيفية تحويل XLT إلى DOCX عبر Java API" %}}
1. افتح ملف XLT باستخدام فئة [Workbook](https://apireference.aspose.com/cells/java/com.aspose.cells/Workbook)
2. تحويل XLT إلى PDF وتعيين SaveFormat على AUTO

4. احفظ المستند بتنسيق DOCX باستخدام [save](https://apireference.aspose.com/pdf/java/com.aspose.pdf/Docxument#save-java.lang.String-com.aspose.pdf.SaveOptions-) طريقة وتعيين Docx كـ SaveFormat
