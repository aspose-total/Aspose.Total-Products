{{% blocks/products/pf/agp/feature-section-col title="تقديم POT إلى OTT في Android" %}}
1. افتح ملف POT باستخدام فئة [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. تحويل POT إلى HTML باستخدام [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.ISaveOptions-) وتعيين Html كـ SaveFormat
3. قم بتحميل ملف HTML المحول باستخدام فئة [Ottument](https://reference.aspose.com/words/java/com.aspose.words/Ottument)
4. احفظ المستند بتنسيق OTT باستخدام طريقة [save](https://reference.aspose.com/words/java/com.aspose.words/Ottument#save (java.lang.String، int)) وقم بتعيين Ott باسم SaveFormat
