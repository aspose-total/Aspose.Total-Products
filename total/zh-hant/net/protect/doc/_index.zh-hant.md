<h2>透過.NET開發DOC密碼保護應用程式</h2>

需要開發 DOC 密碼保護應用程式或實用程式？透過 [Aspose.Words for .NET](https://products.aspose.com/words/zh-hant/net/)（[Aspose.Total for .NET](https://products.aspose.com/total/zh-hant/net/) 的子 API），任何 .NET 開發人員都可以將上述 API 程式碼整合到其文件保護應用程式中。強大的 .NET 程式庫允許對任何文件密碼保護解決方案進行程式設計以保護和鎖定檔案。而且它可以支援許多流行的格式，包括DOC格式。<br /><br />

我們的產品完全跨平台，支援遵循「.NET Standard 2.0」規範的所有主要 .NET 實作：<br /><br />
