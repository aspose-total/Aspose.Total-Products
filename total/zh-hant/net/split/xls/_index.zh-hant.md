<h2>使用 .NET 開發 XLS 檔案分割器應用程式</h2>

需要開發 .NET 腳本或實用程式應用程式來輕鬆分割 XLS 檔案？借助 [Aspose.Total for .NET](https://products.aspose.com/total/zh-hant/net/) 的子 API [Aspose.Cells for .NET](https://products.aspose.com/cells/zh-hant/net/)，任何 .NET 開發人員都可以將上述 API 程式碼整合到分割器應用程式中。強大的.NET 文件分割庫支援多種流行格式，包括 XLS 格式，如 PDF、Word、Excel 和 PowerPoint 文件。<br /><br />
