<h2>透過.NET 開發 Powerpoint 元資料管理應用程式</h2>

需要開發 Powerpoint 元資料管理應用程式或實用程式？透過 [Aspose.Slides for .NET](https://products.aspose.com/slides/zh-hant/net/)（[Aspose.Total for .NET](https://products.aspose.com/total/zh-hant/net/) 的子 API），任何 .NET 開發人員都可以將上述 API 程式碼整合到其文件元資料應用程式中。強大的.NET 程式庫允許對任何文件元資料解決方案進行程式設計。而且它可以支援許多流行的格式，包括Powerpoint格式。<br /><br />
