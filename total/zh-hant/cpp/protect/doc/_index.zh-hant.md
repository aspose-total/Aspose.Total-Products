<h2>透過C++開發DOC密碼保護應用程式</h2>

需要開發 DOC 密碼保護應用程式或實用程式？透過 [Aspose.Words for C++](https://products.aspose.com/words/cpp/)（[Aspose.Total for C++](https://products.aspose.com/total/zh-hant/cpp/) 的子 API），任何 C++ 開發人員都可以將上述 API 程式碼整合到其文件保護應用程式中。強大的 C++ 程式庫允許對任何文件密碼保護解決方案進行程式設計以保護和鎖定檔案。而且它可以支援許多流行的格式，包括DOC格式。<br /><br />
