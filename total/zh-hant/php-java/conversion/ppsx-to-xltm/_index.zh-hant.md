Aspose.Total for PHP via Java 是一個綜合套件，提供了廣泛的功能，用於處理 Excel 電子表格、Powerpoint 演示文稿以及條形碼生成和識別。 借助 Aspose.Total for PHP via Java，您只需幾個步驟即可輕鬆將 PPSX 到 XLTM 轉換添加到任何 PHP 應用程序。

首先，通過Java使用Aspose.Slides for PHP將PPSX文件導出為HTML。 然後，通過 Java 使用 Aspose.Cells for PHP 將 HTML 文件轉換為 XLTM。 Aspose.Cells for PHP via Java 和 Aspose.Slides for PHP via Java 都是 Aspose.Total for PHP via Java API 包的子 API。

{{% blocks/products/pf/agp/feature-section-col title="如何將 PPSX 轉換為 XLTM？" %}}
1. 使用Presentation類打開並加載PPSX文件
2. 使用 HtmlOptions 對象設置 HTML 保存選項或直接調用 save 方法
3. 使用指定路徑和 SaveFormat.HTML 作為參數的 save 方法將 PPSX 文件保存為 HTML
4. 使用 Workbook 類加載導出的 HTML
5. 調用save方法將其轉換為XLTM格式
