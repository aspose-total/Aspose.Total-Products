對於嘗試在應用程序中添加 XLTM 到 MOBI 轉換功能的 Python 開發人員。 [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API 可以幫助自動化轉換過程。 它是處理不同格式（包括 XLTM 和 MOBI 文件）的各種 API 的完整包。

主要分兩步。 首先使用 [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) API 將 XLTM 文件轉換為 HTML。 之後，通過使用 Word Python API [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/)，將創建的 HTML 保存為所需的 Microsoft Word 格式。

{{% blocks/products/pf/agp/feature-section-col title="如何在 Python 中將 XLTM 轉換為 MOBI" %}}
- **步驟1** 使用 Workbook 類打開源 XLTM 文件
- 通過提供文件名和所需目錄路徑，使用 save(file, SaveFormat.HTML) 方法將 XLTM 文件保存為 HTML
-  **第2步** 使用 [Document](https://reference.aspose.com/words/python-net/aspose.words/document/) 類的實例加載 HTML 文件
-  在指定輸出 MOBI 文件路徑時調用 `save` 方法。 這樣你的XLTM文件就在指定路徑下轉成MOBI了

{{% /blocks/products/pf/agp/feature-section-col %}}

{{% blocks/products/pf/agp/feature-section-col title="轉換要求" %}}

- 對於 XLTM 到 MOBI 的轉換，需要 Python 3.5 或更高版本
- 直接從 PyPI 引用項目中的 API（[Aspose.Cells](https://pypi.org/project/aspose-cells-python/) 和 [Aspose.Words](https://pypi.org/project/aspose-words/)）
-  或者使用以下 pip 命令```pip install aspose-cells-python``` 和```pip install aspose.words```
-  此外，基於 Microsoft Windows 或 Linux 的操作系統（請參閱有關 [Cells](https://docs.aspose.com/cells/python-net/getting-started/#installation) 和 [Words](https://docs.aspose.com/words/python-net/system-requirements/) 的更多信息）和 Linux 檢查 gcc 和 libpython 的附加要求並遵循 [分步說明](https://docs.aspose.com/words/python-net/installation/)
