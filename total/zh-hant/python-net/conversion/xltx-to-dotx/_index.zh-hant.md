- **步驟1** 使用 Workbook 類打開源 XLTX 文件
- 通過提供文件名和所需目錄路徑，使用 save(file, SaveFormat.HTML) 方法將 XLTX 文件保存為 HTML

- 對於 XLTX 到 DOTX 的轉換，需要 Python 3.5 或更高版本
