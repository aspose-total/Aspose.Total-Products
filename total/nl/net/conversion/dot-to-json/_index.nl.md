{{% blocks/products/pf/feature-page-summary %}}
Door [Aspose.Total for .NET](https://products.aspose.com/total/net/) te gebruiken, kunt u DOT naar JSON-indeling converteren binnen elke .NET-, C#-, ASP.NET- en VB.NET-toepassing in twee eenvoudige stappen. Ten eerste, door [Aspose.Words for .NET](https://products.aspose.com/words/net/) te gebruiken, kunt u DOT naar HTML exporteren. Daarna kunt u met behulp van [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) Spreadsheet Programming API HTML naar JSON converteren.
