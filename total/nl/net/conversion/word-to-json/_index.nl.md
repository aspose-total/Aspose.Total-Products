{{% blocks/products/pf/feature-page-summary %}}
Door [Aspose.Total for .NET](https://products.aspose.com/total/net/) te gebruiken, kunt u WORD naar JSON-indeling converteren binnen elke .NET-, C#-, ASP.NET- en VB.NET-toepassing in twee eenvoudige stappen. Ten eerste, door [Aspose.Words for .NET](https://products.aspose.com/words/net/) te gebruiken, kunt u WORD naar HTML exporteren. Daarna kunt u met behulp van [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) Spreadsheet Programming API HTML naar JSON converteren.

1. Open het WORD-bestand met de klasse [Wordument](https://apireference.aspose.com/words/net/aspose.words/wordument)
2. Converteer WORD naar HTML met behulp van de [Save](https://apireference.aspose.com/words/net/aspose.words.wordument/save/methods/4) methode
3. Laad HTML-wordument met behulp van [Workbook](https://apireference.aspose.com/cells/net/aspose.cells/workbook) class
4. Sla het wordument op in JSON-indeling met de methode [Save](https://apireference.aspose.com/cells/net/aspose.cells.workbook/save/methods/4)
