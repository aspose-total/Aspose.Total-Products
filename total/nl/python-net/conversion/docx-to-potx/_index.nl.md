Voor een Python-ontwikkelaar, die probeert een DOCX naar POTX-conversiefunctie toe te voegen binnen de applicatie? Sla daarna met behulp van PowerPoint Python API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) de gemaakte PDF op in Presentatie als een POTX-indeling.

-  Roep de `save`-methode aan terwijl u het pad van het uitvoerbestand & SaveFormat.POTX als parameters opgeeft.  uw DOCX-bestand wordt geconverteerd naar POTX op het opgegeven pad.
