{{% blocks/products/pf/feature-page-summary %}}

Voor een Python-ontwikkelaar, die probeert een PPT naar XLSX-conversiefunctie binnen de applicatie toe te voegen. [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API kan helpen om het conversieproces te automatiseren. Het is een volledig pakket van verschillende API's voor verschillende formaten, waaronder PPT- en XLSX-bestanden.

Het gaat voornamelijk in twee stappen. Gebruik eerst de [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) API om het PPT-bestand naar HTML te converteren. Sla daarna met Excel Python API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) de gemaakte HTML op in de gewenste Microsoft Excel-indeling.

{{% blocks/products/pf/agp/feature-section-col title="Hoe PPT naar XLSX te converteren in Python" %}}
- **Stap 1** Gebruik de [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)-klasse-instantie om het PPT-bronbestand te openen 
- Sla het PPT-bestand op in HTML met behulp van de [save](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)-methode door de bestandsnaam en het gewenste mappad op te geven
-  **Stap 2** Laad een HTML-bestand met een exemplaar van de Workbook-klasse
-  Roep de methode 'opslaan' aan terwijl u het pad naar het uitvoer-XLSX-bestand opgeeft. Dus uw PPT-bestand wordt geconverteerd naar XLSX op het opgegeven pad
