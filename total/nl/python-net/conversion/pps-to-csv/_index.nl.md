{{% blocks/products/pf/feature-page-summary %}}

Voor een Python-ontwikkelaar, die probeert een PPS naar CSV-conversiefunctie binnen de applicatie toe te voegen. [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API kan helpen om het conversieproces te automatiseren. Het is een volledig pakket van verschillende API's voor verschillende formaten, waaronder PPS- en CSV-bestanden.

Het gaat voornamelijk in twee stappen. Gebruik eerst de [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) API om het PPS-bestand naar HTML te converteren. Sla daarna met Excel Python API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) de gemaakte HTML op in de gewenste Microsoft Excel-indeling.

{{% blocks/products/pf/agp/feature-section-col title="Hoe PPS naar CSV te converteren in Python" %}}
- **Stap 1** Gebruik de [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)-klasse-instantie om het PPS-bronbestand te openen 
- Sla het PPS-bestand op in HTML met behulp van de [save](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)-methode door de bestandsnaam en het gewenste mappad op te geven
-  **Stap 2** Laad een HTML-bestand met een exemplaar van de Workbook-klasse
-  Roep de methode 'opslaan' aan terwijl u het pad naar het uitvoer-CSV-bestand opgeeft. Dus uw PPS-bestand wordt geconverteerd naar CSV op het opgegeven pad
