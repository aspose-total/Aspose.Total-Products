{{% blocks/products/pf/feature-page-summary %}}

Voor een Python-ontwikkelaar, die probeert een XLTX naar POTX-conversiefunctie binnen de applicatie toe te voegen, [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API kan helpen om het conversieproces te automatiseren. Het is een volledig pakket van verschillende API's voor verschillende formaten, waaronder XLTX- en POTX-bestanden.

Het gaat voornamelijk in twee stappen. Gebruik eerst de [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) API om het XLTX-bestand naar PDF te converteren. Sla daarna met behulp van PowerPoint Python API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) de gemaakte PDF op in de gewenste Microsoft PowerPoint-indeling.

{{% blocks/products/pf/agp/feature-section-col title="Hoe XLTX naar POTX te converteren in Python" %}}
- **Stap 1** Gebruik de Workbook-klasse-instantie om het XLTX-bronbestand te openen 
- Sla het XLTX-bestand op in PDF met behulp van de save-methode door de bestandsnaam en het gewenste mappad op te geven
-  **Stap 2** Laad PDF-bestand met behulp van [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)-klasse
-  Roep de [save](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)-methode aan terwijl u het uitvoerpad van het POTX-bestand opgeeft. Dus uw XLTX-bestand wordt geconverteerd naar POTX op het opgegeven pad
