{{% blocks/products/pf/feature-page-summary %}}

Voor een Python-ontwikkelaar, die probeert een OTT naar XLTM-conversiefunctie binnen de applicatie toe te voegen. [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API kan helpen om het conversieproces te automatiseren. Het is een volledig pakket van verschillende API's die verschillende formaten behandelen.

Het gaat voornamelijk in twee stappen. Gebruik eerst de [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/) API om het OTT-bestand naar HTML te converteren. Sla daarna met Excel Python API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) de gemaakte HTML op in de gewenste Microsoft Excel-indeling.

{{% blocks/products/pf/agp/feature-section-col title="Hoe OTT naar XLTM te converteren in Python" %}}
- **Stap 1** Open het OTT-bronbestand met de [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)-klasse
- Sla het OTT-bestand op in HTML met behulp van de [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/)-methode door de bestandsnaam en het gewenste mappad op te geven
-  **Stap 2** Laad het HTML-bestand met een exemplaar van de Workbook-klasse met bestand en LoadOptions als parameters
-  Roep de methode 'opslaan' aan terwijl u het pad naar het uitvoer-XLTM-bestand opgeeft. Dus uw OTT-bestand wordt geconverteerd naar XLTM op het opgegeven pad

- Voor conversie van OTT naar XLTM is Python 3.5 of hoger vereist
- Referentie-API's binnen het project rechtstreeks vanuit PyPI ([Aspose.Words](https://pypi.org/project/aspose-words/) en [Aspose.Cells](https://pypi.org/project/aspose-cells-python/))
