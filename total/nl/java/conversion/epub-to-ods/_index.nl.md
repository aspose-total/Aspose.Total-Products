Door [Aspose.Total for Java](https://products.aspose.com/total/java/) te gebruiken, kunt u de conversiefunctie van EPUB naar ODS in uw Java-toepassingen integreren in een proces van twee stappen. In de tweede stap kunt u XLSX naar ODS converteren met behulp van de Spreadsheet Programming API [Aspose.Cells for Java](https://products.aspose.com/cells/java/).
