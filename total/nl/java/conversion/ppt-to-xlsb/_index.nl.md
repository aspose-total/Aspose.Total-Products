U kunt het PPT-bestand in twee stappen converteren naar XLSB via [Aspose.Total for Java](https://products.aspose.com/total/java/). In de eerste stap kunt u PPT naar HTML exporteren met [Aspose.Slides for Java](https://products.aspose.com/slides/java/). Ten tweede kunt u met [Aspose.Cells for Java](https://products.aspose.com/cells/java/) HTML naar XLSB converteren.

{{% blocks/products/pf/agp/feature-section-col title="Hoe PPT naar XLSB te converteren via Java" %}}
1. Open het PPT-bestand met de klasse [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Converteer PPT naar HTML met behulp van [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.ISaveOptions-) methode
3. Laad HTML-document met behulp van [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook) klasse
4. Sla het document op in XLSB-indeling met [save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.SaveOptions)) methode
{{% /blocks/products/pf/agp/feature-section-col %}}
{{% blocks/products/pf/agp/feature-section-col title="Conversievereisten" %}}
Om PPT naar XLSB te converteren, kunt u Aspose.Total voor Java eenvoudig rechtstreeks vanuit een [Maven](https://releases.aspose.com/total/java/) gebaseerd project en neem bibliotheken op in uw pom.xml.

U kunt ook een ZIP-bestand krijgen van [downloads](https://releases.aspose.com/total/java).
