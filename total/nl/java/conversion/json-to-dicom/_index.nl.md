Met behulp van de API kunt u JSON ook converteren naar DICOM met watermerk in uw DICOM-document. Om een watermerk toe te voegen kan je eerst JSON naar JPEG converteren en er een watermerk aan toevoegen. Om een watermerk toe te voegen, laadt u een afbeeldingsbestand met behulp van de klasse [Image](https://reference.aspose.com/imaging/java/com.aspose.imaging/Image), maakt u een object van de [Graphics](https://reference.aspose.com/imaging/java/com.aspose.imaging/Graphics) en initialiseer deze met Image-object, maak een nieuwe [Matrix](https://reference.aspose.com/imaging/java/com.aspose.imaging/Matrix) object en stel de vertaling en transformatie in op de gewenste hoek en voeg een watermerk toe met [Graphics.drawString](https://reference.aspose.com/imaging/java/com.aspose.imaging/Graphics#drawString-java.lang.String-com.aspose.imaging.Font-com.aspose.imaging.Brush-float-float-) methode. Nadat u het watermerk aan uw afbeelding hebt toegevoegd, kunt u de JPEG opslaan als DICOM-indeling.
