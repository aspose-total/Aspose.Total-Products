U kunt de conversiefunctie van EPUB naar SXC in uw Android-applicaties in twee stappen integreren. Ten eerste, door [Aspose.PDF voor Android via Java](https://products.aspose.com/pdf/android-java/) te gebruiken, kunt u EPUB verbergen naar XLSX. Ten tweede kunt u XLSX naar SXC converteren met behulp van de krachtige API voor spreadsheetverwerking [Aspose.Cells voor Android via Java](https://products.aspose.com/cells/android-java/). Beide API's vallen onder de productfamilie [Aspose.Total voor Android via Java](https://products.aspose.com/total/android-java/).
