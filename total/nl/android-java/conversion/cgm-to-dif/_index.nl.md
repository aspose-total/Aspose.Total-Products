U kunt de conversiefunctie van CGM naar DIF in uw Android-applicaties in twee stappen integreren. Ten eerste, door [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/) te gebruiken, kunt u CGM verbergen naar XLSX. Ten tweede kunt u XLSX naar DIF converteren met behulp van de krachtige API voor spreadsheetverwerking [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/). Beide API's vallen onder de productfamilie [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/).
