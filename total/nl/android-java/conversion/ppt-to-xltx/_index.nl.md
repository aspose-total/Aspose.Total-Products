{{% blocks/products/pf/feature-page-summary %}}
U kunt het PPT-bestand eenvoudig naar XLTX converteren in uw Android-applicaties in een proces van twee stappen via [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/). In de eerste stap kunt u het PPT-bestand naar HTML exporteren met [Aspose.Slides for Android via Java](https://products.aspose.com/slides/android-java/). Ten tweede, door [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/) te gebruiken, kunt u HTML naar XLTX converteren.

{{% blocks/products/pf/agp/feature-section-col title="Hoe PPT naar XLTX in Android te converteren" %}}
1. Open het PPT-bestand met de klasse [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation)
2. Converteer PPT naar HTML met behulp van [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-com.aspose.slides.ISaveOptions-) methode
3. Laad HTML-document met behulp van [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook) klasse
4. Sla het document op in XLTX-formaat met [save](https://reference.aspose.com/cells/java/com.aspose.cells/)
