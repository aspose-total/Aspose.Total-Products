<h2>Ontwikkel een Android-applicatie voor het zoeken naar XLS-bestanden</h2>

Wilt u een op XLS doorzoekbaar Android-gebaseerd mobiel hulpprogramma ontwikkelen? Met Aspose.Cells voor Android via Java, een onderliggende API van Aspose.Total voor Android via Java, kan elke Android-ontwikkelaar de bovenstaande API-code integreren om tekstzoekopdrachten in documenten te programmeren. Dankzij de krachtige Android-bibliotheek kunnen programmeurs een oplossing voor het zoeken naar documenten ontwikkelen die veel populaire formaten kan ondersteunen, waaronder het XLS-formaat.<br /><br />
