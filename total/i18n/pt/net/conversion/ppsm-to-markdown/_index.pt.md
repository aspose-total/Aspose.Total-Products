{{% blocks/products/pf/feature-page-section  h2="Converter PPSM Protegido para MARKDOWN via C#" %}}
Ao converter o arquivo PPSM para MARKDOWN, se o documento PPSM de entrada estiver protegido por senha, você não poderá convertê-lo para MARKDOWN sem descriptografar o documento. Quando seu documento é protegido por senha, isso significa que ele impõe certas restrições na apresentação. Para remover as restrições, a senha deve ser inserida. Uma apresentação protegida por senha é considerada uma apresentação bloqueada. A API permite que você abra o documento criptografado passando a senha correta em um objeto LoadOptions.
