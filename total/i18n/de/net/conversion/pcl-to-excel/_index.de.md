Mit [Aspose.Total for .NET](https://products.aspose.com/total/net/) können Sie PCL-Dateien in allen .NET-, C#-, ASP.NET- und VB.NET-Anwendungen problemlos in EXCEL konvertieren. Erstens können Sie mit [Aspose.PDF for .NET](https://products.aspose.com/pdf/net/) PCL nach XLSX exportieren. Danach können Sie mithilfe der [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) Spreadsheet Programming API XLSX in EXCEL konvertieren.

{{% blocks/products/pf/agp/feature-section-col title=".NET-API zum Konvertieren von PCL in EXCEL" %}}
1. Öffnen Sie die PCL-Datei mit der Klasse [Document](https://apireference.aspose.com/pdf/net/aspose.pdf/document).
2. Konvertieren Sie PCL mit der Methode [Save](https://apireference.aspose.com/pdf/net/aspose.pdf.document/save/methods/5) in XLSX
3. Laden Sie das XLSX-Dokument mithilfe der Klasse [Workbook](https://apireference.aspose.com/cells/net/aspose.cells/workbook).
4. Speichern Sie das Dokument mit der Methode [Save](https://apireference.aspose.com/cells/net/aspose.cells.workbook/save/methods/4) im EXCEL-Format und legen Sie „Excel“ als SaveFormat fest

{{% blocks/products/pf/feature-page-section  h2="Konvertieren Sie geschütztes PCL in EXCEL über C#" %}}
Wenn Ihr PCL-Dokument passwortgeschützt ist, können Sie es ohne das Passwort nicht in EXCEL konvertieren. Mit der API können Sie das geschützte Dokument zunächst mit einem gültigen Passwort öffnen und anschließend konvertieren. Um die verschlüsselte Datei zu öffnen, können Sie eine neue Instanz der Klasse [Document](https://apireference.aspose.com/pdf/net/aspose.pdf/document) initialisieren und den Dateinamen und das Passwort als Argumente übergeben.
