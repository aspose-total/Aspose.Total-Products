{{% blocks/products/pf/feature-page-summary %}}
Utilizzando [Aspose.Total for .NET](https://products.aspose.com/total/net/) puoi includere la funzione di conversione da DOCM a SXC all'interno di qualsiasi applicazione .NET, C#, ASP.NET e VB.NET in due semplici passaggi. Innanzitutto, utilizzando [Aspose.Words for .NET](https://products.aspose.com/words/net/), puoi esportare DOCM in HTML. Successivamente, utilizzando l'API di programmazione per fogli di calcolo [Aspose.Cells for .NET](https://products.aspose.com/cells/net/), è possibile convertire HTML in SXC.

{{% blocks/products/pf/agp/feature-section-col title="API .NET per convertire DOCM in SXC" %}}
1. Aprire il file DOCM utilizzando la classe [Docmument](https://apiference.aspose.com/words/net/aspose.words/docmument)
2. Converti DOCM in HTML utilizzando il metodo [Save](https://apiference.aspose.com/words/net/aspose.words.docmument/save/methods/4)
3. Caricare il docmumento HTML utilizzando la classe [Workbook](https://apiference.aspose.com/cells/net/aspose.cells/workbook)
4. Salvare il docmumento in formato SXC utilizzando il metodo [Salva](https://apiference.aspose.com/cells/net/aspose.cells.workbook/save/methods/4) e impostare `SXC` come SaveFormat

{{% blocks/products/pf/feature-page-section  h2="Aggiungi proprietà personalizzate nel file SXC tramite C#" %}}
Durante la conversione di DOCM in SXC, [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) consente di aggiungere proprietà personalizzate nei docmumenti SXC. Per aggiungere una proprietà personalizzata, è possibile utilizzare il metodo [Add](https://apiference.aspose.com/cells/net/aspose.cells.properties/customdocmumentpropertycollection/methods/add/index) per il metodo [CustomDocmumentPropertyCollection]( https://apiference.aspose.com/cells/net/aspose.cells.properties/customdocmumentpropertycollection). Il metodo Add aggiunge la proprietà al file Excel e restituisce un riferimento per la nuova proprietà del docmumento come [Aspose.Cells.Properties.DocmumentProperty](https://apiference.aspose.com/cells/net/aspose.cells.properties /proprietà docmumento).
