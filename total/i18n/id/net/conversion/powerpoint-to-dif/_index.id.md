{{% blocks/products/pf/feature-page-section  h2="Konversikan POWERPOINT Terproteksi ke DIF melalui C#" %}}
Saat mengonversi file POWERPOINT ke DIF, jika dokumen POWERPOINT input Anda dilindungi kata sandi, Anda tidak dapat mengonversinya ke DIF tanpa mendekripsi dokumen. Ketika dokumen Anda dilindungi kata sandi, itu berarti memberlakukan pembatasan tertentu pada presentasi. Untuk menghapus batasan, kata sandi harus dimasukkan. Presentasi yang dilindungi kata sandi dianggap sebagai presentasi yang terkunci. API memungkinkan Anda untuk membuka dokumen terenkripsi dengan meneruskan kata sandi yang benar di objek LoadOptions.
