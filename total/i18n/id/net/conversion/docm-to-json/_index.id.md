Dengan menggunakan [Aspose.Total for .NET](https://products.aspose.com/total/net/) Anda dapat mengonversi format DOCM ke JSON dalam aplikasi .NET, C#, ASP.NET, dan VB.NET apa pun dalam dua langkah-langkah sederhana. Pertama, dengan menggunakan [Aspose.Words for .NET](https://products.aspose.com/words/net/), Anda dapat mengekspor DOCM ke HTML.
