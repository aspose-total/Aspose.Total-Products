Используя [Aspose.Total для .NET](https://products.aspose.com/total/net/), вы можете включить функцию преобразования MOBI в XLSX в любое приложение .NET, C#, ASP.NET и VB.NET в два простых шага. После этого, используя [Aspose.Cells for .NET](https://products.aspose.com/cells/net/) API для программирования электронных таблиц, вы можете конвертировать HTML в XLSX.

1. Откройте файл MOBI, используя класс [Document](https://reference.aspose.com/words/net/aspose.words/mobiument).
2. Преобразуйте MOBI в HTML, используя метод [Сохранить](https://reference.aspose.com/words/net/aspose.words.mobiument/save/methods/4).
3. Загрузите HTML-документ с помощью класса [Workbook](https://reference.aspose.com/cells/net/aspose.cells/workbook).
4. Сохраните документ в формате XLSX с помощью метода [Сохранить](https://reference.aspose.com/cells/net/aspose.cells.workbook/save/methods/4) и установите «XLSX» в качестве формата сохранения.
