Вам нужно разработать мобильную утилиту на базе Android с возможностью поиска в формате EXCEL? Благодаря Aspose.Cells для Android через Java, дочернему API Aspose.Total для Android через Java, любой разработчик Android может интегрировать приведенный выше код API для программирования текстового поиска в документах. Мощная библиотека Android позволяет программистам разрабатывать решение для поиска документов, поддерживающее многие популярные форматы, включая формат EXCEL.<br /><br />
