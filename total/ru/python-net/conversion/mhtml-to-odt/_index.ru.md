{{% blocks/products/pf/agp/feature-section-col title="Как преобразовать MHTML в ODT в Python" %}}

- Откройте исходный файл MHTML, используя класс [Document](https://reference.aspose.com/words/python-net/aspose.words/document/).
- Вызовите метод `save`, указав путь к выходному файлу в качестве параметра. Таким образом, ваш файл MHTML преобразуется в ODT по указанному пути.
