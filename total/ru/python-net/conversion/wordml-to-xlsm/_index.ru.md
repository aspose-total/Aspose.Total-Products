{{% blocks/products/pf/agp/feature-section-col title="Как конвертировать WORDML в XLSM в Python" %}}
- **Шаг 1** Откройте исходный WORDML-файл, используя класс [Document](https://reference.aspose.com/words/python-net/aspose.words/document/).
- Сохраните файл WORDML в HTML, используя метод [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/), указав имя файла и желаемый путь к каталогу.
-  **Шаг 2** Загрузите файл HTML с экземпляром класса Workbook с файлом и параметрами LoadOptions в качестве параметров.
-  Вызовите метод `save`, указав путь к выходному файлу XLSM. Таким образом, ваш файл WORDML преобразуется в XLSM по указанному пути.
