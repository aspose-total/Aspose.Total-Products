Для разработчика Python, который пытается добавить в приложение функцию преобразования XLS в PPT., [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API может помочь автоматизировать процесс конвертации. Это полный пакет различных API, работающих с различными форматами, включая файлы XLS и PPT..

Это в основном в два этапа. Сначала используйте API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) для преобразования файла XLS в PDF.. После этого с помощью PowerPoint Python API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) сохраните созданный PDF-файл в желаемом формате Microsoft PowerPoint..

{{% blocks/products/pf/agp/feature-section-col title="Как конвертировать XLS в PPT в Python" %}}
- **Шаг 1** Используйте экземпляр класса Workbook, чтобы открыть исходный файл XLS. 
- Сохраните файл XLS в PDF, используя метод save, указав имя файла и желаемый путь к каталогу.
-  **Шаг 2** Загрузите файл PDF, используя класс [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)
-  Вызовите метод [save](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/), указав путь к выходному файлу PPT.. Таким образом, ваш файл XLS преобразуется в PPT по указанному пути.
