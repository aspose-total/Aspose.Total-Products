- **Шаг 1** Откройте исходный файл WORDML, используя класс [Document](https://reference.aspose.com/words/python-net/aspose.words/document/).

-  Вызовите метод `save`, указав путь к выходному файлу и SaveFormat.PPSX в качестве параметров. им образом, ваш файл WORDML преобразуется в PPSX по указанному пути.
