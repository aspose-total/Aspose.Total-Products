{{% blocks/products/pf/agp/feature-section-col title="Как конвертировать ODT в XLTX в Python" %}}
- **Шаг 1** Откройте исходный ODT-файл, используя класс [Document](https://reference.aspose.com/words/python-net/aspose.words/document/).
- Сохраните файл ODT в HTML, используя метод [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/), указав имя файла и желаемый путь к каталогу.
-  **Шаг 2** Загрузите файл HTML с экземпляром класса Workbook с файлом и параметрами LoadOptions в качестве параметров.
-  Вызовите метод `save`, указав путь к выходному файлу XLTX. Таким образом, ваш файл ODT преобразуется в XLTX по указанному пути.
