DOTM — `.dotm` шаблоны объединяют в себе шаблонизацию с поддержкой макросов, обеспечивая возможность создания многократно используемых шаблонов с поддержкой макросов для автоматизированных действий с документами. Создание файлов DOTM на основе Python позволяет распространять шаблоны, включающие утвержденную автоматизацию VBA, при этом программно управляя содержимым макросов.



Этот формат подходит организациям, которым необходимы повторяющиеся шаблоны с макросами для локальных автоматизированных задач, обеспечивая контроль над макросами, их проверку и при необходимости подписание в процессе создания шаблона.

Автоматизация обеспечивает безопасное управление жизненным циклом макросов — их создание, анализ, подписание и развертывание — в рамках конвейеров CI/CD.
