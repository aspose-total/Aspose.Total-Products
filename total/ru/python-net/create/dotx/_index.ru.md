Python API могут создавать DOTX шаблоны со стилями, элементами управления содержимым и заполнителями содержимого, обеспечивая надежное макро-безопасное шаблонирование для автоматизированных систем.



DOTX подходит для сред, которые запрещают макросы по соображениям безопасности, но все еще требуют последовательного шаблонирования и динамического заполнения содержимого.
