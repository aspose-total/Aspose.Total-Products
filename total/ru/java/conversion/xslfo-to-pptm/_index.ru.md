После преобразования XSLFO в PPTM вы также можете добавить предопределенный тип представления для своей презентации. [Aspose.Slides for Java](https://products.aspose.com/slides/java/) предоставляет возможность установить тип представления для сгенерированной презентации, когда она открывается в PowerPoint через [ViewProperties](https://reference.aspose.com/slides/java/com.aspose.slides/ViewProperties). // save the presentation as Pptm format
```java
presentation.save("output.pptm", SaveFormat.Pptm);    
```
