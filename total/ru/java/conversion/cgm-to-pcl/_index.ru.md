Вы можете преобразовать CGM в PCL, выполнив два простых шага. Сначала вам нужно преобразовать файл CGM в DOC, используя [Aspose.PDF для Java](https://products.aspose.com/pdf/java/). После этого, используя мощный API обработки документов [Aspose.Words for Java](https://products.aspose.com/words/java/), вы можете преобразовать DOC в PCL. Оба API входят в пакет [Aspose.Total for Java](https://products.aspose.com/total/java/).
