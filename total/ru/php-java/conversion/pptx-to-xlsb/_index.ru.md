{{% blocks/products/pf/agp/feature-section-col title="Как конвертировать PPTX в XLSB?" %}}
1. Откройте и загрузите файл PPTX, используя класс Presentation
2. Установите параметры сохранения HTML с помощью объекта HtmlOptions или напрямую вызовите метод сохранения.
3. Сохраните файл PPTX в HTML, используя метод сохранения с указанным путем и SaveFormat.HTML в качестве параметров.
4. Используйте класс Workbook для загрузки экспортированного HTML
5. Вызовите метод сохранения, чтобы преобразовать его в формат XLSB.

1. Установлен PHP 7 или выше
2. Компьютер должен иметь Oracle JDK 7 или более позднюю версию.
