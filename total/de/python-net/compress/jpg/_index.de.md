{{% blocks/products/pf/agp/feature-section-col title="Komprimieren Sie die JPG-Datei über Python" %}}

1. Referenzieren Sie APIs innerhalb des Projekts direkt aus PyPI ([Aspose.Imaging](https://pypi.org/project/aspose-imaging-python-net/))
1. Laden Sie JPG mithilfe der Image-Klasse.
1. Erstellen Sie ein Objekt der Klasse „PngOptions“ oder einer relevanten Formatoptionsklasse.
1. Legen Sie die Eigenschaft „compression_level“ für die Komprimierungsstufe auf einen Wert zwischen 0 und 9 fest.
1. Rufen Sie die Speichermethode auf und rufen Sie die komprimierte JPG-Datei ab.
