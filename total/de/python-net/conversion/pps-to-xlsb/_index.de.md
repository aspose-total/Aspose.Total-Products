{{% blocks/products/pf/feature-page-summary %}}

Für einen Python-Entwickler, der versucht, eine PPS-zu-XLSB-Konvertierungsfunktion in der Anwendung hinzuzufügen. [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API kann helfen, den Konvertierungsprozess zu automatisieren. Es ist ein vollständiges Paket verschiedener APIs, die verschiedene Formate verarbeiten, einschließlich PPS- und XLSB-Dateien.

Es ist hauptsächlich in zwei Schritten. Verwenden Sie zunächst die [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/)-API, um die PPS-Datei in HTML zu konvertieren. Speichern Sie danach mit der Excel-Python-API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) den erstellten HTML-Code im gewünschten Microsoft Excel-Format.

{{% blocks/products/pf/agp/feature-section-col title="So konvertieren Sie PPS in XLSB in Python" %}}
- **Schritt 1** Verwenden Sie die [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)-Klasseninstanz, um die PPS-Quelldatei zu öffnen 
- Speichern Sie die PPS-Datei in HTML mit der [save](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)-Methode, indem Sie den Dateinamen und den gewünschten Verzeichnispfad angeben
-  **Schritt 2** HTML-Datei mit einer Instanz der Workbook-Klasse laden
-  Rufen Sie die `save`-Methode auf, während Sie den Ausgabe-XLSB-Dateipfad angeben. Ihre PPS-Datei wird also unter dem angegebenen Pfad in XLSB konvertiert
