Für einen Python-Entwickler, der versucht, eine PPSM-zu-EXCEL-Konvertierungsfunktion in der Anwendung hinzuzufügen. Es ist hauptsächlich in zwei Schritten. Verwenden Sie zunächst die [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/)-API, um die PPSM-Datei in HTML zu konvertieren. Speichern Sie danach mit der Excel-Python-API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) den erstellten HTML-Code im gewünschten Microsoft Excel-Format.
