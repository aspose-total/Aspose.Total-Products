Es ist hauptsächlich in zwei Schritten. Verwenden Sie zunächst die [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/)-API, um DOTX-Dateien in PDF zu konvertieren. Speichern Sie danach mit der PowerPoint-Python-API [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/) das erstellte PDF in der Präsentation als PPS-Format.

{{% blocks/products/pf/agp/feature-section-col title="Wie konvertiert man DOTX in PPS in Python" %}}
- **Schritt 1** Öffnen Sie die DOTX-Quelldatei mit der Klasse [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)
- Speichern Sie die DOTX-Datei im PDF-Format mit der [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/)-Methode, indem Sie den Dateinamen und den gewünschten Verzeichnispfad angeben.
-  **Schritt 2** Laden Sie eine PDF-Datei mit einer Instanz der [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)-Klasse
-  Rufen Sie die `save`-Methode auf, während Sie den Pfad der Ausgabedatei und SaveFormat.PPS als Parameter angeben. Ihre DOTX-Datei wird also unter dem angegebenen Pfad in PPS konvertiert.
