Die [Aspose.Words for Python via .NET](https://products.aspose.com/words/python-net/)-API, die Teil des [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/)-Pakets ist, erleichtert den Export und Import dieser Konvertierung in MHTML, PDF und verschiedene andere Formate mit Python.
