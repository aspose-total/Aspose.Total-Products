{{% blocks/products/pf/feature-page-summary %}}

Für einen Python-Entwickler, der versucht, eine PPSX-zu-XLSX-Konvertierungsfunktion in der Anwendung hinzuzufügen. [Aspose.Total for Python via .NET](https://products.aspose.com/total/python-net/) API kann helfen, den Konvertierungsprozess zu automatisieren. Es ist ein vollständiges Paket verschiedener APIs, die verschiedene Formate verarbeiten, einschließlich PPSX- und XLSX-Dateien.

Es ist hauptsächlich in zwei Schritten. Verwenden Sie zunächst die [Aspose.Slides for Python via .NET](https://products.aspose.com/slides/python-net/)-API, um die PPSX-Datei in HTML zu konvertieren. Speichern Sie danach mit der Excel-Python-API [Aspose.Cells for Python via .NET](https://products.aspose.com/cells/python-net/) den erstellten HTML-Code im gewünschten Microsoft Excel-Format.

- **Schritt 1** Verwenden Sie die [Presentation](https://reference.aspose.com/slides/python-net/aspose.slides/presentation/)-Klasseninstanz, um die PPSX-Quelldatei zu öffnen

-  **Schritt 2** HTML-Datei mit einer Instanz der Workbook-Klasse laden
-  Rufen Sie die `save`-Methode auf, während Sie den Ausgabe-XLSX-Dateipfad angeben. Ihre PPSX-Datei wird also unter dem angegebenen Pfad in XLSX konvertiert
