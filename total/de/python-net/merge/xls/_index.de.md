{{% blocks/products/pf/agp/feature-section-col title="So führen Sie XLS-Dateien online mit der App zusammen" %}}

1. Laden Sie XLS-Dateien hoch, um sie zu einer einzigen zusammenzuführen.
1. Geben Sie die Parameter wie die Auswahl aus dem Dropdown-Menü ein.
1. Speichern Sie im erforderlichen Format: XLSX, PPTX, PDF, DOCX, DOC, Image, HTML und andere

1. Laden Sie die zusammengeführte XLS-Datei herunter.
1. Sehen Sie sich das Ergebnis der Zusammenführung von XLS als einzelne Datei an und erhalten Sie es.

{{% blocks/products/pf/agp/feature-section-col title="So führen Sie eine XLS-Datei über Python zusammen" %}}

1. Referenzieren Sie APIs innerhalb des Projekts direkt aus PyPI ([Aspose.Cells](https://pypi.org/project/aspose-cells/))
1. Laden Sie Dateien mit dem Workbook-Klassenobjekt.
1. Wählen Sie die Arbeitsmappe als Basisdatei aus.
1. Verwenden Sie die Combine-Methode unter Verwendung des Basisobjekts und des zweiten XLS-Dateiobjekts als Parameter.
1. Rufen Sie die Speichermethode auf und holen Sie sich die zusammengeführten XLS-Dateien in eine.
