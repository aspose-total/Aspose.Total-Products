{{% blocks/products/pf/agp/feature-section-col title="Lesen Sie die EXCEL-Datei über Python für die Viewer-App" %}}

1. Referenzieren Sie erforderliche Pakete innerhalb des Projekts direkt aus PyPI ([Aspose.Cells](https://pypi.org/project/aspose-cells/))
1. Erstellen Sie ein Arbeitsmappenklassenobjekt und laden Sie damit die EXCEL-Quelldatei.
1. Holen Sie sich die gesamte Arbeitsblattsammlung.
1. Durchlaufen Sie jedes Blatt sowie jede Zeile und Zelle, um den Wert zu ermitteln.
1. Um es anzuzeigen, konvertieren Sie die EXCEL-Quelldatei mit der Speichermethode derselben API in HTML.
1. Und verwenden Sie subprocess.run, um den Browser zum Anzeigen der EXCEL-Datei aufzurufen.
1. Oder lesen Sie die Datei in den Speicher und integrieren Sie sie entsprechend in die Python-Viewer-Anwendung.
