<h2>PDF-Rotationsanwendung mit C++ entwickeln</h2>

Müssen Sie eine PDF-Rotations-App oder ein Dienstprogramm entwickeln?Da [Aspose.PDF for C++](https://products.aspose.com/pdf/cpp/) eine untergeordnete API von [Aspose.Total for C++](https://products.aspose.com/total/de/cpp/) ist, kann jeder C++-Entwickler den obigen API-Code in seine Dokumentrotationsanwendung integrieren.Eine leistungsstarke C++-Bibliothek ermöglicht die Programmierung beliebiger Dokumentenrotationslösungen.Darüber hinaus unterstützt es viele gängige Formate, einschließlich des PDF-Formats.<br /><br />
