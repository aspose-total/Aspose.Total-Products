Wenn Ihr Dokument passwortgeschützt ist, bedeutet dies, dass bestimmte Einschränkungen für die Präsentation erzwungen werden. Um die Einschränkungen aufzuheben, muss das Passwort eingegeben werden. Eine passwortgeschützte Präsentation gilt als gesperrte Präsentation. Die API ermöglicht es Ihnen, das verschlüsselte Dokument zu öffnen, indem Sie das richtige Kennwort in einem LoadOptions-Objekt übergeben.
