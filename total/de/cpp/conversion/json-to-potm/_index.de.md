1. Erstellen Sie ein neues [IWorkbook](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook)-Objekt und lesen Sie gültige JSON-Daten aus der Datei

Um dieselbe Spaltenbreite für alle Spalten im Arbeitsblatt festzulegen, verwenden Sie die Methode [SetStandardWidth](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_cell#a48f5dbccc3bf4bb9e6e882094b500bd7) der ICells-Sammlung.

{{% blocks/products/pf/feature-page-section  h2="Konvertieren Sie das JSON-Format in POTM mit Wasserzeichen in C++" %}}
Mit der API können Sie auch JSON in POTM mit Wasserzeichen konvertieren. Um Ihrem POTM-Dokument ein Wasserzeichen hinzuzufügen, können Sie zuerst JSON in PPTX parsen und ein Wasserzeichen hinzufügen.
