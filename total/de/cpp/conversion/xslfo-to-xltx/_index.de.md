{{% blocks/products/pf/agp/feature-section-col title="C++-API zum Konvertieren von XSLFO in XLTX" %}}
1. Öffnen Sie die XSLFO-Datei mit der Klassenreferenz [Document](https://reference.aspose.com/pdf/cpp/class/aspose.pdf.document).
2. Konvertieren Sie XSLFO in XLSX, indem Sie die Member-Funktion [Save](https://reference.aspose.com/pdf/cpp/class/aspose.pdf.document#a6383c010776212483f51cc41235924db) verwenden
3. Laden Sie das XLSX-Dokument mithilfe der Klassenreferenz [IWorkbook](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook).
4. Speichern Sie das Dokument im XLTX-Format mit der Member-Funktion [Save](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook#a9460f52a2dec8f4bf623a4905167d997).
