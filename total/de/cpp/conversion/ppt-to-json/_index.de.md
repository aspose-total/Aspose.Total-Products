{{% blocks/products/pf/feature-page-summary %}}
Durch die Verwendung von [Aspose.Total for C++](https://products.aspose.com/total/cpp/) können Sie PPT-Dateien innerhalb von C++-Anwendungen in zwei einfachen Schritten in das JSON-Format konvertieren. Erstens können Sie mit [Aspose.Slides for C++](https://products.aspose.com/slides/cpp/) PPT in HTML exportieren. Danach können Sie mithilfe der Spreadsheet Programming API [Aspose.Cells for C++](https://products.aspose.com/cells/cpp/) HTML in JSON konvertieren.

{{% blocks/products/pf/agp/feature-section-col title="Konvertieren Sie PPT über C++ in das JSON-Format" %}}
1. Öffnen Sie die PPT-Datei mit der Klassenreferenz [Presentation](https://reference.aspose.com/slides/cpp/class/aspose.slides.presentation).
2. Konvertieren Sie PPT in HTML, indem Sie die Methode [Save](https://reference.aspose.com/slides/cpp/class/aspose.slides.presentation#a06fe2a156063c8c3e5ada2713bb697ba) verwenden
3. Laden Sie das HTML-Dokument mithilfe der Klassenreferenz [IWorkbook](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook).
4. Speichern Sie das Dokument im JSON-Format mit der Methode [Save](https://reference.aspose.com/cells/cpp/class/aspose.cells.i_workbook#a5dc7de23f7ceba76a05dc1d49f51502e).
