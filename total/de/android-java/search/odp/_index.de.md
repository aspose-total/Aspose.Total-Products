{{% blocks/products/pf/agp/feature-section-col title="So durchsuchen Sie eine ODP-Datei über Android" %}}

1. Installieren Sie „Aspose.Slides für Java“.
1. Fügen Sie Ihrem Java-Projekt eine Bibliotheksreferenz hinzu (importieren Sie die Bibliothek).
1. Laden Sie ODP über eine Instanz der Presentation-Klasse.
1. Verwenden Sie die Methode findAndReplaceText, um Text zu suchen und zu ersetzen.
1. Rufen Sie die Methode „save()“ auf, um das Dokument durch Übergabe eines Ausgabedateinamens zu speichern.
