{{% blocks/products/pf/agp/feature-section-col title="So durchsuchen Sie eine WORD-Datei über Android" %}}

1. Installieren Sie „Aspose.Words für Java“.
1. Fügen Sie Ihrem Java-Projekt eine Bibliotheksreferenz hinzu (importieren Sie die Bibliothek).
1. Öffnen Sie die WORD-Quelldatei mit der Document-Klasse.
1. Legen Sie FindReplaceOptions fest. 
1. Verwenden Sie die Funktion getRange().replace, um Text zu suchen und zu ersetzen.
1. Rufen Sie die Methode „save()“ auf, um das Dokument unter Angabe eines Ausgabedateinamens zu speichern.
