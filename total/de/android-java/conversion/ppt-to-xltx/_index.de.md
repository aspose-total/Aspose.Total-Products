{{% blocks/products/pf/agp/feature-section-col title="So konvertieren Sie PPT in XLTX in Android" %}}
1. Öffnen Sie die PPT-Datei mit der Klasse [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation).

4. Speichern Sie das Dokument im XLTX-Format mit [save](https://reference.aspose.com/cells/java/com.aspose.cells/)
