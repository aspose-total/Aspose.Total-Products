{{% blocks/products/pf/feature-page-summary %}}
Sie können das JSON-Format in Ihren Android-Anwendungen in einem zweistufigen Prozess in POWERPOINT konvertieren. Erstens können Sie mit [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/) JSON in PPTX parsen. Danach können Sie mit [Aspose.Slides for Android via Java](https://products.aspose.com/slides/android-java/) PPTX in POWERPOINT konvertieren. Beide APIs sind im Paket [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) enthalten.

{{% blocks/products/pf/agp/feature-section-col title="Konvertieren Sie das JSON-Format in POWERPOINT in Android" %}}
1. Erstellen Sie ein neues [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)-Objekt und öffnen Sie die JSON-Datei
2. Speichern Sie JSON als PPTX mit [save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.SaveOptions) ) Methode
3. Laden Sie das PPTX-Dokument mithilfe der Klasse [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation).
4. Speichern Sie das Dokument im POWERPOINT-Format mit der Methode [save](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation#save-java.lang.String-int-).
