{{% blocks/products/pf/agp/feature-section-col title="Konvertieren Sie ODP in das JSON-Format in Android" %}}
1. Öffnen Sie die ODP-Datei mit der Klasse [Odpument](https://reference.aspose.com/words/java/com.aspose.words/Odpument).
2. Konvertieren Sie ODP in HTML mit [Save](https://reference.aspose.com/words/java/com.aspose.words/Odpument#save(java.lang.String,com.aspose.words.SaveOptions) ) Methode
3. Laden Sie das HTML-Dokument mithilfe der Klasse [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook).
4. Speichern Sie das Dokument im JSON-Format mit [Save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.SaveOptions))-Methode
