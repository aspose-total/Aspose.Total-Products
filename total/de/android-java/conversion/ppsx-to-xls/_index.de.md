Sie können die PPSX-Datei in Ihren Android-Anwendungen in einem zweistufigen Prozess über [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/) ganz einfach in XLS konvertieren. {{% blocks/products/pf/agp/feature-section-col title="So konvertieren Sie PPSX in XLS in Android" %}}
1. Öffnen Sie die PPSX-Datei mit der Klasse [Presentation](https://reference.aspose.com/slides/java/com.aspose.slides/Presentation).
