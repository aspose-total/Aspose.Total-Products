Sie können JSON in Ihren Android-Anwendungen in zwei Schritten in EPUB konvertieren. Erstens können Sie mit der leistungsstarken Spreadsheet Processing API [Aspose.Cells for Android via Java](https://products.aspose.com/cells/android-java/) JSON in PDF parsen. Beide APIs gehören zur Produktfamilie [Aspose.Total for Android via Java](https://products.aspose.com/total/android-java/).

{{% blocks/products/pf/agp/feature-section-col title="Konvertieren Sie das JSON-Format in Android über Java in EPUB" %}}
1. Erstellen Sie ein neues [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)-Objekt und lesen Sie gültige JSON-Daten aus der Datei
2. Importieren Sie die JSON-Datei in das Arbeitsblatt mit der Klasse [JsonUtility](https://reference.aspose.com/cells/java/com.aspose.cells/JsonUtility) und [Save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.SaveOptions)) als PDF
3. Laden Sie das PDF-Dokument mithilfe der Klasse [Document](https://reference.aspose.com/words/java/com.aspose.words/Document).
4. Speichern Sie das Dokument im EPUB-Format mit [Save](https://reference.aspose.com/words/java/com.aspose.words/Document#save(java.lang.String,com.aspose.words.SaveOptions)) Methode

Mit der API können Sie auch JSON in EPUB mit Wasserzeichen konvertieren. Um Ihrem EPUB-Dokument ein Wasserzeichen hinzuzufügen, können Sie zuerst die JSON-Datei in PDF parsen und ihr ein Wasserzeichen hinzufügen. Um ein Wasserzeichen hinzuzufügen, laden Sie die neu erstellte PDF-Datei mit der Klasse [Document](https://reference.aspose.com/words/java/com.aspose.words/Document), erstellen Sie eine Instanz von TextWatermarkOptions und setzen Sie sie seine Eigenschaften, Methode Watermark.setText aufrufen und Wasserzeichentext und Objekt von TextWatermarkOptions übergeben. Nachdem Sie das Wasserzeichen hinzugefügt haben, können Sie das Dokument in EPUB speichern.
