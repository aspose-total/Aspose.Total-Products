{{% blocks/products/pf/agp/feature-section-col title="Konvertieren Sie OTP in das JSON-Format in Android" %}}
1. Öffnen Sie die OTP-Datei mit der Klasse [Otpument](https://reference.aspose.com/words/java/com.aspose.words/Otpument).
2. Konvertieren Sie OTP in HTML mit [Save](https://reference.aspose.com/words/java/com.aspose.words/Otpument#save(java.lang.String,com.aspose.words.SaveOptions) ) Methode
3. Laden Sie das HTML-Dokument mithilfe der Klasse [Workbook](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook).
4. Speichern Sie das Dokument im JSON-Format mit [Save](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20com.aspose.cells.SaveOptions))-Methode
