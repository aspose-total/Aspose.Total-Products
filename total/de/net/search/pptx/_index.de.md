1. Rufen Sie die Methode „Save()“ auf, um das Dokument durch Übergabe eines Ausgabedateinamens zu speichern.

<h2>Entwickeln Sie eine PPTX-Dateisuchmaschine über .NET</h2>

Müssen Sie eine durchsuchbare PPTX-Software oder ein Dienstprogramm entwickeln? Mit Aspose.Slides for .NET, einer untergeordneten API von Aspose.Total for .NET, kann jeder Entwickler den oben genannten API-C#-Code integrieren, um die Textsuche in Dokumenten zu programmieren. Die leistungsstarke C#-Bibliothek ermöglicht die Programmierung einer Softwarelösung für die Dokumentensuche, die viele gängige Formate einschließlich des PPTX-Formats unterstützt.<br /><br />
