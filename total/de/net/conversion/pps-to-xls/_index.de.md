{{% blocks/products/pf/feature-page-section  h2="Konvertieren Sie Protected PPS über C# in XLS" %}}
Wenn Ihr eingegebenes PPS-Dokument beim Konvertieren der PPS-Datei in XLS passwortgeschützt ist, können Sie es nicht in XLS konvertieren, ohne das Dokument zu entschlüsseln. Wenn Ihr Dokument passwortgeschützt ist, bedeutet dies, dass bestimmte Einschränkungen für die Präsentation erzwungen werden. Um die Einschränkungen aufzuheben, muss das Passwort eingegeben werden. Eine passwortgeschützte Präsentation gilt als gesperrte Präsentation. Die API ermöglicht es Ihnen, das verschlüsselte Dokument zu öffnen, indem Sie das richtige Kennwort in einem LoadOptions-Objekt übergeben.
