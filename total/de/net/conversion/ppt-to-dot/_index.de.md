1. Fügen Sie den Verweis auf Aspose.Slides für .NET und Aspose.Words für .NET hinzu

1. Speichern Sie das Dokument mit [Aspose.Words.Document.Save("output.dot", SaveFormat.Dot)](https://apireference.aspose.com/words/net/aspose.words.document/save/methods/3)

{{% blocks/products/pf/agp/code-block title="Dieses Codebeispiel zeigt, wie Sie ein PPT mit C# in DOT konvertieren." offSpacer="" %}}

```cs
// Load the Microsoft PowerPoint PPT file
Aspose.Slides.Presentation ppt = new Aspose.Slides.Presentation("source.ppt");

var stream = new MemoryStream();

ppt.Save(stream, Aspose.Slides.Export.SaveFormat.Html);
stream.Flush();
stream.Seek(0, SeekOrigin.Begin);
// stream.Position = 0;

// Load the content of the presentation to a Word document
var dot = new Aspose.Words.Document(stream);
      
// Save the Word DOT document
dot.Save("output.dot", Aspose.Words.SaveFormat.Dot);

```
