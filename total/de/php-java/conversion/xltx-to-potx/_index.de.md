Aspose.Total für PHP über Java ist eine umfassende Suite, die eine breite Palette von Funktionen für die Arbeit mit Excel-Tabellen, Powerpoint-Präsentationen sowie die Barcode-Generierung und -Erkennung bietet. Mit Aspose.Total für PHP über Java können Sie in nur wenigen Schritten die XLTX-zu-POTX-Konvertierung zu jeder PHP-Anwendung hinzufügen.

Verwenden Sie zunächst Aspose.Cells für PHP über Java, um die XLTX-Datei in PDF zu exportieren. Verwenden Sie dann Aspose.Slides für PHP über Java, um die PDF-Datei in POTX zu konvertieren. Sowohl Aspose.Cells für PHP über Java als auch Aspose.Slides für PHP über Java sind untergeordnete APIs des Aspose.Total für PHP über Java-APIs-Pakets.

{{% blocks/products/pf/agp/feature-section-col title="Wie konvertiert man XLTX in POTX?" %}}
1. Öffnen und laden Sie die XLTX-Datei mit der Workbook-Klasse
2. Legen Sie die PDF-Speicheroptionen mithilfe des PdfSaveOptions-Objekts fest
3. Speichern Sie die XLTX-Datei mit der Speichermethode im PDF-Format
4. Verwenden Sie die Presentation-Klasse, um das exportierte PDF mit getSlides()->addFromPdf in Folien zu laden
5. Rufen Sie die Methode save auf, um sie in das POTX-Format zu konvertieren
