Mit [Aspose.Total for Java](https://products.aspose.com/total/java/) können Sie das JSON-Format in jeder Java-Anwendung in zwei einfachen Schritten in IMAGE konvertieren. Erstens können Sie mit [Aspose.Cells for Java](https://products.aspose.com/cells/java/) JSON in JPEG parsen. Danach können Sie mit [Aspose.Imaging for Java](https://products.aspose.com/imaging/java/) JPEG in IMAGE konvertieren.

Mit der API können Sie auch JSON in IMAGE mit Wasserzeichen in Ihrem IMAGE-Dokument konvertieren. Um ein Wasserzeichen hinzuzufügen, können Sie zuerst JSON in JPEG konvertieren und ein Wasserzeichen darin hinzufügen. Um ein Wasserzeichen hinzuzufügen, laden Sie eine Bilddatei mit der Klasse [Image](https://reference.aspose.com/imaging/java/com.aspose.imaging/Image), erstellen Sie ein Objekt der Klasse [Graphics](https://reference.aspose.com/imaging/java/com.aspose.imaging/Graphics) und initialisieren Sie sie mit dem Image-Objekt, erstellen Sie eine neue [Matrix](https://reference.aspose.com/imaging/java/com.aspose.imaging/Matrix) Objekt und setzen Sie die Übersetzung und Transformation auf den gewünschten Winkel und fügen Sie Wasserzeichen mit [Graphics.drawString](https://reference.aspose.com/imaging/java/com.aspose.imaging/Graphics#drawString-java.lang.String-com.aspose.imaging.Font-com.aspose.imaging.Brush-float-float-)-Methode. Nachdem Sie das Wasserzeichen in Ihr Bild eingefügt haben, können Sie das JPEG im IMAGE-Format speichern.
