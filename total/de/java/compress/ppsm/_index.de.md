{{% blocks/products/pf/agp/feature-section-col title="Komprimieren Sie die PPSM-Datei über Java" %}}

1. Fügen Sie dem Projekt eine Java-Bibliotheksreferenz hinzu
1. Laden Sie das Dokument mit der Presentation-Klasse.
1. Verwenden Sie das Klassenobjekt [Compress](https://reference.aspose.com/slides/java/com.aspose.slides/compress/).
1. Stellen Sie die Komprimierung entsprechend Ihren Anforderungen ein
1. Verwenden Sie eine der Optionen „removeUnusedMasterSlides“, „removeUnusedLayoutSlides“ oder „compressEmbeddedFonts“.
1. Rufen Sie die Speichermethode auf und rufen Sie die komprimierte PPSM-Datei ab.
