За допомогою Aspose.Total for Python via .NET або GIF6, дочірніх API GIF7, будь-який розробник Python може інтегрувати наведений вище код API для програмування програми для об’єднання документів. Потужна бібліотека Python дозволяє програмувати рішення для об’єднання документів, яке підтримує багато популярних форматів, включаючи формат PDF.<br /><br />
